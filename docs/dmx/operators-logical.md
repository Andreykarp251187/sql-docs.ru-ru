# <a name="operators---logical"></a>Операторы — логические
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  Можно использовать логические операторы в выражениях расширений интеллектуального анализа данных (DMX) для оценки значений и возвращают логическое значение в [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)].  
  
 В следующей таблице приводятся логические операторы, поддерживаемые расширениями интеллектуального анализа данных:  
  
|Оператор|Описание|  
|--------------|-----------------|  
|[И &#40;РАСШИРЕНИЙ ИНТЕЛЛЕКТУАЛЬНОГО АНАЛИЗА ДАННЫХ&#41;](../dmx/and-dmx.md)|Выполняет логическое умножение двух числовых выражений.|  
|[НЕ &#40;РАСШИРЕНИЙ ИНТЕЛЛЕКТУАЛЬНОГО АНАЛИЗА ДАННЫХ&#41;](../dmx/not-dmx.md)|Выполняет операцию логического отрицания над числовым выражением.|  
|[ИЛИ &#40;РАСШИРЕНИЙ ИНТЕЛЛЕКТУАЛЬНОГО АНАЛИЗА ДАННЫХ&#41;](../dmx/or-dmx.md)|Выполняет логическое сложение двух числовых выражений.|
