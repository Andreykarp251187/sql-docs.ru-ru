# <a name="data-mining-extensions-dmx-statements"></a>Инструкции расширений интеллектуального анализа данных
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  Работа с моделями интеллектуального анализа данных [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] включает в себя следующие основные задачи:  
  
-   Создание структур и моделей интеллектуального анализа данных  
  
-   Обработка структур и моделей интеллектуального анализа данных  
  
-   Удаление структур и моделей интеллектуального анализа данных  
  
-   Копирование моделей интеллектуального анализа данных  
  
-   Обзор моделей интеллектуального анализа данных  
  
-   Прогнозирование по моделям интеллектуального анализа данных  
  
 Можно использовать инструкции расширений интеллектуального анализа данных для выполнения каждой из этих задач программно.  
  
 Создание структур и моделей интеллектуального анализа данных  
 Используйте [CREATE MINING STRUCTURE &#40;расширений интеллектуального анализа данных&#41; ](../dmx/create-mining-structure-dmx.md) инструкцию, чтобы добавить новую структуру интеллектуального анализа данных в базе данных. Затем можно использовать [ALTER MINING STRUCTURE &#40;расширений интеллектуального анализа данных&#41; ](../dmx/alter-mining-structure-dmx.md) инструкцию, чтобы добавить модели интеллектуального анализа данных к структуре интеллектуального анализа данных.  
  
 Используйте [CREATE MINING MODEL &#40;расширений интеллектуального анализа данных&#41; ](../dmx/create-mining-model-dmx.md) инструкцию, чтобы построить новую структуру интеллектуального анализа данных модели и связанные с ней интеллектуального анализа данных.  
  
 Обработка структур и моделей интеллектуального анализа данных  
 Используйте [INSERT INTO &#40;расширений интеллектуального анализа данных&#41; ](../dmx/insert-into-dmx.md) инструкции для обработки структуры интеллектуального анализа данных и модели интеллектуального анализа данных.  
  
 Удаление структур и моделей интеллектуального анализа данных  
 Используйте [удалить &#40;расширений интеллектуального анализа данных&#41; ](../dmx/delete-dmx.md) инструкцию, чтобы удалить все данные обучения из модели интеллектуального анализа данных или структуры интеллектуального анализа данных. Используйте [DROP MINING STRUCTURE &#40;расширений интеллектуального анализа данных&#41; ](../dmx/drop-mining-structure-dmx.md) или [DROP MINING MODEL &#40;расширений интеллектуального анализа данных&#41; ](../dmx/drop-mining-model-dmx.md) инструкции, чтобы полностью удалить структуру интеллектуального анализа данных или модели интеллектуального анализа данных из базы данных.  
  
 Копирование моделей интеллектуального анализа данных  
 Используйте [SELECT INTO &#40;расширений интеллектуального анализа данных&#41; ](../dmx/select-into-dmx.md) инструкции для копирования структуры существующей модели интеллектуального анализа данных в новую модель интеллектуального анализа данных и для обучения новой модели с теми же данными.  
  
 Обзор моделей интеллектуального анализа данных  
 Используйте [ВЫБЕРИТЕ &#40;расширений интеллектуального анализа данных&#41; ](../dmx/select-dmx.md) инструкции для обзора сведений, алгоритм интеллектуального анализа данных вычисляет и сохраняет в модели интеллектуального анализа данных при обучении модели. Также как и с [!INCLUDE[tsql](../includes/tsql-md.md)], можно использовать несколько предложений с инструкцией SELECT, для увеличения ее эффективности. Эти предложения включают [DISTINCT FROM \<модели >](../dmx/select-distinct-from-model-dmx.md), [FROM \<модели >. СЛУЧАЯХ](../dmx/select-from-model-cases-dmx.md), [FROM \<модели >. SAMPLE_CASES](../dmx/select-from-model-sample-cases-dmx.md), [FROM \<модели >. СОДЕРЖИМОГО](../dmx/select-from-model-content-dmx.md) и [FROM \<модели >. DIMENSION_CONTENT](../dmx/select-from-model-dimension-content-dmx.md).  
  
 Прогнозирование по моделям интеллектуального анализа данных  
 Используйте [PREDICTION JOIN](../dmx/select-from-model-prediction-join-dmx.md) предложение инструкции SELECT для создания прогнозов, основанных на существующую модель интеллектуального анализа данных.  
  
 Можно также импортировать и экспортировать модели при помощи [ИМПОРТА &#40;расширений интеллектуального анализа данных&#41; ](../dmx/import-dmx.md) и [Экспорт &#40;расширений интеллектуального анализа данных&#41; ](../dmx/export-dmx.md) инструкций.  
  
 Эти задачи разделяются на две категории: инструкции определения данных и инструкции управления данными, которые описываются в следующей таблице.  
  
|Раздел|Описание|  
|-----------|-----------------|  
|[Инструкции определения расширений интеллектуального анализа данных](../dmx/dmx-statements-data-definition.md)|Часть языка определения данных (DDL). Используется для определения новой модели интеллектуального анализа данных (включая обучение).|  
|[Расширения интеллектуального анализа данных &#40;расширений интеллектуального анализа данных&#41; инструкций по обработке данных](../dmx/dmx-statements-data-manipulation.md)|Часть языка обработки данных (DML). Используется для работы с существующими моделями интеллектуального анализа данных, включая обзор модели и создание прогнозов.|
