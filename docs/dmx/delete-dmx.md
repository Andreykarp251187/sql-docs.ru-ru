# <a name="delete-dmx"></a>DELETE (расширения интеллектуального анализа данных)
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  Позволяет удалить модель, структуру интеллектуального анализа данных или структуру вместе со всеми связанными моделями, в зависимости от используемых предложений расширений интеллектуального анализа данных.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
DELETE FROM [MINING MODEL] <model>[.CONTENT]  
DELETE FROM [MINING STRUCTURE] <structure>[.CONTENT]|[.CASES]  
```  
  
## <a name="arguments"></a>Аргументы  
 *model*  
 Идентификатор модели.  
  
 *дереве*  
 Идентификатор структуры.  
  
## <a name="remarks"></a>Remarks  
 Если не указать **модель** или **структуру интеллектуального**анализа данных, [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] ищет тип объекта на основе имени и обрабатывает правильный объект. Если сервер содержит структуру и модель интеллектуального анализа данных с одинаковыми именами, возвращается ошибка.  
  
 В следующей таблице описан результат использования различных форм синтаксиса.  
  
|Инструкция|Результат|  
|---------------|------------|  
|Удалить из структуры структуры*\<интеллектуального анализа данных>*<br /><br /> или диспетчер конфигурации служб<br /><br /> Удаление из*\<структуры *структуры интеллектуального анализа данных>. СОДЕРЖАНИ|Выполняет ProcessClear в структуре интеллектуального анализа данных. Все содержимое структуры интеллектуального анализа данных и связанных с ней моделей интеллектуального анализа данных удаляется.|  
|Удаление из*\<структуры *структуры интеллектуального анализа данных>. СЛУЧАЯХ|Выполняет Процессклеарструктуреонли в структуре интеллектуального анализа данных. Все содержимое структуры интеллектуального анализа данных удаляется, а связанные с ней модели интеллектуального анализа данных остаются без изменений. После удаления структуры интеллектуального анализа данных детализация связанных с ней моделей становится невозможной.|  
|Удаление из модели модели*\<интеллектуального анализа данных>*<br /><br /> или диспетчер конфигурации служб<br /><br /> Удаление из*\<модели *модели интеллектуального анализа данных>. СОДЕРЖАНИ|Выполняет ProcessClear в модели интеллектуального анализа данных, но оставляет значения состояния без изменений. Значения состояний представляют собой возможные состояния столбца. Например, значениями состояний для столбца «Пол» являются «Мужской» и «Женский».|  
  
 Дополнительные сведения о типах обработки см. в разделе [элемент Type &#40;XMLA&#41;](https://docs.microsoft.com/bi-reference/xmla/xml-elements-properties/type-element-xmla).  
  
## <a name="examples"></a>Примеры  
 Следующий пример удаляет все содержимое модели NB_Sample.  
  
```  
DELETE FROM NB_Sample.CONTENT  
```
