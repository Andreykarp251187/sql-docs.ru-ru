# <a name="sql-server-migration-assistant-for-access-accesstosql"></a>Помощник по миграции SQL Server для Access (Акцесстоскл)

[!INCLUDE[msCoName](../../includes/msconame_md.md)][!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Помощник по миграции (SSMA) для Access — это инструмент для переноса баз данных [!INCLUDE[msCoName](../../includes/msconame_md.md)] из версий Access с 97 по [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2010 в [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2012, [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2014, [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2016, 2017 в Windows и [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Linux, 2019 в Windows и Linux [!INCLUDE[msCoName](../../includes/msconame_md.md)] , а также в базе данных SQL Azure. SSMA для Access преобразует объекты базы данных Access [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] в или объекты базы данных SQL Azure, загружает [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] эти объекты в или базу данных SQL Azure, а затем переносит [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] данные из Access в базу данных SQL Azure или.
  
В этой документации рассказывается о SSMA для доступа и содержатся пошаговые инструкции по переносу баз данных Access [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] в базу данных SQL Azure и сведения о проблемах, которые могут возникнуть после миграции.
