# <a name="refresh-from-database-accesstosql"></a>Обновление из базы данных (AccessToSQL)
**Обновление из базы данных** диалоговое окно позволяет выбрать объекты для обновления из базы данных Access. Строки в диалоговом окне, маркируются цветом, исходя из состояния метаданных:  
  
-   Если метаданные объекта был изменен локально и в базе данных Access, строке отображается синим цветом.  
  
-   Если метаданные объекта был изменен в базе данных Access, но не в SSMA, что строка является желтый.  
  
-   Если метаданные объекта был изменен локально, но не в базе данных Access, строке отображается зеленым цветом.  
  
-   Если объект является новой базы данных Access, строки розовый.  
  
Можно указать параметры обновления объекта по умолчанию в **параметры проекта** диалоговое окно. Дополнительные сведения см. в разделе [параметры проекта &#40;загрузка объектов&#41; &#40;AccessToSQL&#41;](../../ssma/access/project-settings-loading-objects-accesstosql.md)  
  
Для доступа к **обновление из базы данных** диалоговом окне щелкните правой кнопкой мыши любой **базы данных** узел в обозревателе метаданных доступа и нажмите кнопку **обновление из базы данных**.
