# <a name="convert-load-and-migrate-accesstosql"></a>Преобразование, Загрузка и миграция (Акцесстоскл)
Диалоговое окно **Преобразование, Загрузка и миграция** сообщает о состоянии миграции при использовании кнопок преобразовать, загрузить и выполнить миграцию или мастера миграции.  
  
После завершения миграции можно просмотреть информационные, предупреждения и сообщения об ошибках, щелкнув ссылку в каждом разделе. Ссылка показывает, сколько доступно ошибок, предупреждений и информационных сообщений, например "0 ошибок, 4 предупреждения".  
  
В разделе **Миграция данных для выбранных объектов** страницы также содержится кнопка **отчет** . Можно нажать эту кнопку, чтобы просмотреть отчет о переносе данных.  
  
## <a name="options"></a>Параметры  
**Отчет**  
Отображает связанный отчет о преобразовании или переносе.
