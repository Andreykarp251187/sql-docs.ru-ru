# <a name="global-settings-output-window--sybasetosql"></a>Глобальные параметры (окно вывода) (SybaseToSQL)
Используйте **глобальные параметры** диалоговое окно, чтобы указать параметры для SSMA.  
  
Для доступа к параметрам окно вывода, на **средства** меню, выберите **глобальные параметры**, нажмите кнопку **графического пользовательского интерфейса** в нижней части левой панели, а затем выберите **выходных данных Окно**.  
  
## <a name="options"></a>Параметры  
**Показать область списка ошибок**  
Выберите **True** Если вы хотите, чтобы SSMA для отображения на панели списка ошибок, если сообщения.  
  
**Показать область вывода**  
Выберите **True** Если SSMA для отображения области вывода, если сообщения.
