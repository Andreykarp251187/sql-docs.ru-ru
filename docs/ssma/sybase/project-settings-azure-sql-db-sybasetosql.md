# <a name="project-settings-azure-sql-db--sybasetosql"></a>Параметры проекта (база данных SQL Azure) (SybaseToSQL)
С помощью параметров проекта базы данных SQL Azure можно настроить добавляемый в диалоговом окне подключения суффикс Database, а также разрешить использование механизма пульса в подключении к базе данных SQL Azure.  
  
Панель базы данных SQL Azure доступна в диалоговых окнах **Параметры проекта** и **Параметры проекта по умолчанию** .  
  
-   Используйте диалоговое окно Параметры проекта, чтобы задать параметры конфигурации для текущего проекта. Чтобы получить доступ к параметрам базы данных SQL Azure, в меню **Сервис** выберите **Параметры проекта**, щелкните **Общие** в нижней части левой панели, а затем выберите **база данных SQL Azure**.  
  
-   Используйте диалоговое окно Параметры проекта по умолчанию, чтобы задать параметры конфигурации для всех проектов. Чтобы получить доступ к параметрам базы данных SQL Azure, в меню **Сервис** выберите **Параметры Дефаултпрожект**, щелкните **Общие** в нижней части левой панели, а затем выберите **база данных SQL Azure**.  
  
## <a name="connectivity"></a>Соединение  
**Интервал пульса**  
  
Указывает интервал времени, используемый для механизма пульса, чтобы обеспечить активность подключения к базе данных SQL Azure в формате "минуты: секунды".  
  
**Значение по умолчанию**: ' 4:45 '  
  
Значение должно быть указано в формате "м:СС" (например, "4:45" или "0:50").  
  
**Суффикс сервера базы данных SQL Azure**  
  
Указывает суффикс сервера базы данных SQL Azure  
  
**Значение по умолчанию**: "Database.Windows.NET".
