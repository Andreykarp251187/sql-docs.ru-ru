# <a name="connect-to-mysql-mysqltosql"></a>Подключение к MySQL (MySQLToSQL)
Используйте **подключение к MySQL** диалоговое окно для подключения к базе данных MySQL, которые требуется перенести.  
  
Чтобы открыть это диалоговое окно, в **файл** меню, выберите **подключение к MySQL**. Если вы ранее подключались, команда является **повторное подключение к MySQL**.  
  
## <a name="options"></a>Параметры  
**Поставщик**  
  
Доступен поставщик MySQL — драйвер 5.1 ODBC для MySQL (доверенные).  
  
**Режим**  
  
Режим по умолчанию является обычным режимом. В стандартном режиме введите или выберите значения для MySQL, имя сервера, порт сервера, имя пользователя и пароль.  
  
**Имя сервера**  
  
Введите имя сервера MySQL. Это стандартный режим.  
  
**Порт сервера**  
  
Введите порт сервера. Порт сервера по умолчанию — 3306. Это стандартный режим.  
  
**Имя пользователя**  
  
Введите имя пользователя, который будет использоваться SSMA для подключения к базе данных MySQL.  
  
**Пароль**  
  
Введите пароль для имени пользователя.  
  
**SSL**  
  
Для безопасного подключения к MySQL, внесите использовать из Secure Socket Layer (SSL), установив флажок **SSL** флажок.  
  
**Настройка**  
  
Он предоставляет параметр для настройки подключения к MySQL через Secure Socket Layer (SSL).  
  
> [!NOTE]  
> Чтобы включить **Настройка**, должно быть присвоено SSL **True**.  
  
На кнопку «Настроить», появляется диалоговое окно. Использование шифрования при подключении к базе данных MySQL, путь к следующим файлам три сертификата, в диалоговом окне должен быть определен [конфиденциальности усиленной Mail сертификаты (PEM)]:  
  
-   **Центр сертификации SSL:** Указывает путь к файлу со списком доверия центров сертификации SSL.  
  
-   **SSL-сертификат:** Задает имя файла сертификата SSL для установления безопасного соединения.  
  
-   **КЛЮЧ SSL:** Задает имя файла ключа SSL, используемый для создания безопасного подключения.  
  
> [!NOTE]  
> -   **ОК** кнопка включена, когда предоставила необходимые сведения. Если любой из пути к файлам являются недействительными, кнопку «ОК» будет отключен.  
> -   **Отменить** кнопки закрывает диалоговое окно и **приводит к отключению** параметр SSL из основной формы подключения.
