# <a name="server-groups-in-includename-sosincludesname-sos-shortmd"></a>Группы серверов в [!INCLUDE[name-sos](../includes/name-sos-short.md)]

Группы серверов предоставляют способ упорядочить свои подключения к серверам и базам данных, которыми вы работаете. При создании групп серверов, сведения о конфигурации сохраняются в *параметры пользователя*.

## <a name="create-and-edit-server-groups"></a>Создание и изменение групп серверов

1. Нажмите кнопку **группа серверов** в верхней части *СЕРВЕРЫ* боковой панели.
2. Введите имя группы и выберите цвет для группы. При необходимости добавьте описание.

   ![Добавление группы серверов](./media/server-groups/add-server-group.png)

Чтобы изменить существующую группу серверов, щелкните правой кнопкой мыши группу и выберите **изменить группу серверов**.

Для изменения цветов группы сервера доступности, изменить *групп серверов* значения в [параметры пользователя](settings.md).

> [!TIP]
> Серверы между различные группы серверов можно перетаскивать.
