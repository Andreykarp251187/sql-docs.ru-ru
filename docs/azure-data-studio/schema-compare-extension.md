# <a name="schema-compare-extension-preview"></a>Расширение Сравнения схем (предварительная версия)
Расширение сравнения схем предоставляет удобный интерфейс для сравнения файлов DACPAC и баз данных, а также применения изменений из источника к целевому объекту.

В настоящее время это расширение находится в режиме начальной предварительной версии. Сообщить о проблемах и запросить функции можно [здесь](https://github.com/microsoft/azuredatastudio/issues).

## <a name="install-the-schema-compare-extension"></a>Установка расширения сравнения схем

1. Чтобы открыть диспетчер расширений и получить доступ к имеющимся расширениям, щелкните значок расширений или выберите пункт **Расширения** в меню **Вид**.
2. Выберите доступное расширение и просмотрите сведения о нем.
1. Выберите нужное расширение ("Сравнение схем") и **установите** его.

## <a name="how-do-i-start-a-schema-comparison"></a>Запуск сравнения схем
* Самый простой способ начать сравнение схем — щелкнуть базу данных правой кнопкой мыши в обозревателе объектов и выбрать пункт **Сравнение схем**.
* Пользователь может также открыть диалоговое окно сравнения схем из палитры команд (CTRL+SHIFT+P), выполнив поиск по запросу **Сравнение схем**.

## <a name="why-would-i-use-the-schema-compare"></a>Назначение сравнения схем
Расширение сравнения схем было создано, чтобы предоставить возможность сравнивать схемы из файлов DACPAC и баз данных, а также применять изменения.
