# <a name="monitor-the-appliance-with-system-views---analytics-platform-system"></a>Мониторинг устройства с помощью системных представлений — система анализа платформы
В этой статье перечислены системные представления, которые можно использовать для мониторинга SQL Server PDW.  
  
## <a name="to-monitor-the-appliance-by-using-system-views"></a>Мониторинг устройства с помощью системных представлений  
SQL Server PDW включает в себя комплексные системные представления, позволяющие получать подробные сведения о работоспособности, состоянии и производительности устройства. Эта таблица содержит ссылки на системные представления, которые можно использовать для каждой возможности мониторинга.
