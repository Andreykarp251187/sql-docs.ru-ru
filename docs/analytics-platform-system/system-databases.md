# <a name="system-databases-in-parallel-data-warehouse-pdw"></a>Системные базы данных в параллельном хранилище данных (PDW)
Системные базы данных — это базы данных, которые SQL Server PDW используются для управления пользователями, базами данных и временными данными. SQL Server PDW включает следующие системные базы данных: Master и tempdb.  
  
## <a name="system-database-topics"></a>Статьи о системной базе данных  
  
-   [База данных master](master-database.md)  
  
-   [База данных tempdb](tempdb-database.md)
