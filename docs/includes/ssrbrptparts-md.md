  Части отчета — это самостоятельные элементы отчета, хранящиеся на сервере отчетов, которые можно включать в другие отчеты. Для просмотра коллекции частей отчетов и выбора элементов для добавления в отчет используйте построитель отчетов. Для сохранения частей отчета в коллекции частей отчета используйте конструктор отчетов или построитель отчетов. Дополнительные сведения см. в разделах **Части отчета (построитель отчетов и службы SSRS)** и **Части отчета в конструкторе отчетов (SSRS)** на веб-сайте microsoft.com.
