Мы накапливаем и храним документацию по очень старым версиям Microsoft SQL Server в наборе архивных веб-страниц. Архивные веб-страницы не обрабатываются поисковыми системами, такими как Bing.com и Google.com. Эти архивы доступны в нашей документации по адресу _previous-versions/sql/_ :

- [https://docs.microsoft.com/previous-versions/sql/](https://docs.microsoft.com/previous-versions/sql/)

В них входит документация, по крайней мере, по следующим старым версиям:

- SQL Server 2012
- SQL Server 2008 R2
- SQL Server 2008
- SQL Server 2005

[Документация по SQL Server 2014](https://docs.microsoft.com/sql/2014-toc/index?view=sql-server-2014) по-прежнему доступна по адресу основной документации.

[Документация по SQL Server 2019](https://docs.microsoft.com/sql/sql-server/sql-server-technical-documentation?toc=/sql/toc/toc.json&view=sql-server-ver15) доступна по адресу основной документации. Выберите нужную версию в раскрывающемся списке в верхней части страницы.

Дополнительные сведения о документации по предыдущим версиям SQL Server см. на странице [Документация по предыдущим версиям SQL Server](/sql/toc/previous-versions-sql-server?view=sql-server-previousversions).
