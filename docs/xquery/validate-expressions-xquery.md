# <a name="validate-expressions-xquery"></a>Проверка выражений (XQuery)
[!INCLUDE[sqlserver](../includes/applies-to-version/sqlserver.md)]

  В этой реализации выражение **Validate** не поддерживается. Результаты выражений конструкции XQuery всегда нетипизированы. Если результат выражения XQuery должен быть типизирован, используйте выражение SQL CAST для приведения результата к типу данных **XML** с предпочтительной коллекцией схем.
