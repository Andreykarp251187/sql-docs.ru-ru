# <a name="open-an-existing-solution"></a>Открытие существующего решения
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]
В обозревателе решений одновременно может быть открыто только одно решение.  
  
### <a name="to-open-an-existing-solution"></a>Открытие существующего решения  
  
1.  В меню **Файл** выберите пункт **Открыть**, затем выберите **Проект/Решение**.  
  
2.  В диалоговом окне **Открыть проект** перейдите к требуемому решению и щелкните его, а затем выберите **Открыть**.
