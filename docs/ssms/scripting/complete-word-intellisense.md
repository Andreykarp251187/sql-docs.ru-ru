# <a name="complete-word-intellisense"></a>Автоматическое завершение слов (технология IntelliSense)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
  Технология IntelliSense «Завершение слов» [!INCLUDE[msCoName](../../includes/msconame-md.md)] завершает название переменной, команды или имени функции, когда введено достаточно символов, чтобы уникальным образом определить этот термин.  
  
 Возможны четыре способа включить завершение слов.  
  
-   В меню **Правка** укажите пункт **IntelliSense**и выберите **Завершение слов**.  
  
-   Нажмите клавиши ALT+ВПРАВО.  
  
-   Щелкните правой кнопкой мыши в редакторе текстов и выберите пункт **Завершение слов**.  
  
-   Нажмите кнопку **Завершение слов** на панели инструментов **Редактор текстов** .  
  
 Если для набранных символов возможно несколько соответствий, вызовите функцию «Завершение слов», чтобы она отобразила все возможные соответствия, из которых можно выбрать термин и вставить его в программный код.
