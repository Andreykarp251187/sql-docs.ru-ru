# <a name="kpicurrenttimemember-mdx"></a>KPICurrentTimeMember (многомерные выражения)


  Возвращает текущий элемент времени указанного ключевого показателя эффективности (KPI).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
KPICurrentTimeMember(KPI_Name)  
```  
  
## <a name="arguments"></a>Аргументы  
 *KPI_Nam*e  
 Допустимое строковое выражение, указывающее имя ключевого показателя эффективности.  
  
## <a name="remarks"></a>Remarks  
 Ключевой показатель эффективности может содержать различные элементы времени из элемента по умолчанию измерения времени.
