# <a name="mdx-statement-reference-mdx"></a>Справка по инструкциям многомерных выражений (многомерные выражения)


  Инструкции многомерных выражений делятся на три группы (см. следующую таблицу).  
  
## <a name="in-this-section"></a>в этом разделе  
  
|Раздел|Описание|  
|-----------|-----------------|  
|[Инструкции для создания скриптов многомерных выражений (многомерные выражения)](../mdx/mdx-scripting-statements-mdx.md)|Многомерные инструкции скриптов для управления областью, контекстом и потоками в скриптах многомерных выражений.|  
|[Инструкции определения данных многомерных выражений &#40;&#41;многомерных выражений ](../mdx/mdx-data-definition-statements-mdx.md)|Многомерные инструкции языка определения данных (DDL) для создания, сбрасывания и работы с многомерными объектами.|
