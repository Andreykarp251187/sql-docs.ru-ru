# <a name="bitwise-operators"></a>Побитовые операторы


  Логические операторы вычисляют значения и возвращают логическое значение. В многомерных выражениях логические операторы не выполняют побитовых операций.  
  
 В многомерных выражениях поддерживаются логические операторы, перечисленные в следующей таблице.  
  
|Оператор|Описание|  
|--------------|-----------------|  
|[ПЕРЕТАСКИВАНИ](../mdx/and-mdx.md)|Выполняет логическое умножение двух числовых выражений.|  
|[IS](../mdx/is-mdx.md)|Выполняет логическое сравнение двух выражений объектов.|  
|[НЕДОСТАТОЧНО](../mdx/not-mdx.md)|Выполняет операцию логического отрицания над числовым выражением.|  
|[OR](../mdx/or-mdx.md)|Выполняет логическое сложение двух числовых выражений.|  
|[XOR](../mdx/xor-mdx.md)|Выполняет логическое исключение над двумя числовыми выражениями.|
