# <a name="using-mathematical-functions"></a>Математические функции


  Математические функции выполняют математические операции с числовыми выражениями и возвращают результат операции.  
  
 По умолчанию передаваемое в математическую функцию число интерпретируется как число с плавающей точкой двойной точности.
