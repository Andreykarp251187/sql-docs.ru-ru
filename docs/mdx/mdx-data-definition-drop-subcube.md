# <a name="mdx-data-definition---drop-subcube"></a>Определение данных многомерных выражений — DROP SUBCUBE


  Сбрасывает указанный вложенный куб, возвращаясь к ранее определенному кубу или определению вложенного куба с указанным именем.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
DROP SUBCUBE Subcube_Name   
```  
  
## <a name="arguments"></a>Аргументы  
 *Subcube_Name*  
 Допустимая строка выражения, представляющая имя вложенного куба.
