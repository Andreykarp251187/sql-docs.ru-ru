# <a name="create-perspectives"></a>Создание перспектив

[!INCLUDE[ssas-appliesto-sql2017-later-aas](../../includes/ssas-appliesto-sql2017-later-aas.md)]

В этом занятии вы создадите перспективы Internet Sales. Перспектива определяет просматриваемое подмножество модели, реализующее точки наблюдения, которые сосредоточены на определенном аспекте бизнеса либо предназначены для использования в конкретном приложении. Когда пользователь подключается к модели с помощью перспективы, они увидят лишь те объекты модели (таблицы, столбцы, меры, иерархии и ключевые показатели эффективности) как поля, определенные в перспективе. Дополнительные сведения см. в разделе [Перспективы](../tabular-models/perspectives-ssas-tabular.md).
  
Перспективы Internet Sales, созданные в этом занятии исключает объект таблицы DimCustomer. При создании перспективы, исключающей из представления определенные объекты, объекты продолжают существовать в модели. Тем не менее не отображается в списке полей клиента отчетов. Вычисляемые столбцы и меры, как входящие в перспективу, так и нет, все равно смогут проводить вычисления с использованием данных из исключенного объекта.  
  
Цель этого занятия — описать создание перспектив и ознакомиться со средствами разработки табличных моделей. Если вы потом развернуть эту модель, чтобы включить дополнительные таблицы, можно создать дополнительные перспективы для определения разных точек обзора модели, например, запасов и продаж.  
  
Предполагаемое время для выполнения этого занятия: **Пять минут**

## <a name="create-perspectives"></a>Создание перспектив  
  
#### <a name="to-create-an-internet-sales-perspective"></a>Создание перспективы «Продажи через Интернет»  
  
1.  Нажмите кнопку **модели** меню > **перспективы** > **Создание и управление**.  
  
2.  В диалоговом окне **Перспективы** щелкните **Создать перспективу**.  
  
3.  Дважды щелкните **Новая перспектива** заголовок столбца, а затем переименуйте **Интернет-продажи**.  
  
4.  Выберите все таблицы *за исключением* **DimCustomer**.  
  
    ![как lesson8 перспективы](../tutorial-tabular-1400/media/as-lesson8-perspectives.png)
  
    В одном из следующих занятий вы использования функции анализ в Excel для тестирования этой перспективы. Список полей сводной таблицы Excel включает в себя все таблицы, кроме таблицы DimCustomer.
