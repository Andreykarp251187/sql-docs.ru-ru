# <a name="getting-data"></a>Получение данных
[Основы ADO](../../../ado/guide/data/ado-fundamentals.md), а также пример [HelloData](../../../ado/guide/data/hellodata-a-simple-ado-application.md) в частности, представляют четыре основные операции, связанные с созданием приложения ADO: получение данных, проверка данных, редактирование данных и обновление данных. В этом разделе подробно рассматривается получение данных.  
  
 На базовом уровне несколько объектов ADO участвуют в операциях получения данных. Сначала необходимо подключиться к источнику данных с помощью объекта **соединения** ADO. Затем инструкции передаются в источник данных с помощью объекта **команды** ADO. Наконец, чаще всего данные поступают в объект **набора записей** ADO.

-   [Подготовка и выполнение команд](../../../ado/guide/data/preparing-and-executing-commands.md)
