# <a name="getting-data"></a>Получение данных
[Основные принципы ADO](../../../ado/guide/data/ado-fundamentals.md)и [HelloData](../../../ado/guide/data/hellodata-a-simple-ado-application.md) пример, в частности, представили четыре основные операции, связанные с созданием приложения ADO: получение данных, изучения данных, изменения данных и обновление данных. В этом разделе рассматриваются получение данных более подробно.  
  
 На базовом уровне несколько объектов ADO участвуют в операции получения данных. Сначала необходимо подключиться к источнику данных с помощью ADO **подключения** объекта. Затем передайте инструкции к источнику данных, с помощью ADO **команда** объекта. Наконец, наиболее часто для получения данных в ADO **записей** объекта.

-   [Подготовка и выполнение команд](../../../ado/guide/data/preparing-and-executing-commands.md)
