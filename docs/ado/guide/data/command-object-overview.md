# <a name="command-object-overview"></a>Обзор объекта Command
С помощью **команда** объекта, можно сделать следующее:  
  
-   Определение исполняемого файла текста команды (например, инструкцию SQL или хранимую процедуру) с помощью **CommandText** свойство.  
  
-   Определить параметризованные запросы или аргументов хранимой процедуры с помощью **параметр** объектов и **параметры** коллекции.  
  
-   Выполнение команды и возвращать **записей** объекта, при необходимости с помощью **Execute** метод.  
  
-   Укажите тип команды с помощью **CommandType** свойство перед выполнением для оптимизации производительности.  
  
-   Укажите конкретные сведения о текст команды с помощью **диалект** свойство **команда** объекта.  
  
-   Элемент управления сохраняет ли поставщик подготовленную (или скомпилированную) версию команды перед выполнением с помощью **подготовленных** свойство.  
  
-   Задайте число секунд, поставщик будет ожидать выполнения с помощью команды **CommandTimeout** свойство.  
  
-   Связать открытое соединение с **команда** объекта, задав его **ActiveConnection** свойство.  
  
-   Задайте **имя** свойство для идентификации **команда** объекту в качестве метода в связанном **подключения** объекта.  
  
-   Передайте **команда** объект **источника** свойство **записей** для получения данных.  
  
-   Передайте **Stream** объект, содержащий команды (например, XML-команды) для поставщика, который поддерживает его.
