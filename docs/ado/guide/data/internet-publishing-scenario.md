# <a name="internet-publishing-scenario"></a>Сценарий публикации в Интернете
Данный пример кода демонстрирует, как использовать ADO с поставщиком данных Microsoft OLE DB для публикаций в Интернете. В этом сценарии вы создадите приложение Visual Basic, который использует **записей**, **записи**, и **Stream** публикации объектов для отображения содержимого ресурсов с помощью Публикации в Интернете поставщика.  
  
 Чтобы создать этот сценарий необходимы следующие действия:  
  
-   [Шаг 1. Настройка проекта Visual Basic](../../../ado/guide/data/step-1-set-up-the-visual-basic-project.md)  
  
-   [Шаг 2. Инициализация главного списка](../../../ado/guide/data/step-2-initialize-the-main-list-box.md)  
  
-   [Шаг 3. Заполнение списка полей](../../../ado/guide/data/step-3-populate-the-fields-list-box.md)  
  
-   [Шаг 4. Заполнение текстового поля сведений](../../../ado/guide/data/step-4-populate-the-details-text-box.md)
