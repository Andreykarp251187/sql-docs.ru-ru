# <a name="ado-errors"></a>Ошибки ADO
**ErrorValueEnum** константа описывает значения ошибок ADO. Полный список этих перечислимых констант, включая значения, см. в разделе [приложении б: Ошибки ADO](../../../ado/guide/appendixes/appendix-b-ado-errors.md). В этом разделе будет изучить некоторые из более интересных ошибок и объясните некоторых определенных ситуациях, которые могут вызывать их или решений для устранения проблемы. Оба **ErrorValueEnum** перечислены константы и короткий положительное десятичное число.

|Number|Константа ErrorValueEnum|Описание и возможные причины|
|------------|-----------------------------|----------------------------------|
|**3000**|**adErrProviderFailed**|Поставщику не удалось выполнить запрошенную операцию.|
|**3001**|**adErrInvalidArgument**|Аргументы имеют неправильный тип, выходят за пределы допустимого диапазона или конфликтуют друг с другом. Эта ошибка обычно вызвана опечатка в инструкцию SQL SELECT. Например имя поля с ошибками или имя таблицы можно создать эту ошибку. Эта ошибка может возникнуть, если поле или таблицу с именем в инструкции SELECT не существует в хранилище данных.|
|**3002**|**adErrOpeningFile**|Не удалось открыть файл. Имя файла с ошибкой был указан, или переместить файл, его переименована или удалена. В сети диск может быть временно недоступна или сетевой трафик, препятствующих соединения.|
|**3003**|**adErrReadFile**|Не удалось прочитать файл. Неверно указано имя файла, может, файл был перемещен или удален или возможно, файл поврежден.|
|**3004**|**adErrWriteFile**|Запись в один файл не удалось. Может закрыть файл и попытался запись в него, или файл может быть поврежден. Если файл находится на сетевом диске, временное состояние сети может привести к записи на сетевой диск.|
|**3021**|**adErrNoCurrentRecord**|Либо **BOF** или **EOF** имеет значение True, или текущую запись была удалена. Запрошенная операция требует наличия текущей записи.<br /><br /> Была предпринята попытка обновить записи с помощью **найти** или **Seek** для перемещения указателя записи к нужной записи. Если запись не найдена, **EOF** будет иметь значение True. Эта ошибка также может возникать после сбоя **AddNew** или **удалить** так как отсутствует текущая запись, если эти методы вызывают ошибку.|
|**3219**|**adErrIllegalOperation**|Операция не разрешена в этом контексте.|
|**3220**|**adErrCantChangeProvider**|Указанный поставщик отличается от одной уже для использования.|
|**3246**|**adErrInTransaction**|**Подключение** объекта не могут быть закрыты явным образом в транзакции. Объект **записей** или **подключения** объект, который в настоящее время участвует в транзакции не могут быть закрыты. Вызвать либо метод **RollbackTrans** или **CommitTrans** перед тем как закрыть объект.|
|**3251**|**adErrFeatureNotAvailable**|Объект или поставщик не может выполнить запрошенную операцию. Некоторые операции зависеть от конкретного поставщика версии.|
|**3265**|**adErrItemNotFound**|Элемент не найден в коллекции, соответствующий запрашиваемое имя или порядковый номер. Задано неверное имя поля или таблицы.|
|**3367**|**adErrObjectInCollection**|Объект уже присутствует в коллекции. Не удается записать. Невозможно добавить объект в коллекцию дважды.|
|**3420**|**adErrObjectNotSet**|Объект больше не является допустимым.|
|**3421**|**adErrDataConversion**|Приложение использует значение неверного типа для текущей операции. Возможно, указан строку к операции, которая принимает поток, например.|
|**3704**|**adErrObjectClosed**|Операция запрещена, если данный объект закрыт. **Подключения** или **записей** был закрыт. Например некоторые другие подпрограммы закрытием глобального объекта. Чтобы предотвратить эту ошибку можно путем проверки **состояние** свойство перед выполнением операции.|
|**3705**|**adErrObjectOpen**|Операция запрещена, если объект является открытым. Не удалось открыть объект, который открыт. Поля не удается добавить открытый **записей**.|
|**3706**|**adErrProviderNotFound**|Не удалось найти поставщик. Он может быть неправильно установлен.<br /><br /> Имя поставщика может быть неправильно определен, указанный поставщик не может быть установлен на компьютере, где выполняется код, или установки может быть поврежден.|
|**3707**|**adErrBoundToCommand**|**ActiveConnection** свойство **записей** object, который содержит **команда** объекта в качестве источника, не может быть изменено. Приложение попыталось присвоить новый **подключения** объект **записей** с **команда** объект в качестве источника.|
|**3708**|**adErrInvalidParamInfo**|**Параметр** объект определен неправильно. Неправильные или неполные сведения отсутствуют.|
|**3709**|**adErrInvalidConnection**|Соединение не может использоваться для выполнения этой операции. Он является closed или недопустимым в данном контексте.|
|**3710**|**adErrNotReentrant**|Невозможно выполнить операцию во время обработки событий. Невозможно выполнить операцию обработчика событий, который запускает событие срабатывает еще раз. Например, методы навигации не должен вызываться изнутри **события WillMove** обработчик событий.|
|**3711**|**adErrStillExecuting**|Невозможно выполнить операцию во время выполнения в асинхронном режиме.|
|**3712**|**adErrOperationCancelled**|Операция была отменена пользователем. Приложение вызвало **CancelUpdate** или **CancelBatch** метод и текущая операция была отменена.|
|**3713**|**adErrStillConnecting**|Невозможно выполнить операцию при подключении асинхронно.|
|**3714**|**adErrInvalidTransaction**|Координацию транзакций является недопустимым или не запущена.|
|**3715**|**adErrNotExecuting**|Операция не может быть выполнена во время выполнения не.|
|**3716**|**adErrUnsafeOperation**|Параметры безопасности этого компьютера запрещают доступ к источнику данных в другом домене.|
|**3717**|**adWrnSecurityDialog**|Только для внутреннего применения. Не следует использовать. (Записи был включен для полноты картины. Эта ошибка не должны отображаться в коде.)|
|**3718**|**adWrnSecurityDialogHeader**|Только для внутреннего применения. Не следует использовать. (Запись включен для полноты картины. Эта ошибка не должны отображаться в коде.)|
|**3719**|**adErrIntegrityViolation**|Значение данных противоречит ограничениям целостности поля. Новое значение для **поле** вызовет повторяющийся ключ. Значения, которое образует одной стороны отношения между двумя записями не может быть обновляемым.|
|**3720**|**adErrPermissionDenied**|Недостаточно разрешений предотвращает запись поле. Пользователь с именем в строке подключения не имеет необходимых разрешений на запись в **поле**.|
|**3721**|**adErrDataOverflow**|Значение данных слишком велик для представления типа данных поля. Числовое значение, слишком велик для предполагаемой поля было назначено. Например значение типа long integer был назначен поле короткое целое число.|
|**3722**|**adErrSchemaViolation**|Конфликтует с типом данных или ограничения поля значения данных. Хранилище данных имеет ограничения проверки, которые отличаются от **поле** значение.|
|**3723**|**adErrSignMismatch**|Преобразование завершилось неудачей, так как значение данных было со знаком и тип данных поля, используемый поставщиком, беззнаковый.|
|**3724**|**adErrCantConvertvalue**|Данных значение не может быть преобразовано по причинам, отличным от несоответствия знаков и данные переполнения. Например преобразование будет усечение данных.|
|**3725**|**adErrCantCreate**|Значение данных нельзя задать или извлечь, потому что тип данных поля неизвестно, или поставщик было недостаточно ресурсов для выполнения операции.|
|**3726**|**adErrColumnNotOnThisRow**|Запись не содержит этого поля. Указано недопустимое имя неправильные поля или поля не в **поля** ссылка на коллекцию текущей записи.|
|**3727**|**adErrURLDoesNotExist**|URL-адрес источника или родительский URL-адрес назначения не существует. Нет опечатка в исходном или конечном URL-адрес. Возможно, `https://mysite/photo/myphoto.jpg` должно фактически Получив `https://mysite/photos/myphoto.jpg` вместо этого. Опечатка в родительский URL-адрес (в этом случае *photo* вместо *фотографии*) вызвала ошибку.|
|**3728**|**adErrTreePermissionDenied**|Недостаточно разрешений для доступа к дерева или поддерева. Пользователь, с именем в строке подключения не имеет соответствующих разрешений.|
|**3729**|**adErrInvalidURL**|URL-адрес содержит недопустимые символы. Убедитесь, что правильно ввели URL-адрес. URL-адрес соответствует схеме, зарегистрированные для текущего поставщика (например, Интернет публикации зарегистрирован поставщик для http).|
|**3730**|**adErrResourceLocked**|Объект, представленный указанным URL-адрес заблокирован одного или нескольких процессов. Подождите, пока процесс завершится и повторите операцию. Объект, который вы пытаетесь получить доступ к заблокирована другим пользователем или другим процессом, в приложении. Это чаще всего возникают в многопользовательской среде.|
|**3731**|**adErrResourceExists**|Невозможно выполнить операцию копирования. Объект с именем URL-адрес назначения уже существует. Укажите **adCopyOverwrite** объект. Если вы не укажете **adCopyOverwrite** при копировании файлов в каталоге, копирование завершается неудачно, при попытке скопировать элемент, который уже существует в месте назначения.|
|**3732**|**adErrCannotComplete**|Серверу не удается выполнить операцию. Возможно, сервер занят с другими операциями, или может быть недостаточно ресурсов.|
|**3733**|**adErrVolumeNotFound**|Поставщик не может найти устройство хранения, URL-адрес. Убедитесь, что правильно ввели URL-адрес. URL-адрес запоминающего устройства могут быть неправильными, но эта ошибка может возникать по другим причинам. Устройство может находиться вне сети или большой объем сетевого трафика может предотвратить подключение было нельзя сделать.|
|**3734**|**adErrOutOfSpace**|Невозможно выполнить операцию. Поставщик не может получить достаточно свободного дискового пространства. Существует может оказаться недостаточно оперативной памяти или дискового пространства для хранения временных файлов на сервере.|
|**3735**|**adErrResourceOutOfScope**|URL-адрес источника или назначения выходит за пределы текущей записи.|
|**3736**|**adErrUnavailable**|Не удалось завершить операцию, и состояние недоступна. Поле может быть недоступен или операция не выполнялась. Другой пользователь может измененных или удаленных поле, которое вы пытаетесь получить доступ.|
|**3737**|**adErrURLNamedRowDoesNotExist**|Запись с таким именем не существует. При попытке открыть файл с помощью **записи** объекта, имя файла или путь к файлу был с ошибками.|
|**3738**|**adErrDelResOutOfScope**|URL-адрес объекта для удаления выходит за пределы текущей записи.|
|**3747**|**adErrCatalogNotSet**|Операция требует действительного **ParentCatalog**.|
|**3748**|**adErrCantChangeConnection**|Подключение было запрещено. Новое соединение, которое вы запросили имеет разные характеристики, отличный от уже заняты.|
|**3749**|**adErrFieldsUpdateFailed**|Не удалось обновить поля. Для получения дополнительных сведений проверьте **состояние** свойство объектов отдельного поля. Эта ошибка может возникать в двух случаях: при изменении **поле** значение объекта, изменение или Добавление записи в базе данных; и при изменении свойств **поле** сам объект.<br /><br /> **Записи** или **записей** не удалось обновить из-за проблемы с одним из полей в текущей записи. Перечислить **поля** сбора и проверки **состояние** свойства каждого поля, чтобы определить причину проблемы.|
|**3750**|**adErrDenyNotSupported**|Поставщик не поддерживает ограничения управления доступом. Была предпринята попытка ограничить общий доступ к файлам и ваш поставщик не поддерживает концепцию.|
|**3751**|**adErrDenyTypeNotSupported**|Поставщик не поддерживает запрошенный тип управления доступом для ограниченного использования программ. Была предпринята попытка установить общий доступ к файлам определенного типа ограничение, которое не поддерживается поставщиком. См. в разделе документации по поставщику, чтобы определить, какие ограничения доступа к файлам, поддерживаются.|
