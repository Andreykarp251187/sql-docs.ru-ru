# <a name="appendix-c-programming-with-ado-in-development-environments"></a>Приложение В. Программирование с использованием ADO в средах разработки
ADO — это компонент интерфейс автоматизации COM, который может использоваться с многих языков программирования, включая Microsoft Visual Basic, VBScript, JScript и Visual C++. ADO версии с каждым из этих средств и других приложений, таких как Microsoft Office и Microsoft SQL Server.

 Библиотека для ADO msado15.dll и префикс идентификатор (ProgID) программы «ADODB.» Например, для явной ссылки ADO [записей](../../../ado/reference/ado-api/recordset-object-ado.md), используйте `ADODB.Recordset`.

 Дополнительные сведения о программировании с использованием ADO в различных средах разработки см. в разделах:

-   [Использование объектов ADO с Microsoft Visual Basic](../../../ado/guide/appendixes/using-ado-with-microsoft-visual-basic.md)

-   [Использование объектов ADO с языками сценариев](../../../ado/guide/appendixes/using-ado-with-scripting-languages.md)

-   [Использование объектов ADO с Microsoft Visual C++](../../../ado/guide/appendixes/using-ado-with-microsoft-visual-c.md)
