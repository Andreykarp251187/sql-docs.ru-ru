# <a name="internet-explorer-error-codes-for-remote-data-service"></a>Коды ошибки Internet Explorer для службы удаленных данных
В следующей таблице перечислены коды ошибок Microsoft® Internet Explorer, связанные с использованием службы удаленных данных. Показан положительный десятический перевод низких двух байтов, отрицательный десятический перевод полного кода ошибки и гексадецимальные значения.

|Ошибки Internet Explorer (Wininet)|Number|Описание|
|------------------------------------------|------------|-----------------|
|**IDS_WinInet_CantConnect**|8195 -2146820093 0x800A2003|Ошибка интернет-клиента: не может подключиться к серверу.|
|**IDS_WinInet_ ConnectionReset**|12031 -2146816257 0x800A2EFF|Ошибка интернет-клиента: перезагрузка подключения.|
|**IDS_WinInet_Error**|8193 -2146820095 0x800A2001|Ошибка интернет-клиента.|
|**IDS_WinInet_ НедействительноеОтвет Сервера**|8430 -2146819858 0x800A20EE|Ошибка интернет-клиента: Недействительный ответ сервера.|
|**IDS_WinInet_ SSLPostLimitation**|8196 -2146820092 0x800A2004|Ошибка интернет-клиента: ошибка TLS/SSL (возможно ограничение загрузки данных 32K).|
|**IDS_WinInet_Timeout**|8194 -2146820094 0x800A2002|Ошибка интернет-клиента: Запрос тайм-аута.|
