# <a name="stringformatenum"></a>StringFormatEnum
Задает формат при извлечении [набора записей](../../../ado/reference/ado-api/recordset-object-ado.md) в виде строки.  
  
|Константа|Значение|Описание|  
|--------------|-----------|-----------------|  
|**adClipString**|2|Отделяет строки от *RowDelimiter*, Columns по *ColumnDelimiter*и значения NULL значениями *нуллекспр*. Эти три параметра метода [GetString](../../../ado/reference/ado-api/getstring-method-ado.md) допустимы только с *StringFormat* **адклипстринг**.|
