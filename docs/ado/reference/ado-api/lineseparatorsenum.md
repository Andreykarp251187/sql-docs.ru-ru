# <a name="lineseparatorsenum"></a>LineSeparatorsEnum
Определяет символ, используемый в качестве разделителя строки в тексте [Stream](../../../ado/reference/ado-api/stream-object-ado.md) объектов.  
  
|Константа|Значение|Описание|  
|--------------|-----------|-----------------|  
|**adCR**|13|Указывает символ возврата каретки.|  
|**adCRLF**|-1|По умолчанию. Указывает строку возврата каретки.|  
|**adLF**|10|Указывает, перевод строки.|  
  
## <a name="adowfc-equivalent"></a>Эквивалент ADO и WFC  
 Эти константы не имеют эквивалентов ADO и WFC.
