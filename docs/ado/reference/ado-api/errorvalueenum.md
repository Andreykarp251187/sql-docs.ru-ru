# <a name="errorvalueenum"></a>ErrorValueEnum
Указывает тип ошибки времени выполнения ADO.  
  
 Перечислены три формы номер ошибки.  
  
-   Положительное десятичной низкой два байта полного числа в десятичном формате. Это число отображается в Visual Basic по умолчанию появления сообщения об ошибке. Например, ошибка времени выполнения "3707".  
  
-   Отрицательное значение десятичной десятичное преобразование числа ошибку переполнения.  
  
-   Шестнадцатеричный шестнадцатеричное представление числа ошибку переполнения. Код устройства Windows находится в четвертом цифра. Код устройства для номера ошибок ADO — *объект*. Пример: 0x800***A***0E7B.  
  
> [!NOTE]
>  Приложения ADO могут передаваться ошибок OLE DB. Как правило, они могут идентифицироваться по код устройства Windows из *4*. Например, 0x800***4***.
