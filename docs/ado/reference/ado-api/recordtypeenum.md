# <a name="recordtypeenum"></a>RecordTypeEnum
Указывает тип объекта [записи](../../../ado/reference/ado-api/record-object-ado.md) .  
  
|Постоянно|Значение|Description|  
|--------------|-----------|-----------------|  
|**адсимплерекорд**|0|Указывает на *простую* запись (не содержит дочерних узлов).|  
|**адколлектионрекорд**|1|Указывает запись *коллекции* (содержит дочерние узлы).|  
|**адрекордункновн**|-1|Указывает, что тип этой **записи** неизвестен.|  
|**адструктдок**|2|Указывает Специальный тип записи *коллекции* , представляющий структурированные документы com.|  
  
## <a name="adowfc-equivalent"></a>Эквивалент ADO/WFC  
 Эти константы не имеют эквивалентов ADO/WFC.
