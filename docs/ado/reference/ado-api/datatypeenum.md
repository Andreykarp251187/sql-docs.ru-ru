# <a name="datatypeenum"></a>DataTypeEnum
Указывает тип данных [поля](../../../ado/reference/ado-api/field-object.md), [параметра](../../../ado/reference/ado-api/parameter-object.md)или [Свойства](../../../ado/reference/ado-api/property-object-ado.md). Соответствующий индикатор типа OLE DB отображается в круглых скобках в столбце Описание следующей таблицы.  
  
|Постоянно|Значение|Description|  
|--------------|-----------|-----------------|  
|**адаррай**|0x2000|Значение флага, которое всегда объединяется с другой константой типа данных, которая указывает на массив другого типа данных. Не применяется к ADOX.|  
|**адбигинт**|20|Указывает 8-байтовое целое число со знаком (DBTYPE_I8).|  
|**адбинари**|128|Указывает двоичное значение (DBTYPE_BYTES).|  
|**адбулеан**|11|Указывает **логическое** значение (DBTYPE_BOOL).|  
|**адбстр**|8|Указывает строку символов (Юникод), завершающуюся нулем (DBTYPE_BSTR).|  
|**адчаптер**|136|Указывает 4-байтовое значение главы, определяющее строки в дочернем наборе строк (DBTYPE_HCHAPTER).|  
|**адчар**|129|Указывает строковое значение (DBTYPE_STR).|  
|**адкурренци**|6|Указывает значение валюты (DBTYPE_CY). Currency — это число с фиксированной запятой с четырьмя цифрами справа от десятичной запятой. Он хранится в 8-байтном целом число со знаком, масштабированном на 10 000.|  
|**аддате**|7|Указывает значение даты (DBTYPE_DATE). Дата хранится в виде Double, целой части, равной количеству дней с 30 декабря 1899, и дробной части, которая является долей дня.|  
|**аддбдате**|133|Указывает значение даты (ГГГГММДД) (DBTYPE_DBDATE).|  
|**аддбтиме**|134|Указывает значение времени (ЧЧММСС) (DBTYPE_DBTIME).|  
|**аддбтиместамп**|135|Указывает отметку даты и времени (YYYYMMDDHHMMSS плюс часть в биллионсс) (DBTYPE_DBTIMESTAMP).|  
|**аддеЦимал**|14|Указывает точное числовое значение с фиксированной точностью и масштабом (DBTYPE_DECIMAL).|  
|**аддаубле**|5|Указывает значение с плавающей точкой двойной точности (DBTYPE_R8).|  
|**адемпти**|0|Не указывает значение (DBTYPE_EMPTY).|  
|**адеррор**|10|Указывает 32-разрядный код ошибки (DBTYPE_ERROR).|  
|**адфилетиме**|64|Указывает 64-разрядное значение, представляющее число 100-наносекундных интервалов с 1 января 1601 (DBTYPE_FILETIME).|  
|**адгуид**|72|Указывает глобальный уникальный идентификатор (GUID) (DBTYPE_GUID).|  
|**адидиспатч**|9|Указывает указатель на интерфейс **IDispatch** для COM-объекта (DBTYPE_IDISPATCH).<br /><br /> **Примечание** . Этот тип данных в настоящее время не поддерживается ADO. Использование может привести к непредсказуемым результатам.|  
|**адинтежер**|3|Указывает 4-байтовое целое число со знаком (DBTYPE_I4).|  
|**адиункновн**|13|Указывает указатель на интерфейс **IUnknown** для COM-объекта (DBTYPE_IUNKNOWN).<br /><br /> **Примечание** . Этот тип данных в настоящее время не поддерживается ADO. Использование может привести к непредсказуемым результатам.|  
|**адлонгварбинари**|205|Указывает на Длинное двоичное значение.|  
|**адлонгварчар**|201|Указывает длинное строковое значение.|  
|**адлонгварвчар**|203|Указывает длинное строковое значение Юникода, завершающееся нулем.|  
|**аднумерик**|131|Указывает точное числовое значение с фиксированной точностью и масштабом (DBTYPE_NUMERIC).|  
|**адпропвариант**|138|Указывает ПРОПВАРИАНТ автоматизации (DBTYPE_PROP_VARIANT).|  
|**адсингле**|4|Указывает значение с плавающей точкой одиночной точности (DBTYPE_R4).|  
|**адсмаллинт**|2|Указывает двухбайтовое целое число со знаком (DBTYPE_I2).|  
|**адтининт**|16|Указывает Однобайтовое целое число со знаком (DBTYPE_I1).|  
|**адунсигнедбигинт**|21|Указывает целое число без знака размером 8 байт (DBTYPE_UI8).|  
|**адунсигнединт**|19|Указывает целое число без знака длиной 4 байта (DBTYPE_UI4).|  
|**адунсигнедсмаллинт**|18|Указывает двухбайтовое целое число без знака (DBTYPE_UI2).|  
|**адунсигнедтининт**|17|Указывает Однобайтовое целое число без знака (DBTYPE_UI1).|  
|**адусердефинед**|132|Указывает определяемую пользователем переменную (DBTYPE_UDT).|  
|**адварбинари**|204|Указывает двоичное значение.|  
|**адварчар**|200|Указывает строковое значение.|  
|**адвариант**|12|Указывает **вариант** автоматизации (DBTYPE_VARIANT).<br /><br /> **Примечание** . Этот тип данных в настоящее время не поддерживается ADO. Использование может привести к непредсказуемым результатам.|  
|**адварнумерик**|139|Указывает числовое значение.|  
|**адварвчар**|202|Указывает строку символов в Юникоде, завершающуюся нулем.|  
|**адвчар**|130|Указывает строку символов в Юникоде, завершающуюся нулем (DBTYPE_WSTR).|
