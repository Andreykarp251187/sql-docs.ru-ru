# <a name="fieldstatusenum"></a>FieldStatusEnum
Указывает [состояние](../../../ado/reference/ado-api/status-property-ado-field.md) из [объект поля](../../../ado/reference/ado-api/field-object.md).  
  
 **AdFieldPending\***  значения указывают операции, который вызвал состояние, которое необходимо настроить и могут объединяться с другими значениями состояния.  
  
|Константа|Значение|Описание|  
|--------------|-----------|-----------------|  
|**adFieldAlreadyExists**|26|Указывает, что указанное поле уже существует.|  
|**adFieldBadStatus**|12|Указывает, что значение недопустимое состояние было отправлено от ADO к поставщику OLE DB. Возможные причины включают OLE DB 1.0 или 1.1 поставщика или неправильное сочетание [значение](../../../ado/reference/ado-api/value-property-ado.md) и [состояние](../../../ado/reference/ado-api/status-property-ado-field.md).|  
|**adFieldCannotComplete**|20|Указывает, что URL-адреса сервера, указанный в параметре [источника](../../../ado/reference/ado-api/source-property-ado-record.md) не удалось выполнить операцию.|  
|**adFieldCannotDeleteSource**|23|Указывает, что во время операции перемещения, дерева или поддерева был перемещен в новое расположение, но не удалось удалить источник.|  
|**adFieldCantConvertValue**|2|Указывает, что поле не может получить или хранить их без потерь данных.|  
|**adFieldCantCreate**|7|Указывает, что поля не могут быть добавлены, так как поставщик превысило ограничение (например, допустимое число полей).|  
|**adFieldDataOverflow**|6|Указывает, что данные, возвращенные из поставщика переполняет тип данных поля.|  
|**adFieldDefault**|13|Указывает, что было использовано значение по умолчанию для поля, при вводе данных.|  
|**adFieldDoesNotExist**|16|Указывает, что указанное поле не существует.|  
|**adFieldIgnore**|15|Указывает, что это поле был пропущен при данные параметров значений в источнике. Поставщик не значение.|  
|**adFieldIntegrityViolation**|10|Указывает, что поле не может изменяться, так как это вычисленное или производное сущности.|  
|**adFieldInvalidURL**|17|Указывает, что URL-адрес источника данных содержит недопустимые символы.|  
|**adFieldIsNull**|3|Указывает, что поставщик возвратил значение VARIANT типа VT_NULL и поле не заполнено.|  
|**adFieldOK**|0|По умолчанию. Указывает, что поле был успешно добавлен или удален.|  
|**adFieldOutOfSpace**|22|Указывает, что поставщик не удается получить достаточно места для завершения перемещения или копирования операции.|  
|**adFieldPendingChange**|0x40000|Указывает, либо, поля были удалены, а затем повторно добавить, возможно с другой тип данных, или значение поля, которое ранее было состояние из **adFieldOK** был изменен. Измените конечную форму поля [поля](../../../ado/reference/ado-api/fields-collection-ado.md) коллекцию после [обновления](../../../ado/reference/ado-api/update-method.md) вызывается метод.|  
|**adFieldPendingDelete**|0x20000|Указывает, что **удалить** операция вызвала заданное состояние. Поле была помечена для удаления из **поля** коллекцию после **обновления** вызывается метод.|  
|**adFieldPendingInsert**|0x10000|Указывает, что **Append** операция вызвала заданное состояние. **Поле** была помечена для добавления **поля** коллекцию после **обновления** вызывается метод.|  
|**adFieldPendingUnknown**|0x80000|Указывает, что поставщик может определить, какое состояние поля Операция вызвала для задания.|  
|**adFieldPendingUnknownDelete**|0x100000|Указывает, что поставщик не может определить, какая операция вызвала состояние поля для задания, и что поля будут удалены из **поля** коллекцию после **обновления** вызывается метод.|  
|**adFieldPermissionDenied**|9|Указывает, что поле не может изменяться, поскольку он был определен только для чтения.|  
|**adFieldReadOnly**|24|Указывает, что поле в источнике данных определен только для чтения.|  
|**adFieldResourceExists**|19|Указывает, что поставщик не удалось выполнить операцию, поскольку объект уже существует в URL-адрес назначения и не может перезаписать объект.|  
|**adFieldResourceLocked**|18|Указывает, что поставщик не удалось выполнить операцию, поскольку источник данных заблокирован другие приложения или процесса.|  
|**adFieldResourceOutOfScope**|25|Указывает, что URL-адрес источника или назначения выходит за пределы текущей записи.|  
|**adFieldSchemaViolation**|11|Указывает, что значение нарушает ограничение схемы источника данных для поля.|  
|**adFieldSignMismatch**|5|Указывает, что поставщик вернул значение данных было со знаком, но беззнаковый тип данных значения поля ADO.|  
|**adFieldTruncated**|4|Указывает, что данные переменной длины были усечены при чтении из источника данных.|  
|**adFieldUnavailable**|8|Указывает, что поставщик не удалось определить значение, при чтении из источника данных. Например строке был только что создан, значение по умолчанию для столбца не была доступна и новое значение было еще не было определено.|  
|**adFieldVolumeNotFound**|21|Указывает, что поставщик не удалось найти том хранилища, URL-адрес.|  
  
## <a name="adowfc-equivalent"></a>Эквивалент ADO и WFC  
 Эти константы не имеют эквивалентов ADO и WFC.
