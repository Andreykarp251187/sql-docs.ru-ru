# <a name="rds-objects"></a>Объекты службы удаленных рабочих столов
> [!IMPORTANT]
>  Начиная с Windows 8 и Windows Server 2012, серверные компоненты служб удаленных рабочих СТОЛОВ, больше не включаются в операционной системе Windows (см. в разделе Windows 8 и [настольная книга по совместимости Windows Server 2012](https://www.microsoft.com/download/details.aspx?id=27416) для получения дополнительных сведений). Клиентские компоненты служб удаленных рабочих СТОЛОВ будет поддерживаться в будущих версиях Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Приложения, использующие служб удаленных рабочих СТОЛОВ, следует перевести [WCF-сервиса данных](https://go.microsoft.com/fwlink/?LinkId=199565).  
  
|||  
|-|-|  
|[DataControl (служба удаленных рабочих СТОЛОВ)](../../../ado/reference/rds-api/datacontrol-object-rds.md)|Привязывает запрос данных **записей** на один или несколько элементов управления (например, текстовое поле, элемент управления DataGrid или поле со списком) для отображения **записей** данных на веб-странице.<br /><br /> **DataControl** объект безопасен для скриптов.|  
|[DataFactory (RDSServer)](../../../ado/reference/rds-api/datafactory-object-rdsserver.md)|Реализует методы, которые обеспечивают доступ к данным на чтение и запись к указаны источники данных для клиентских приложений.<br /><br /> **DataFactory** объект не является безопасным для использования в сценариях.|  
|[DataSpace (служба удаленных рабочих СТОЛОВ)](../../../ado/reference/rds-api/dataspace-object-rds.md)|Создает клиентский прокси для пользовательских бизнес-объектов, расположенных на среднем уровне.<br /><br /> **DataSpace** объект безопасен для скриптов.|  
|[Интерфейс IRDSService (служба удаленных рабочих столов)](../../../ado/reference/rds-api/irdsservice-interface-rds.md)|Предоставляет [InvokeService (служба удаленных рабочих СТОЛОВ)](../../../ado/reference/rds-api/invokeservice-rds.md) метод, который используется для возврата указателя на запрошенный интерфейс на более мощным версии объекта.|
