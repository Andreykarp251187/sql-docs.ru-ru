# <a name="ruleenum"></a>RuleEnum
Указывает правило, когда выполните [ключ](../../../ado/reference/adox-api/key-object-adox.md) удаляется.  
  
|Константа|Значение|Описание|  
|--------------|-----------|-----------------|  
|**adRICascade**|1|Для каскадных изменений.|  
|**adRINone**|0|По умолчанию. Не выполняется никаких действий.|  
|**adRISetDefault**|3|Значение внешнего ключа присваивается значение по умолчанию.|  
|**adRISetNull**|2|Значение внешнего ключа задано значение null.|
