# <a name="keytypeenum"></a>KeyTypeEnum
Указывает тип [ключа](../../../ado/reference/adox-api/key-object-adox.md): первичный, внешний или уникальный.  
  
|Постоянно|Значение|Description|  
|--------------|-----------|-----------------|  
|**адкэйпримари**|1|По умолчанию. Ключ является первичным ключом.|  
|**адкэйфореигн**|2|Ключ является внешним ключом.|  
|**адкэйуникуе**|3|Ключ уникален.|  
  
## <a name="applies-to"></a>Применяется к  
 [Свойство Property (Key) (ADOX)](../../../ado/reference/adox-api/type-property-key-adox.md)
