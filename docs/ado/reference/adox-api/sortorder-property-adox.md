# <a name="sortorder-property-adox"></a>Свойство SortOrder (ADOX)
Указывает последовательность сортировки для столбца (только столбцы индекса).  
  
## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения  
 Задает и возвращает **Long** значение, которое может принимать одно из [SortOrderEnum](../../../ado/reference/adox-api/sortorderenum.md) константы. Значение по умолчанию — **adSortAscending**.  
  
## <a name="remarks"></a>Примечания  
 Это свойство применяется только к [столбец](../../../ado/reference/adox-api/column-object-adox.md) объекты в [столбцы](../../../ado/reference/adox-api/columns-collection-adox.md) коллекцию [индекс](../../../ado/reference/adox-api/index-object-adox.md).  
  
## <a name="applies-to"></a>Объект применения  
 [Объект Column (ADOX)](../../../ado/reference/adox-api/column-object-adox.md)
