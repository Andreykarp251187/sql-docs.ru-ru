# <a name="sortorderenum"></a>SortOrderEnum
Задает последовательность сортировки для индексированного столбца.  
  
|Постоянно|Значение|Description|  
|--------------|-----------|-----------------|  
|**адсортасцендинг**|1|По умолчанию. Последовательность сортировки для столбца — по возрастанию.|  
|**адсортдесцендинг**|2|Последовательность сортировки для столбца — по убыванию.|  
  
## <a name="applies-to"></a>Применяется к
