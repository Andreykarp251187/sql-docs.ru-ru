# <a name="definedsize-property-adox"></a>Свойство DefinedSize (ADOX)
Указывает максимальный размер указанного столбца.  
  
## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения  
 Задает и возвращает **Long** значение, которое определяет его максимальную длину в символах значений данных.  
  
## <a name="remarks"></a>Примечания  
 Значение по умолчанию равно нулю (0).  
  
 Это свойство доступно только для чтения для [столбец](../../../ado/reference/adox-api/column-object-adox.md) объектов, уже добавлен в коллекцию.
