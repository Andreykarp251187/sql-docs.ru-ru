# <a name="indexnulls-property-adox"></a>Свойство IndexNulls (ADOX)
Указывает, имеют ли записи индекса элементы, имеющие значения NULL в полях индекса.  
  
## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения  
 Задает и возвращает значение [алловнуллсенум](../../../ado/reference/adox-api/allownullsenum.md) . Значение по умолчанию — **адиндекснуллсдисаллов**.  
  
## <a name="remarks"></a>Remarks  
 Это свойство доступно только для чтения в объектах [индекса](../../../ado/reference/adox-api/index-object-adox.md) , уже добавленных в коллекцию.
