# <a name="membertypeenum"></a>MemberTypeEnum
Задает параметр [тип](../../../ado/reference/ado-md-api/type-property-ado-md.md) свойство [член](../../../ado/reference/ado-md-api/member-object-ado-md.md) объекта.  
  
|Константа|Значение|Описание|  
|--------------|-----------|-----------------|  
|**adMemberAll**|4|Указывает, что **член** представляет все элементы уровня.|  
|**adMemberFormula**|3|Указывает, что **член** объекта вычисляется с помощью формул выражения.|  
|**adMemberMeasure**|2|Указывает, что **член** объект принадлежит измерения Measures и представляет количественный атрибут.|  
|**adMemberRegular**|1|По умолчанию. Указывает, что **член** объект представляет экземпляр бизнес-сущности.|  
|**adMemberUnknown**|0|Не удается определить тип элемента.|
