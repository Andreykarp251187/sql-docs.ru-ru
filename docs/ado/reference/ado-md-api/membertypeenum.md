# <a name="membertypeenum"></a>MemberTypeEnum
Задает параметр для свойства [Type](../../../ado/reference/ado-md-api/type-property-ado-md.md) объекта- [члена](../../../ado/reference/ado-md-api/member-object-ado-md.md) .  
  
|Постоянно|Значение|Description|  
|--------------|-----------|-----------------|  
|**адмембералл**|4|Указывает, что объект **member** представляет все элементы уровня.|  
|**адмемберформула**|3|Указывает, что объект- **член** вычисляется с помощью выражения формулы.|  
|**адмембермеасуре**|2|Указывает, что объект- **член** принадлежит измерению Measures и представляет количественный атрибут.|  
|**адмемберрегулар**|1|По умолчанию. Указывает, что объект **member** представляет экземпляр бизнес-сущности.|  
|**адмемберункновн**|0|Не удается определить тип элемента.|
