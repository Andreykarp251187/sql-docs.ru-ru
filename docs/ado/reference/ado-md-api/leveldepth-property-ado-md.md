# <a name="leveldepth-property-ado-md"></a>Свойство LevelDepth (многомерные объекты ADO)
Указывает количество уровней между корень иерархии и [член](../../../ado/reference/ado-md-api/member-object-ado-md.md).  
  
## <a name="return-values"></a>Возвращаемые значения  
 Возвращает **Long** целое число и только для чтения.  
  
## <a name="remarks"></a>Примечания  
 Используйте **LevelDepth** свойства, чтобы определить расстояние от [член](../../../ado/reference/ado-md-api/member-object-ado-md.md)объекта корневого уровня иерархии. **LevelDepth**элемента на корневой уровень равен 0. Это соответствует [глубина](../../../ado/reference/ado-md-api/depth-property-ado-md.md) свойство [уровень](../../../ado/reference/ado-md-api/level-object-ado-md.md) объекта.
