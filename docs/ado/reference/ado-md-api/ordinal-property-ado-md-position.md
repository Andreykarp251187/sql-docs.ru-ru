# <a name="ordinal-property-ado-md-position"></a>Свойство Ordinal (многомерный объект ADO Position)
Уникально идентифицирует [позиции](../../../ado/reference/ado-md-api/position-object-ado-md.md) вдоль оси.  
  
## <a name="return-values"></a>Возвращаемые значения  
 Возвращает **Long** целое число и доступен только для чтения.  
  
## <a name="remarks"></a>Примечания  
 **Порядковый номер** свойство [позиции](../../../ado/reference/ado-md-api/position-object-ado-md.md) соответствует индекс **позиции** в [позиций](../../../ado/reference/ado-md-api/positions-collection-ado-md.md) коллекции.  
  
 Ячейки можно быстро получить с помощью **порядковый номер** из **позиции** каждой оси с [элемент](../../../ado/reference/ado-md-api/item-property-ado-md-cellset.md) свойство [набора ячеек](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) объекта .  
  
## <a name="applies-to"></a>Объект применения  
 [Объект Position (многомерные объекты ADO)](../../../ado/reference/ado-md-api/position-object-ado-md.md)
