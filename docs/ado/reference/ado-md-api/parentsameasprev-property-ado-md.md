# <a name="parentsameasprev-property-ado-md"></a>Свойство ParentSameAsPrev (многомерные объекты ADO)
Указывает, совпадает ли родительский элемент данного [элемента](../../../ado/reference/ado-md-api/member-object-ado-md.md) позиционирования с родительским элементом непосредственно предыдущего элемента.  
  
## <a name="return-values"></a>Возвращаемые значения  
 Возвращает **логическое** значение и доступно только для чтения.  
  
## <a name="remarks"></a>Remarks  
 Это свойство поддерживается только для объектов [member](../../../ado/reference/ado-md-api/member-object-ado-md.md) , принадлежащих объекту- [положению](../../../ado/reference/ado-md-api/position-object-ado-md.md) . Ошибка возникает, когда на это свойство ссылаются объекты- **члены** , принадлежащие объекту [уровня](../../../ado/reference/ado-md-api/level-object-ado-md.md) .  
  
## <a name="applies-to"></a>Применяется к  
 [Объект Member (многомерные объекты ADO)](../../../ado/reference/ado-md-api/member-object-ado-md.md)
