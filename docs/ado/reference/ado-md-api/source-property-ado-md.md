# <a name="source-property-ado-md"></a>Свойство Source (многомерные объекты ADO)
Указывает источник данных в [cellset](../../../ado/reference/ado-md-api/cellset-object-ado-md.md).  
  
## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения  
 Возвращает или задает **Variant**, и доступно для чтения/записи для закрыто [набора ячеек](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) объектов и только для чтения для открытой **набора ячеек** объектов. **Variant** должен содержать допустимый **строка**, например, запрос многомерных Выражений.
