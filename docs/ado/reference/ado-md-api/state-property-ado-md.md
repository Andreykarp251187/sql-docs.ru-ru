# <a name="state-property-ado-md"></a>Свойство State (многомерные объекты ADO)
Указывает текущее состояние набора ячеек.  
  
## <a name="return-values"></a>Возвращаемые значения  
 Возвращает **длинное** целое число, указывающее текущее условие объекта набора [ячеек](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) и доступно только для чтения. Допустимы следующие значения: **адстатеклосед** (0) и **адстатеопен** (1).  
  
## <a name="remarks"></a>Примечания  
 Чтобы использовать имена констант [обжектстатинум](../../../ado/reference/ado-api/objectstateenum.md) , необходимо, чтобы в проекте была ссылка на БИБЛИОТЕКУ типов ADO. Дополнительные сведения см. в разделе [Использование ADO с объекты данных ActiveX (MD)](../../../ado/guide/multidimensional/using-ado-with-ado-md.md) .
