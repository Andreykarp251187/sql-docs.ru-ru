# <a name="sql-data-warehouse-stored-procedures"></a>Хранимые процедуры хранилища данных SQL

  [!INCLUDE[ssSDW](../../includes/sssdw-md.md)] предоставляет встроенные процедуры, которые можно использовать для выполнения операций, которые относятся к ролям базы данных. [!INCLUDE[ssSDW](../../includes/sssdw-md.md)] включает в себя следующие системные процедуры:  
  
##  <a name="AggregateFunctions"></a> [sp_datatype_info_90 &#40;хранилище данных SQL&#41;](../../relational-databases/system-stored-procedures/sp-datatype-info-90-sql-data-warehouse.md)  
  
 [sp_pdw_add_network_credentials &#40;хранилище данных SQL&#41;](../../relational-databases/system-stored-procedures/sp-pdw-add-network-credentials-sql-data-warehouse.md)  
  
 [sp_pdw_database_encryption &#40;хранилище данных SQL&#41;](../../relational-databases/system-stored-procedures/sp-pdw-database-encryption-sql-data-warehouse.md)  
  
 [sp_pdw_database_encryption_regenerate_system_keys &#40;хранилище данных SQL&#41;](../../relational-databases/system-stored-procedures/sp-pdw-database-encryption-regenerate-system-keys-sql-data-warehouse.md)  
  
 [sp_pdw_log_user_data_masking &#40;хранилище данных SQL&#41;](../../relational-databases/system-stored-procedures/sp-pdw-log-user-data-masking-sql-data-warehouse.md)

 [sp_special_columns_100 &#40;хранилище данных SQL&#41;](../../relational-databases/system-stored-procedures/sp-special-columns-100-sql-data-warehouse.md)  
  
> [!NOTE]  
>  Некоторые дополнительные системные хранимые процедуры используются только в пределах экземпляра [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] или через клиентские API-интерфейсов и не предназначены для общего использования. Эти процедуры перечислены в [системные хранимые процедуры (Transact-SQL)](https://msdn.microsoft.com/library/ms187961.aspx). Эти процедуры могут быть изменены и совместимость не гарантируется. Все процедуры в списке не доступны в [!INCLUDE[ssSDW](../../includes/sssdw-md.md)].
