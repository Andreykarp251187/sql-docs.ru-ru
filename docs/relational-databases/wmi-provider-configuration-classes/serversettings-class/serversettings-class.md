# <a name="serversettings-class"></a>Класс ServerSettings
[!INCLUDE [SQL Server](../../../includes/applies-to-version/sqlserver.md)]
  [Класс класс serversettings](../../../relational-databases/wmi-provider-configuration-classes/serversettings-class/serversettings-class.md) представляет параметры сервера в экземпляре [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] .  
  
 [Класс класс serversettings](../../../relational-databases/wmi-provider-configuration-classes/serversettings-class/serversettings-class.md) позволяет выполнять следующие задачи:  
  
-   получать доступ к общим флагам настройки экземпляра [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)];  
  
-   задавать значения по умолчанию для экземпляра [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)];  
  
-   задавать сертификат безопасности для экземпляра [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].
