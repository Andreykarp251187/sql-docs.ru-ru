# <a name="sinstance-class"></a>Класс SInstance
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  Класс [SInstance Class](../../../relational-databases/wmi-provider-configuration-classes/sinstance-class/sinstance-class.md) представляет экземпляр [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 Класс [SInstance Class](../../../relational-databases/wmi-provider-configuration-classes/sinstance-class/sinstance-class.md) можно использовать для управления протоколами и сетевыми библиотеками, связанными с экземпляром [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]. Для экземпляра [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]можно также задать сертификат безопасности.
