## <a name="details"></a>Сведения  
  
| attribute | Значение |  
| :-------- | :---- |  
|Название продукта|SQL Server|  
|Идентификатор события|12307|  
|Источник события|MSSQLSERVER|  
|Компонент|SQLEngine|  
|Символическое имя|HK_UNSUPPORTED_PARAM_DEFAULT|  
|Текст сообщения|Значения по умолчанию для параметров в "*конструкция*" должны быть константами.|
