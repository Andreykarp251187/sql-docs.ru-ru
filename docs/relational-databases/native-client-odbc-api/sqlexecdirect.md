# <a name="sqlexecdirect"></a>SQLExecDirect
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  Если атрибут оператора SQL_SOPT_SS_PARAM_FOCUS не 0, sLExecDirect вернет SQL_ERROR и создаст диагностическую запись с помощью S'LSTATE-HY024 и сообщения "Недействительное значение атрибута, SQL_SOPT_SS_PARAM_FOCUS (должно быть нулевым во время выполнения)". Дополнительные сведения об атрибуте SQL_SOPT_SS_PARAM_FOCUS см. в разделе [SQLSetStmtAttr](../../relational-databases/native-client-odbc-api/sqlsetstmtattr.md).
