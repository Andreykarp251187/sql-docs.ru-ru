# <a name="sqlnumparams"></a>SQLNumParams
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

  Улучшения в движке [!INCLUDE[ssSQL11](../../includes/sssql11-md.md)] базы данных, начиная с позволяют S'LNumParams получить более точное описание ожидаемых результатов. Эти более точные результаты могут отличаться от значений, [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]возвращенных S'LNumParams в предыдущих версиях .  
  
 Дополнительные сведения см. в разделе [Обнаружение метаданных](../../relational-databases/native-client/features/metadata-discovery.md).
