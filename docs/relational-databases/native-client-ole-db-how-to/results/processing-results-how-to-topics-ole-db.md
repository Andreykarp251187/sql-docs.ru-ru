# <a name="processing-results-how-to-topics-ole-db"></a>Инструкции по обработке результатов (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
