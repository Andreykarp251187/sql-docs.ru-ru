# <a name="database-mail-views-transact-sql"></a>Представления компонента Database Mail (Transact-SQL)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
