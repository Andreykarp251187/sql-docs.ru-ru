# <a name="detect-failed-input-and-output-requests"></a>Detect Failed Input and Output Requests
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
