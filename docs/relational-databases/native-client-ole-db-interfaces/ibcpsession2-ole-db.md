# <a name="ibcpsession2-native-client-ole-db-provider"></a>Метода IBCPSession2 (поставщик собственного клиента OLE DB)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]

  Интерфейс IBCPSession2 является расширением интерфейса IBCPSession, который предоставляет функцию-элемент, что является альтернативой вызову интерфейса IBCPSession::BCPColFmt для каждого столбца.  Метода IBCPSession2 наследует от IBCPSession и добавляет один новый метод: [метода IBCPSession2:: BCPSetBulkMode](../../relational-databases/native-client-ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md).
