# <a name="license-terms"></a>Условия лицензии
  На странице «Условия лицензии» принимаются условия лицензии для этого выпуска.  
  
## <a name="options"></a>Варианты  
 Чтобы продолжить установку, необходимо принять условия лицензии [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
 Нажмите кнопку **Печать** , чтобы распечатать копию условий лицензии для этой версии. Чтобы завершить программу установки, нажмите кнопку **Отмена**.
