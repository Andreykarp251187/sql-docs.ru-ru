# <a name="complete---repair"></a>Завершение (восстановление)
  На странице **Завершение (восстановление)** можно убедиться, что восстановление завершено.  
  
## <a name="options"></a>Параметры  
 Диалоговое окно содержит ссылку на файл журнала установки для данной операции. Чтобы продолжить, нажмите кнопку **Закрыть**.
