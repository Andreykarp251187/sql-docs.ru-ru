# <a name="distributed-replay-controller-configuration"></a>Конфигурация контроллера распределенного воспроизведения
  Страница **Конфигурация контроллера распределенного воспроизведения** мастера установки [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] позволяет указать пользователей, которым нужно предоставить административные разрешения на службу контроллера распределенного воспроизведения.  
  
 Пользователи, имеющие административные разрешения, будут иметь неограниченный доступ к службе контроллера распределенного воспроизведения.  
  
## <a name="options"></a>Параметры  
 **Добавить текущего пользователя**  
 Нажмите эту кнопку, чтобы добавить пользователей, которым требуется предоставить разрешения на доступ к службе контроллера распределенного воспроизведения.  
  
 **Добавление**  
 Нажмите эту кнопку, чтобы добавить разрешения на доступ к службе контроллера распределенного воспроизведения.  
  
 **Удалить**  
 Нажмите эту кнопку, чтобы удалить разрешения на доступ к службе контроллера распределенного воспроизведения.
