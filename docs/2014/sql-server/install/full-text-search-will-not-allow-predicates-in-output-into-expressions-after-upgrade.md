# <a name="after-upgrade-full-text-search-will-not-allow-predicates-in-output-into-expression"></a>После обновления в полнотекстовом поиске будут недопустимы предикаты в выражениях OUTPUT INTO
  Полнотекстовые предикаты недопустимы в предложениях OUTPUT, когда уровень совместимости базы данных равен 100 и более.  
  
## <a name="description"></a>Описание  
 Дополнительные сведения о предложении OUTPUT, см. в разделе [предложение OUTPUT &#40;Transact-SQL&#41;](/sql/t-sql/queries/output-clause-transact-sql).
