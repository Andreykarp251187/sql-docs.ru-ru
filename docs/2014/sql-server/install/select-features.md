# <a name="select-features"></a>Выбор компонентов
  На странице «Выбор компонентов» выбираются компоненты и функции [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] для обновления.  
  
## <a name="options"></a>Параметры  
 Щелкните имя компонента в левой панели, чтобы просмотреть сведения о языке, операционной системе, выпуске и состоянии.  
  
 Чтобы обновить компонент или функцию, установите флажок рядом с именем компонента или функции. Чтобы исключить компонент или функцию из текущего обновления, снимите флажок рядом с именем компонента или функции.  
  
 Несколько экземпляров [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] на одном компьютере можно обновлять в разное время, однако все компоненты одного экземпляра [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] должны обновляться одновременно. Должны быть обновлены все общие компоненты, в том числе компоненты рабочей станции, синтаксический анализатор MSXML, SQLXML, компоненты обратной совместимости и службы синхронизации контроля версий.  
  
 Если компонент нельзя обновить, в панели «Подробности» будет отображено пояснение.
