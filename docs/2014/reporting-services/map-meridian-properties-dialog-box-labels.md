# <a name="map-meridian-properties-dialog-box-labels"></a>Диалоговое окно «Свойства меридианов карты», вкладка «Метки»
  Используйте диалоговое окно **Свойства мапмеридиан** , чтобы изменить параметры меток для вертикальной сетки в окне просмотра карт. Меридиан представляет одно из следующих значений в зависимости от выбранной системы координат области просмотра.  
  
-   **Плоская**. Координата Y.  
  
-   **Географическое**положение. Долгота в текущей проекции.  
  
 Нажмите кнопку **Выражение** (*fx*), чтобы изменить выражение, задающее значение параметра.  
  
## <a name="options"></a>Параметры  
 **Пределах**  
 Введите целое значение в градусах, задающее интервал между меридианами. По умолчанию выбрано значение **Авто** . **Auto** указывает, что значение автоматически определяется пространственными данными.  
  
 **Показывать метки**  
 Выберите этот параметр для отображения меток меридианов.  
  
 **Размещение**  
 Выберите место отображения меток относительно верха, центра и низа области просмотра. Расположение по умолчанию — **Рядом**.  
  
-   **Рядом с** Отображение меток на левом краю.  
  
-   **Один квартал** Отображать метки посередине между левым и центральным краями.  
  
-   По **центру** Отображение меток в центре.  
  
-   **Три четверти** Отображать метки посередине между центром и правым ребром.  
  
-   **Далеко** Отображение меток на правой границе.
