# <a name="databasename-property-wmi-msreportserver_configurationsetting"></a>Свойство DatabaseName (WMI MSReportServer_ConfigurationSetting)
  Задает имя экземпляра [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] , в котором размещается база данных сервера отчетов. Только для чтения.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
Public Dim DatabaseName As String  
```  
  
```csharp  
public string DatabaseName;  
```  
  
## <a name="property-values"></a>Значения свойств  
 Объект `String`, значение которого представляет имя экземпляра базы данных сервера отчетов.
