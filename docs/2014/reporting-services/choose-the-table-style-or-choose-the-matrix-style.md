# <a name="choose-the-table-style-or-choose-the-matrix-style"></a>Выбор стиля таблицы или матрицы
  На этих страницах мастера отчетов можно выбрать стили текста и линии отчета. Список стилей основан на настраиваемом шаблоне. Конструктор отчетов предоставляет следующие исходные шаблоны: Содержание рекламы, леса, организации, полужирным шрифтом, «Аквамарин» и универсальный.  
  
 Существует также возможность добавлять другие шаблоны.
