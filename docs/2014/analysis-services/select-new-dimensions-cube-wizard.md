# <a name="select-new-dimensions-cube-wizard"></a>Выбор новых измерений (мастер кубов)
  Страница **Выбор новых измерений** позволяет выбрать вновь создаваемые измерения.  
  
## <a name="options"></a>Параметры  
 **Dimensions**  
 Выберите создаваемые измерения. По умолчанию отображаются все измерения.  
  
 Вновь создаваемые измерения отображаются в виде корневых элементов дерева. Связанные таблицы, которые будут использоваться при создании измерения, отображаются внутри узла этого измерения.  
  
 Чтобы переименовать измерение, щелкните его и введите новое имя. Имена таблиц изменить нельзя.
