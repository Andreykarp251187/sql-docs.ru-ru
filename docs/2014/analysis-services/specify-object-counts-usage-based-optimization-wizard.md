# <a name="specify-object-counts-usage-based-optimization-wizard"></a>Задание счетчиков объектов (мастер оптимизации с учетом использования)
  Для автоматического вычисления количества объектов в кубе или ввода оценочного числа вручную используется страница **Определение счетчиков объектов** . С помощью счетчиков объектов мастер оптимизации с учетом использования оценивает требования к хранилищу.  
  
## <a name="options"></a>Параметры  
 **Объекты куба**  
 Выводит измерения и атрибуты куба. Отображаются только те атрибуты, `AggregationUsage` свойство которых не имеет значение None на странице **Анализ использования статистической обработки** мастера, так как это единственные атрибуты, требующие указания счетчиков.  
  
 **Оценочное число**  
 Выводит оценочное число строк группы мер и оценочное число элементов атрибута в измерениях базы данных. Значения оценочного числа можно ввести вручную или вычислить. Чтобы вычислить значения этих чисел, введите в это поле 0 и нажмите кнопку **Подсчет**. Поля, где уже отображаются оценочные числа, не обновляются.  
  
 **Число секций**  
 Введите оценочное количество строк группы мер и оценочное количество элементов атрибута в секциях (необязательно).  
  
 **Расчета**  
 Вычисляет и повторно заполняет значения в столбце **Оценочное число** для всех пустых полей. Поля, где уже отображаются оценочные числа, не обновляются.
