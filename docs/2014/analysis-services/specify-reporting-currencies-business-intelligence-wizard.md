# <a name="specify-reporting-currencies-business-intelligence-wizard"></a>Задание валют отчетов (мастер бизнес-аналитики)
  Страница **Определение валют отчетов** используется для определения валют для отчетов для функции конвертации валюты, отражающей типы конвертации «многие ко многим» и «один ко многим», заданные на странице **Выбор типа конвертации валют** . (Валюта отчетов представляет собой валюту, в которую конвертируются значения для мер, выбранных на странице **Выбор мер** .) Выбранные элементы из измерения валюты определяют измерение валюты отчетов для конвертации валют, сформированной мастером бизнес-аналитики.  
  
> [!NOTE]  
>  Эта страница не отображается, если мастер бизнес-аналитики был запущен из конструктора измерений или для его запуска пользователь щелкнул правой кнопкой мыши измерение в обозревателе решений в среде [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]. Эта страница также не отображается, если на странице **Выбор типа конвертации валют** был выбран пункт **Многие к одному** .  
  
## <a name="options"></a>Параметры  
 **Валюты отчета.**  
 Выберите параметр **Валюты для отчетов** для включения всех элементов списка из измерения валют в измерение валют отчетов. Снимите флажок **Валюты для отчетов** для исключения всех элементов списка.  
  
 Выберите конкретный элемент для включения его в измерение валют отчетов. Удалите конкретный элемент, чтобы исключить его.
