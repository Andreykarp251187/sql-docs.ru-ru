# <a name="define-a-new-attribute-manually"></a>Определение нового атрибута вручную
  В этой процедуре описывается создание нового атрибута в измерении с помощью панели **атрибуты** в конструкторе измерений.  
  
### <a name="to-create-a-new-attribute-manually"></a>Создание нового атрибута вручную  
  
1.  В конструкторе измерений откройте измерение, в котором необходимо создать атрибут.  
  
2.  Убедитесь, что панель **атрибуты** находится в формате сетки.  
  
     Щелкните правой кнопкой мыши панель **атрибуты** , наведите указатель на пункт **Отображение атрибутов в**и выберите пункт **Сетка**.  
  
3.  Перейдите к последней строке сетки и нажмите кнопку ** \<создать атрибут>**.  
  
4.  Введите имя атрибута для задания свойства `Name` и нажмите клавишу ВВОД.  
  
     Можно использовать значения по умолчанию для использования атрибута, типа, ключевого столбца и столбца имени или можно выбрать новые значения из раскрывающегося списка.  
  
5.  Щелкните правой кнопкой мыши атрибут и выберите пункт **Свойства** , чтобы настроить другие свойства.
