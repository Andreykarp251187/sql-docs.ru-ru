# <a name="define-relationship-dialog-box-analysis-services---multidimensional-data"></a>Диалоговое окно «Задание связи» (службы Analysis Services — многомерные данные)
  Используйте диалоговое окно **Задание связи** для определения связи между измерением и группой мер в конструкторе кубов. Для вывода диалогового окна **Задание связей** нажмите кнопку **...** на ячейке на панели **Сетка** на вкладке **Использование измерений** в конструкторе кубов.  
  
## <a name="options"></a>Параметры  
 **Выберите тип связи**  
 Выберите тип создаваемой связи измерения между измерением куба и группой мер. Выбор типа связи измерения изменяет содержимое панели **Подробности** .  
  
 При выборе параметра **Нет связи** связь измерения не создается.  
  
 **Detail**  
 Выводит параметры, доступные для типа связи измерения, выбранной в поле **Выберите тип связи**.  
  
## <a name="detail-pane-options"></a>Параметры панели «Подробности»  
 На панели **Подробности** выводятся следующие параметры в зависимости от типа связи, выбранной в поле **Выберите тип связи**:  
  
|Тип связи|Описание|Параметр|  
|-----------------------|-----------------|------------|  
|**Нет связи**|Связь не определена, и параметры не выводятся на панели **Подробности** .||  
|**Regular**|Задает обычную связь измерения. На панели **Подробности** отображаются следующие параметры:|**Атрибут гранулярности**: <br />                      Выберите атрибут, определяющий гранулярность группы мер относительно измерения. Данный атрибут обычно является ключевым атрибутом измерения.|  
|||**Таблица измерения** : Выводит главную таблицу для измерения.|  
|||**Таблица группы мер** : Выводит таблицу фактов для группы мер.|  
|||**Связь**: Выводит сетку столбцов измерения и столбцы группы мер, на которых основывается связь. Сетка содержит следующие столбцы:<br /><br /> **Столбцы измерений**: Содержит столбцы, связанные с выбранным атрибутом гранулярности. Примечание. Если измерение еще не создано, этот параметр имеет значение **формирования**.<br />**Столбцы группы мер** :<br />                              Выберите столбцы в группе мер, связанные со столбцами измерения.|  
|||**Дополнительно**:<br />                      Щелкните здесь для вывода диалогового окна **Привязки группы мер** и изменения дополнительных свойств, например обработки значений NULL, для связей между столбцами атрибутов и групп мер. Дополнительные сведения о диалоговом окне **Привязки группы мер** см. в разделе [Диалоговое окно "Привязки группы мер" (службы Analysis Services — многомерные данные)](measure-group-bindings-dialog-box-analysis-services-multidimensional-data.md).|  
|**Факт**|Задает связь измерения фактов. На панели **Подробности** отображаются следующие параметры:|**Атрибут гранулярности**: Выберите атрибут, определяющий гранулярность группы мер относительно измерения. Данный атрибут обычно является ключевым атрибутом измерения.|  
|||**Таблица измерения**: Выводит первичную таблицу измерения.|  
|||**Таблица группы мер**: <br />                      Выводит таблицу, на которой основывается группа мер.|  
|**Ссылка на**|Задает связь с ссылочным измерением. На панели **Подробности** отображаются следующие параметры:|**Ссылочное измерение**: <br />                      Отображает выбранное измерение.|  
|||**Промежуточное измерение**: <br />                      Выберите промежуточное измерение.|  
|||**Атрибут ссылочного измерения**: <br />                      Выберите атрибут в ссылочном измерении, который связан с атрибутом промежуточного измерения, указанным в поле **Атрибут промежуточного измерения**.|  
|||**Атрибут промежуточного измерения**: <br />                      Выберите атрибут в промежуточном измерении, который связан с атрибутом ссылочного измерения, указанным в поле **Ссылочное измерение**.|  
|||**Материализовать**: <br />                      Выберите, чтобы сохранить элемент атрибута в промежуточном измерении, связывающем атрибут в ссылочном измерении с таблицей фактов в структуре MOLAP. Материализация связи — используемый по умолчанию способ повышения производительности выполнения запросов за счет увеличения времени обработки и занимаемого места.|  
|**Многие ко многим**|Указывает связь измерения «Многие ко многим». На панели **Подробности** отображаются следующие параметры:|**Измерение** : Отображает выбранное измерение.|  
|||**Промежуточная группа мер** : <br />                      Выберите связанную промежуточную группу мер.<br /><br /> Примечание. Промежуточная группа мер должна иметь по крайней мере одно измерение, общее с выбранной группой мер. Кроме того, гранулярность связи между промежуточной группой мер и общим измерением должна быть больше или равной детализации между общим измерением и выбранной группой мер.|  
|**Интеллектуальный анализ данных**|Задает связь измерения интеллектуального анализа данных. На панели **Подробности** отображаются следующие параметры:|**Целевое измерение**: Отображает выбранное измерение интеллектуального анализа.<br /><br /> Примечание. Необходимо выбрать измерение интеллектуального анализа данных для создания связи измерения интеллектуального анализа данных.|  
|||**Исходное измерение**: Выберите измерение, по которому измерение интеллектуального анализа данных строит прогноз.|
