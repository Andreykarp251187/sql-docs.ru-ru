# <a name="advanced-settings-ssas"></a>Дополнительные параметры (SSAS)
  В этом диалоговом окне **мастера импорта таблиц** указывается кодировка и локаль для неструктурированного файла, с которым выполняется соединение. Для доступа к мастеру из [!INCLUDE[ssBIDevStudio](../includes/ssbidevstudio-md.md)]выберите пункт **Импорт из источника данных** в меню **Модель**.  
  
## <a name="uielement-list"></a>Список элементов пользовательского интерфейса  
  
|||  
|-|-|  
|Термин|Определение|  
|**Кодирование**|Укажите тип используемой кодировки — ANSI или Юникод.<br /><br /> Значение **Юникод** соответствует кодировке UTF-16.|  
|**Языкового стандарта**|Укажите локаль, чтобы предоставить определяемые языком правила для сортировки данных и формата даты и времени.|
