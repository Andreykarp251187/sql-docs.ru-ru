# <a name="analysis-services-duplicate-security-role-dialog-box-analysis-services---multidimensional-data"></a>Диалоговое окно «Дублирование роли безопасности служб Analysis Services» (службы Analysis Services — многомерные данные)
  Для дублирования существующей роли в [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] базе данных используйте **Analysis Services дублировать роль безопасности \<— имя базы данных>- \<копия в поле Имя роли>** в. [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)] Для вывода диалогового окна **Дублирование роли** щелкните роль в обозревателе объектов правой кнопкой мыши и выберите пункт **Дублировать**.  
  
## <a name="options"></a>Параметры  
  
|Термин|Определение|  
|----------|----------------|  
|**Новое имя роли**|Введите имя роли, создаваемой с теми же элементами и разрешениями, что и роль, выбранная в обозревателе объектов.|
