# <a name="advanced-save-options"></a>Дополнительные параметры сохранения
  Используйте это диалоговое окно, чтобы выбрать схему языковой кодировки и указать символы, которые должны вставляться в конце каждой строки. Для доступа к диалоговому окну **Дополнительные параметры сохранения** в меню **файл** выберите команду **сохранить** \<имя запроса>**. SQL как** , чтобы открыть диалоговое окно **сохранить файл как** , щелкните стрелку вниз на кнопке **сохранить** , а затем выберите пункт **сохранить с кодировкой**.  
  
## <a name="options"></a>Параметры  
 **Кодирование**  
 Используйте этот список, чтобы применить к нужному файлу новую схему языковой кодировки.  
  
 **Окончания строк**  
 Чтобы гарантировать, что данный файл сможет выполняться в определенной операционной системе, выберите параметр метки конца каждой строки — символ возврата каретки (для систем Macintosh), символ перевода строки (для систем Unix) или оба этих символа (для систем [!INCLUDE[msCoName](../../includes/msconame-md.md)] Windows).
