# <a name="proxy-editor-and--add-principal"></a>Редактор прокси-сервера и добавление субъекта
  Используйте эту страницу, чтобы предоставить участникам сервера доступ [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] к учетной записи-посреднику агента.  
  
## <a name="options"></a>Параметры  
 **Тип участника**  
 Тип участника для отображения.  
  
 **Доступные субъекты**  
 Список участников выбранного типа.  
  
 **Название**  
 Выберите имя участника, которому необходимо предоставить доступ, или выберите **Имя** , чтобы предоставить доступ всем участникам в списке.
