# <a name="job-properties-new-job-alerts-page"></a>Свойства задания: создание задания (страница "Оповещения")
  Используйте эту страницу, чтобы просмотреть и систематизировать предупреждения для задания агента [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
## <a name="options"></a>Параметры  
 **Список предупреждений**  
 Перечисляет предупреждения для данного задания.  
  
 **Добавить**  
 Создает новое предупреждение для задания.  
  
 **Изменить**  
 Изменяет выбранное определение предупреждения.  
  
 **Удалить**  
 Удаляет выбранное предупреждение из задания.
