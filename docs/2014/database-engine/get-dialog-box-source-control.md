# <a name="get-dialog-box-source-control"></a>Диалоговое окно «Получить» (система управления версиями)
  Получает доступную только для чтения копию выбранного элемента из базы данных системы управления версиями, помещая ее в вашу рабочую папку или в другую указанную папку.  
  
## <a name="dialog-box-access"></a>Доступ к диалоговому окну  
 В среде [!INCLUDE[ssManStudioFull](../includes/ssmanstudiofull-md.md)]выберите элемент в обозревателе решений. В меню **Файл** выберите пункт **Система управления версиями** , затем — **Получить**.  
  
> [!NOTE]  
>  Вызвать это диалоговое окно также можно, щелкнув правой кнопкой мыши элемент в обозревателе решений.  
  
## <a name="options"></a>Параметры  
 **Действие**  
 Указывает действие, выполняемое для элементов при их получении.  
  
 **Столбцы**  
 Определяет столбцы для вывода и порядок, в котором они будут выводиться.  
  
 **Представлении без структуры**  
 Выводит получаемые файлы как плоские списки под их соединениями системы управления версиями.  
  
 **Время изменения**  
 Выводит время последнего изменения данного элемента.  
  
 **Name**  
 Выводит имена получаемых элементов. Напротив элементов отображаются флажки, которые установлены. Если какой-либо элемент получать не нужно, снимите его флажок.  
  
 **Параметры**  
 Выводит параметры получения, специфичные для подключаемых модулей, если щелкнуть стрелку справа от данной кнопки.  
  
 **Sort**  
 Сортировка порядка отображаемых столбцов.  
  
 **Представление в виде дерева**  
 Выводит иерархию папок и файлов для получаемых элементов.
