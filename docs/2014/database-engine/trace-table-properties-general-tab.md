# <a name="trace-table-properties-general-tab"></a>Свойства таблицы трассировки (вкладка «Общие»)
  Вкладка **Общие** диалогового окна **Свойства таблицы трассировки** выводит свойства таблицы трассировки.  
  
 Для открытия этого окна откройте таблицу трассировки с помощью приложения [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssSqlProfiler](../includes/sssqlprofiler-md.md)] , затем в меню **Файл** выберите пункт **Свойства**.  
  
## <a name="options"></a>Параметры  
 **Имя таблицы**  
 Отображает сервер, базу данных, схему и имя для выводимой таблицы трассировки.  
  
 **Имя поставщика трассировки**  
 Выводит имя трассируемого экземпляра SQL Server.  
  
 **Тип поставщика трассировки**  
 Тип сервера, обеспечившего трассировку.  
  
 **Версия**  
 Версия сервера, обеспечившего трассировку.  
  
 **Сквоз**  
 Число строк в таблице трассировки.  
  
 **Размер данных (количество страниц)**  
 Число страниц данных в таблице трассировки.
