# <a name="symmetric-keys-on-system-databases"></a>Симметричные ключи в системных базах данных
  Это правило проверяет наличие пользовательских симметричных ключей в базах данных master, msdb, model и tempdb.  
  
## <a name="best-practices-recommendations"></a>Рекомендации  
 Не создавайте симметричные ключи в системных базах данных.
