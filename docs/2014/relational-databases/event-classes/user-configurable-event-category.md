# <a name="user-configurable-event-category"></a>Категория событий User-Configurable
  Категория событий **User-Configurable** содержит классы событий **User-Configurable** .  
  
## <a name="in-this-section"></a>в этом разделе  
  
|Раздел|Description|  
|-----------|-----------------|  
|[Класс событий User-Configurable](user-configurable-event-class.md)|Используется для мониторинга пользовательских событий, мониторинг которых с помощью системных событий других категорий невозможен.|
