# <a name="sha-property-securitycertificate-class"></a>Свойство SHA (класс SecurityCertificate)
  Возвращает атрибут отпечатка SHA сертификата безопасности.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object  
.SHA [= value]  
```  
  
## <a name="parts"></a>Компоненты  
 *object*  
 Securitycertificate [SecurityCertificate Class]-class.md), представляющий сертификат безопасности.  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Строковое значение, определяющее атрибут отпечатка SHA сертификата безопасности.
