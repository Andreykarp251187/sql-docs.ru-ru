# <a name="protocoldisplayname-property-servernetworkprotocol-class"></a>Свойство ProtocolDisplayName (класс ServerNetworkProtocol)
  Возвращает отображаемое имя сетевого протокола сервера.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object  
.ProtocolDisplayName [= value]  
```  
  
## <a name="parts"></a>Компоненты  
 *object*  
 A [класса ServerNetworkProtocol](servernetworkprotocol-class.md) , который представляет сетевой протокол, используемый экземпляром [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Строковое значение, определяющее отображаемое имя сетевого протокола сервера.
