# <a name="getting-started-in-smo"></a>Приступая к работе с объектами SMO
  Этот раздел содержит сведения о начале работы с [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] управляющих объектов (SMO). Посвященный модели объектов SMO раздел предназначен для разработчиков. Помещенный ниже перечень поможет находить информацию об иерархии объектов SMO, о том, как подготовиться к написанию программ в модели SMO, как начать составление программ SMO на различных языках программирования, а также выполнять общие и конкретные задачи программирования.
