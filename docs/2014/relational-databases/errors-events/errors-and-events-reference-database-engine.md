# <a name="errors-and-events-reference-database-engine"></a>Справочник по ошибкам и событиям (компонент Database Engine)

Этот раздел содержит выбранные сообщения об ошибках ядра СУБД, которые требуют дополнительных пояснений.
  
## <a name="in-this-section"></a>в этом разделе  
 [Базы данных, события и ошибки Engine](database-engine-events-and-errors.md) описан формат [!INCLUDE[ssDE](../../includes/ssde-md.md)] ошибка сообщения и объясняется, как просмотреть сообщения об ошибках и выходные сообщения об ошибках для приложений.  
  
 Содержит объяснение сообщений об ошибках компонента [!INCLUDE[ssDE](../../includes/ssde-md.md)] , возможные причины и действия по устранению неполадок.
