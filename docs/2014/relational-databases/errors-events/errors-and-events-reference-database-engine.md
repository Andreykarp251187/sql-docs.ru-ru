# <a name="errors-and-events-reference-database-engine"></a>Справочник по ошибкам и событиям (компонент Database Engine)

Этот раздел содержит выбранные ядро СУБД сообщения об ошибках, для которых требуется дополнительное объяснение.
  
## <a name="in-this-section"></a>в этом разделе  
 [События и ошибки ядро СУБД](database-engine-events-and-errors.md) Описывает формат сообщений [!INCLUDE[ssDE](../../includes/ssde-md.md)] об ошибках и объясняет, как просматривать сообщения об ошибках и возвращать сообщения об ошибках в приложения.  
  
 Содержит объяснение сообщений об ошибках компонента [!INCLUDE[ssDE](../../includes/ssde-md.md)] , возможные причины и действия по устранению неполадок.
