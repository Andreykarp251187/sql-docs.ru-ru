# <a name="mssqlserver_12300"></a>MSSQLSERVER_12300
    
## <a name="details"></a>Сведения  
  
|attribute|Значение|  
|-|-|  
|Название продукта|SQL Server|  
|Идентификатор события|12300|  
|Источник события|MSSQLSERVER|  
|Компонент|SQLEngine|  
|Символическое имя|HK_UNSUPPORTED_COMPUTED_COLUMNS|  
|Текст сообщения|Вычисляемые столбцы не поддерживаются для "*конструкция*".|  
  
## <a name="user-action"></a>Действие пользователя  
 Вычисляемые столбцы не поддерживаются оптимизированными для памяти таблицами.
