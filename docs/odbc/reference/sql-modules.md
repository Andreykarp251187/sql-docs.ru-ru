# <a name="sql-modules"></a>Модули SQL
Второй прием для отправки инструкций SQL СУБД — через модули. Коротко говоря модуль состоит из нескольких процедур, которые вызываются из языка программирования ведущих приложений. Каждая процедура содержит одну инструкцию SQL и данных передается через параметры и процедуры.  
  
 Модуль может рассматриваться как библиотеку объектов, чтобы код приложения. Тем не менее точно как процедуры и остальной части приложения связаны зависит от реализации. Например процедуры может быть скомпилирован в объектный код и связанных непосредственно с кодом приложения, может, они компилируются и хранятся на СУБД и вызывает для доступа к идентификаторы плана, помещаются в коде приложения или они могут интерпретироваться во время выполнения.  
  
 Главным преимуществом модулей является четко отделить инструкций SQL в языке программирования. В теории должна быть возможность изменить одно не затрагивая другую и просто связать их.
