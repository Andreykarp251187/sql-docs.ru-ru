# <a name="sql-modules"></a>Модули SQL
Второй метод отправки заявлений S'L в DBMS — это модули. Вкратце, модуль состоит из группы процедур, которые называются из языка программирования хоста. Каждая процедура содержит одно заявление S'L, и данные передаются в процедуру и из процедуры по параметрам.  
  
 Модуль можно рассматривать как библиотеку объектов, связанную с кодом приложения. Однако точное, как связаны процедуры и остальная часть приложения, зависит реализация. Например, процедуры могут быть компилированы в объектный код и связаны непосредственно с кодом приложения, они могут быть компилированы и сохранены на DBMS и вызовы для доступа к идентификаторам плана доступа, размещенным в коде приложения, или они могут быть интерпретированы во время выполнения.  
  
 Основным преимуществом модулей является то, что они чисто отделяют операторы S'L от языка программирования. Теоретически, можно изменить одно, не изменяя другое и просто переувязывая их.
