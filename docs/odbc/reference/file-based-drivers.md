# <a name="file-based-drivers"></a>Драйверы на основе файлов
Драйверы на основе файлов используются с источниками данных, такими как dBASE, которые не обеспечивают автономный движок базы данных для водителя для использования. Эти драйверы получают доступ к физическим данным напрямую и должны внедрить движок базы данных для обработки инструкций по S'L. В качестве стандартной практики, движки базы данных в файловых драйверах реализуют подмножество ODBC S'L, определяемое минимальным уровнем соответствия S'L; для списка заявлений в соответствии с этим [Appendix C: SQL Grammar](../../odbc/reference/appendixes/appendix-c-sql-grammar.md)уровнем, см.  
  
 Сравнивая драйверы на основе файлов и DBMS, драйверы на основе файлов сложнее писать из-за компонента движка базы данных, менее сложны для настройки, поскольку нет сетевых частей, и менее мощные, потому что мало кто имеет время, чтобы написать двигатели баз данных, как мощные, как те, которые производятся компаниями базы данных.
