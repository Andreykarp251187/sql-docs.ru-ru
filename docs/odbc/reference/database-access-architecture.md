# <a name="database-access-architecture"></a>Архитектура доступа к базе данных
Один из вопросов разработки ODBC был частью архитектуры доступа к базе данных для стандартизации. Интерфейсы программирования SQL, описанные в предыдущем разделе, Embedded SQL, модули SQL и CLI — являются только одной частью этой архитектуры. На самом деле, поскольку ODBC в основном предназначался для подключения персональных компьютерных приложений к миникомпутер и мэйнфреймам, существует также ряд сетевых компонентов, некоторые из которых могут быть стандартизированы.
