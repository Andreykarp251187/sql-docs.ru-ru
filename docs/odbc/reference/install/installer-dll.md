# <a name="installer-dll"></a>Библиотека DLL установщика
> [!NOTE]  
>  Начиная с Windows XP и Windows Server 2003, ODBC входит в операционную систему Windows. Следует явно устанавливать ODBC только в более ранних версиях Windows.  
  
 Библиотека DLL установщика содержит функции для установки и удаления компонентов ODBC, поддержания данных реестра об этих компонентах и сохранения данных реестра об источниках данных. Она написана корпорацией Майкрософт и может быть перераспределена. Полное описание функций в библиотеке DLL установщика см. в [справочнике по API DLL установщика](../../../odbc/reference/syntax/installer-dll-api-reference-function.md).
