# <a name="installer-dll"></a>Библиотека DLL установщика
> [!NOTE]  
>  Начиная с Windows XP и Windows Server 2003, ODBC включен в систему работы Windows. Вы должны только явно установить ODBC на более ранних версиях Windows.  
  
 Установщик DLL содержит функции по установке и удалению компонентов ODBC, ведению реестра информации об этих компонентах и ведению реестра информации об источниках данных. Она написана корпорацией Майкрософт и может быть перераспределена. Для полного описания функций в установщике DLL [см.](../../../odbc/reference/syntax/installer-dll-api-reference-function.md)
