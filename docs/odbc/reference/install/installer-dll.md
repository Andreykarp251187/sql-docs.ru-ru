# <a name="installer-dll"></a>Библиотека DLL установщика
> [!NOTE]  
>  Начиная с Windows XP и Windows Server 2003, ODBC включена в операционную систему Windows. ODBC следует только явным образом установить в более ранних версиях Windows.  
  
 Библиотека DLL установщика содержит функции для установки компонентов ODBC, настройки реестра информации о этих компонентов и сопровождать реестра сведения об источниках данных. Он записывается корпорацией Майкрософт и может свободно распространяться. Полное описание функций в установщик библиотеки DLL, см. в разделе [Справочник по API библиотеки DLL установщика](../../../odbc/reference/syntax/installer-dll-api-reference-function.md).
