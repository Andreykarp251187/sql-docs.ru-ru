# <a name="installing-odbc-components"></a>Установка компонентов ODBC
> [!NOTE]  
>  Начиная с Windows XP и Windows Server 2003, ODBC включена в операционную систему Windows. ODBC следует только явным образом установить в более ранних версиях Windows.  
  
 В этом разделе описывается способ установки и удаления компонентов ODBC. Так как разработчики всегда устанавливайте компонент ODBC (драйвер их), они должны прочитать этот раздел. Разработчики приложений должны прочитать этот раздел только в том случае, если они будут отправлены компонентов ODBC вместе со своими приложениями. Компоненты ODBC включают диспетчера драйверов, драйверы, преобразователей, библиотека DLL установщика, библиотека курсоров и все связанные файлы. В целях этого раздела приложения ODBC не считаются компонентов ODBC.  
  
> [!NOTE]  
>  В этом разделе относится только к платформы Microsoft Windows. Способ установки компонентов ODBC на других платформах зависит от платформы.  
  
 Компоненты ODBC устанавливаются или удаляются для каждого компонента, компонент, не принципу файлы. Например если преобразователь состоит из переводчик себя и количество файлов данных, эти файлы устанавливаются и удален как группу; они должны не устанавливается и удалена на основе файла файл. Причиной этого является убедитесь, что в системе существует только полный набор компонентов.  
  
 Для целей установки и удаления компонентов ниже определены как компоненты ODBC:  
  
-   **Компоненты ядра.** Диспетчер драйверов, библиотека курсоров, библиотека DLL установщика и любые другие связанные файлы составляют основные компоненты и должен быть установлен и удален как группу.  
  
-   **Драйверы.** Каждый драйвер — это отдельный компонент.  
  
-   **Переводчики.** Каждого из них — это отдельный компонент.
