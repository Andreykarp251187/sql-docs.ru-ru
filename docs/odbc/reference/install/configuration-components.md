# <a name="configuration-components"></a>Компоненты конфигурации
> [!NOTE]  
>  Начиная с Windows XP и Windows Server 2003, ODBC включен в систему работы Windows. Вы должны только явно установить ODBC на более ранних версиях Windows.  
  
 Источники данных настраиваются на установщик DLL, который, в свою очередь, вызывает установку DLL драйверов и настройки DLL переводчика по мере необходимости. Установщик DLL либо вызывается непосредственно из панели управления или загружается и называется другой программой, известной как *программа администрирования.* Следующая иллюстрация показывает взаимосвязь между компонентами конфигурации.  
  
 ![Связь между компонентами конфигурации](../../../odbc/reference/install/media/pr30.gif "pr30")  
  
 Для получения дополнительной информации об этих компонентах смотрите следующие темы в конце этого раздела.  
  
-   [Программа установки](../../../odbc/reference/install/setup-program.md)  
  
-   [Библиотека DLL установщика](../../../odbc/reference/install/installer-dll.md)  
  
-   [Библиотека DLL программы установки драйвера](../../../odbc/reference/install/driver-setup-dll.md)
