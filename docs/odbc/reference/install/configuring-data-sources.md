# <a name="configuring-data-sources"></a>Настройка источников данных
Информация об источниках данных хранится в системном реестре. Пользователи изменяют информацию об источнике данных с помощью программы администрирования. Это может быть администратор ODBC, устройство панели управления ODBC или программа администрирования, написанная приложением или разработчиком драйверов.  
  
 Команды PowerShell можно использовать для изменения источников данных. Для получения дополнительной информации об [Windows Data Access Components PowerShell Commands](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx)этих командах PowerShell см.  
  
> [!NOTE]  
>  Этот раздел специфичен для платформ Microsoft Windows®. Настройка источников данных на других платформах зависит от платформы.
