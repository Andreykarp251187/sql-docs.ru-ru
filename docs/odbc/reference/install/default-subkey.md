# <a name="default-subkey"></a>Подраздел по умолчанию
Реестр может указать источник данных по умолчанию с подключкой по умолчанию. Этот подключ является особым случаем подключаемого ключа спецификации источника данных и имеет те же значения, что и любой другой подключки спецификации источника данных. Разница лишь в том, что она не указана в качестве значения в соответствии с подключкой ODBC Data Sources.
