# <a name="default-subkey"></a>Подраздел по умолчанию
Реестр может указать источник данных по умолчанию в разделе по умолчанию. Этот подраздел является особым случаем подраздела спецификации источника данных и имеет те же значения, как любые другие данные источника спецификации подраздел. Единственное различие — что он не указан как значение в подразделе источников данных ODBC.
