# <a name="default-subkey"></a>Подраздел по умолчанию
В реестре может указываться источник данных по умолчанию с подразделом по умолчанию. Этот подраздел является особым случаем для подраздела спецификации источника данных и имеет те же значения, что и любой другой подраздел спецификации источника данных. Единственное отличие состоит в том, что она не указана в качестве значения в подразделе источники данных ODBC.
