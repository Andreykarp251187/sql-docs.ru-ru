# <a name="machine-data-sources"></a>Машинные источники данных
*Источники машинных данных* хранятся в системе с определенным именем пользователя. С именем источника данных связана вся информация, необходимая диспетчеру и водителю для подключения к источнику данных. Для источника данных Xbase это может быть имя драйвера Xbase, полный путь каталога, содержащий файлы Xbase, и некоторые параметры, которые говорят водителю, как использовать эти файлы, такие как режим для одного пользователя или только для чтения. Для источника данных Oracle это может быть имя драйвера Oracle, сервера, на котором находится Oracle DBMS,\*строку подключения S'L-Net, которая идентифицирует драйвер сети S'L для использования, и идентификатор системы базы данных на сервере.
