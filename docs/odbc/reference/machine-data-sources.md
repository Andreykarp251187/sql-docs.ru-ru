# <a name="machine-data-sources"></a>Машинные источники данных
*Машинные источники данных* хранятся в системе с определяемым пользователем именем. Связь с именем источника данных — это все сведения, необходимые диспетчеру драйверов и драйверу для подключения к источнику данных. Для источника данных xbase это может быть имя драйвера xbase, полный путь к каталогу, содержащий файлы xbase, а также некоторые параметры, указывающие драйверу, как использовать эти файлы, например однопользовательский режим или только для чтения. Для источника данных Oracle это может быть имя драйвера Oracle, сервера, на котором находится СУБД Oracle, строка подключения SQL * NET, определяющая используемый драйвер SQL\*NET, и системный идентификатор базы данных на сервере.
