# <a name="version-number"></a>Номер версии
Существует несколько версий ODBC, каждый из которых различных функций. Приложение определяет, поддерживаемый ODBC версии диспетчера драйверов и конкретный драйвер, вызвав **SQLGetInfo** параметры SQL_ODBC_VER и SQL_DRIVER_ODBC_VER.
