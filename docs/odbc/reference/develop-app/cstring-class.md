# <a name="cstring-class"></a>Класс CString
Поскольку объекты класса **CString** в Microsoft® Visual C++® являются подписанными, а строковые аргументы в ФУНКЦИЯх ODBC не подписаны, приложения, передающие объекты **CSTRING** в функции ODBC без их приведения, будут принимать предупреждения компилятора.
