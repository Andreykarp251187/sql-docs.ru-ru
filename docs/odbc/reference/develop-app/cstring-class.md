# <a name="cstring-class"></a>Класс CString
Поскольку объекты класса **CString** в корпорации Майкрософт® Visual C-® подписаны и строки аргументы в функциях ODBC не подписаны, приложения, которые передают объекты **CString** функциям ODBC без их литья, будут получать предупреждения компилятора.
