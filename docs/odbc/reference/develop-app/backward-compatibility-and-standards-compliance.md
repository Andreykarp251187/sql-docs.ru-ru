# <a name="backward-compatibility-and-standards-compliance"></a>Обратная совместимость и соответствие стандартам
Обратная совместимость — это способность новых компонентов ODBC работать со старыми компонентами ODBC. В следующих разделах обсуждаются, как эти компоненты влияют на изменения в ODBC *3.x*. Информация, содержащаяся в них, в основном касается написания приложения ODBC *3.x* и того, как вопросы совместимости отсталости решаются драйверами ODBC. Для конкретных рекомендаций о том, как проблемы обратной совместимости влияют на написание драйвера ODBC *3.x,* [см.](../../../odbc/reference/appendixes/appendix-g-driver-guidelines-for-backward-compatibility.md)
