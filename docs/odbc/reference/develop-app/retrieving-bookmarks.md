# <a name="retrieving-bookmarks"></a>Извлечение закладок
Если приложение будет использовать закладки, оно должно установить атрибут SQL_ATTR_USE_BOOKMARKS оператора SQL_UB_VARIABLE перед подготовкой или выполнением оператора. Это необходимо, потому что создание и поддержание закладок может быть дорогостоящей операцией, поэтому закладки должны быть включены только тогда, когда приложение может эффективно использовать их.  
  
 Закладки возвращаются как столбец 0 набора результатов. Существует три способа их получения приложения:  
  
-   Связать столбец 0 из набора результатов. **S'LFetch** или **S'LFetchScroll** возвращает закладки для каждой строки в строке вместе с данными для других связанных столбцов.  
  
-   Позвоните в **S'LSetPos,** чтобы позиционировать сяврую в строке в строке, а затем позвоните в **S'LGetData** для столбца 0. Если драйвер поддерживает закладки, он должен всегда поддерживать возможность вызова **S'LGetData** для столбца 0, даже если он не позволяет приложениям вызывать **S'LGetData** для других столбцов до последнего связанного столбца.  
  
-   Позвоните **в S'LBulkOperations** с аргументом *операции,* установленным для SQL_ADD, и столбец 0 связан. Курсор вставляет строку и возвращает закладку для строки в связанном буфере.
