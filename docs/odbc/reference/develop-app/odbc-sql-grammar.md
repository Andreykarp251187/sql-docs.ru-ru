# <a name="odbc-sql-grammar"></a>Грамматика SQL (ODBC)
Приложения с возможностью взаимодействия всегда должны использовать SQL-грамматику ODBC в инструкциях SQL. Однако значительные отклонения возможен даже в пределах данной грамматикой. Дополнительные сведения см. в разделе [взаимодействие инструкций SQL](../../../odbc/reference/develop-app/interoperability-of-sql-statements.md).
