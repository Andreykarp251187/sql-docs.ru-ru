# <a name="odbc-sql-grammar"></a>Грамматика SQL (ODBC)
В приложениях, поддерживающих взаимодействие, всегда следует использовать грамматику ODBC SQL в инструкциях SQL. Однако значительная разновидность возможна даже в рамках этой грамматики. Дополнительные сведения см. в разделе [взаимодействие инструкций SQL](../../../odbc/reference/develop-app/interoperability-of-sql-statements.md).
