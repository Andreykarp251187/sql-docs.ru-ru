# <a name="result-set-metadata"></a>Метаданные результирующего набора
*Метаданные* — это данные, описывающие другие данные. Например, метаданные набора результатов описывают набор результатов, такие как количество столбцов в наборе результатов, типы данных этих столбцов, их имена, точность, необрядопроизм и так далее.  
  
 Совместимые приложения должны всегда проверять метаданные столбцов набора результатов. Метаданные для столбца в наборе результатов могут отличаться от метаданных для столбца, как возвращается функцией каталога. Например, предположим, что столбец updatable включен в набор результатов, созданный путем присоединения к двум таблицам. В то время как **S'LColumnPrivileges** может указывать на то, что пользователь может обновить столбец, метаданные набора результатов могут не быть, если столбец находится на «многих» сторонах соединения; многие источники данных могут обновлять столбцы на "одной" стороне соединения, но не на "многих" сторонах. Даже типы данных нельзя считать одинаковыми, поскольку источник данных может способствовать типу данных при создании набора результатов.
