# <a name="result-set-metadata"></a>Метаданные результирующего набора
*Метаданные* — это данные, которые описывают другие данные. Например, метаданные результирующего набора описывают результирующий набор, например число столбцов в результирующем наборе, типы данных этих столбцов, их имена, точность, допустимость значений NULL и т. д.  
  
 Приложения, поддерживающие взаимодействие, должны всегда проверять метаданные столбцов результирующего набора. Метаданные для столбца в результирующем наборе могут отличаться от метаданных для столбца, возвращаемых функцией каталога. Например, предположим, что обновляемый столбец включен в результирующий набор, созданный при соединении двух таблиц. Хотя **SQLColumnPrivileges** может указывать, что пользователь может обновить столбец, метаданные результирующего набора могут не допустить, если столбец находится на стороне «многие» в соединении. Многие источники данных могут обновлять столбцы на стороне «один» в соединении, но не на стороне «многие». Даже типы данных нельзя считать одинаковыми, так как источник данных может повысить уровень данных при создании результирующего набора.
