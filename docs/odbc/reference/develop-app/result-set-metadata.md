# <a name="result-set-metadata"></a>Метаданные результирующего набора
*Метаданные* — это данные, описывающие другие данные. Например метаданные результирующего набора описывают результирующий набор, такие как число столбцов в результирующем наборе, типы данных этих столбцов, их имена, точность, допустимость значений NULL и т. д.  
  
 Взаимодействующие приложения всегда следует выполнять проверку метаданных столбцы результирующего набора. Метаданные для столбца в результирующем наборе могут отличаться от метаданные для столбца, возвращаемое функцией каталога. Например предположим, что обновляемый столбец включен в результирующий набор, созданный путем соединения двух таблиц. Хотя **SQLColumnPrivileges** может указать, что пользователь может обновить столбец, метаданные результирующего набора может не в том случае, если столбец является на стороне «многие» соединения; много источников данных можно обновить столбцы на стороне «один», соединения, но не на " сторона «многие». Даже типы данных не предполагается должны совпадать, так как источник данных может повысить тип данных при создании результирующего набора.
