# <a name="diagnostic-tools"></a>Диагностические средства
Два средства могут помочь при диагностике проблем в приложениях ODBC. Обе возможности доступны администратору ODBC и реализуются диспетчером драйверов. *Трассировка* предоставляет средства для записи в файл журнала последовательность вызовов функций. Анализатор Visual Studio позволяет анализировать сведения о взаимодействии компонентов в распределенной среде.  
  
 Этот раздел содержит следующие подразделы.
