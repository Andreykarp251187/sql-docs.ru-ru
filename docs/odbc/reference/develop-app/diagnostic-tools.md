# <a name="diagnostic-tools"></a>Средства диагностики
Два средства могут помочь в диагностике проблем в приложениях ODBC. Оба доступны из программы администрирования ODBC реализованного диспетчером драйверов. *Трассировка* служит для записи в файл журнала последовательности вызовов функций. Visual Studio Analyzer позволяет анализа информацию о взаимодействии компонентов в распределенной среде.  
  
 Этот раздел содержит следующие подразделы.  
  
-   [Трассировка](../../../odbc/reference/develop-app/tracing.md)  
  
-   [Анализатор Visual Studio](../../../odbc/reference/develop-app/visual-studio-analyzer.md)
