# <a name="application-and-driver-compatibility"></a>Совместимость приложения и драйвера
Приложения ODBC и драйвер подпадают под несколько категорий в дополнение к своей версии. Некоторые из этих приложений несовместимы с некоторыми драйверами; в других случаях тип приложения или драйвер может иметь отношение к проблемам обратной совместимости между ними.  
  
 Этот раздел содержит следующие подразделы.  
  
-   [Типы приложений](../../../odbc/reference/develop-app/types-of-applications.md)  
  
-   [Типы драйверов](../../../odbc/reference/develop-app/types-of-drivers.md)  
  
-   [Матрица совместимости](../../../odbc/reference/develop-app/compatibility-matrix.md)
