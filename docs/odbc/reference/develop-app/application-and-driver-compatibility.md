# <a name="application-and-driver-compatibility"></a>Совместимость приложения и драйвера
Приложения и драйверы ODBC делятся на несколько категорий в дополнение к их версии. Некоторые из этих приложений несовместимы с некоторыми драйверами. в других случаях тип приложения или драйвера может влиять на проблемы обратной совместимости между ними.

-   [Типы драйверов](../../../odbc/reference/develop-app/types-of-drivers.md)
