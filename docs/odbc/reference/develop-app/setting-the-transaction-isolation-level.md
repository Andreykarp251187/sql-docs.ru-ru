# <a name="setting-the-transaction-isolation-level"></a>Установка уровня изоляции транзакций
Для установки уровня изоляции транзакции приложение использует атрибут SQL_ATTR_TXN_ISOLATION соединения. Если источник данных не поддерживает запрашиваемый уровень изоляции, драйвер или источник данных может установить более высокий уровень. Чтобы определить, какие уровни изоляции транзакций поддерживает источник данных и каков уровень изоляции по умолчанию, приложение вызывает **S'LGetInfo** с SQL_TXN_ISOLATION_OPTION и SQL_DEFAULT_TXN_ISOLATION опций, соответственно.  
  
 Более высокие уровни изоляции транзакций обеспечивают наибольшую защиту целостности данных базы данных. Сериализируемые транзакции гарантированно не затрагиваются другими транзакциями и, следовательно, гарантируют сохранение целостности базы данных.  
  
 Однако более высокий уровень изоляции транзакций может привести к снижению производительности, поскольку это увеличивает вероятность того, что приложению придется ждать блокировки данных, которые будут выпущены. Приложение может указать более низкий уровень изоляции для повышения производительности в следующих случаях:  
  
-   Когда можно гарантировать, что не существует других транзакций, которые могли бы помешать транзакциям приложения. Эта ситуация возникает только в ограниченных обстоятельствах, например, когда один человек в небольшой компании поддерживает файлы dBASE, которые содержат данные персонала на одном компьютере и не делятся этими файлами.  
  
-   Когда скорость более критична, чем точность, и любые ошибки могут быть небольшими. Например, предположим, что компания делает много небольших продаж и что большие продажи редки. Сделка, которая оценивает общую стоимость всех открытых продаж, может безопасно использовать уровень изоляции Read Uncommitted. Хотя транзакция будет включать в себя ордера, которые открываются или закрываются и затем откатываются, они, как правило, отменяют друг друга, и транзакция будет гораздо быстрее, потому что она не блокируется каждый раз, когда она сталкивается с таким заказом.
