# <a name="enabling-visual-studio-analyzer"></a>Включение анализатора Visual Studio
> [!IMPORTANT]  
>  Поддержка Visual Studio Analyzer была удалена с Windows 8 (Visual Studio Analyzer была включена только в старые версии Visual Studio.). Для альтернативного механизма устранения неполадок используйте отслеживание BID.  
  
 Microsoft Visual Studio Analyzer — это инструмент анализа приложений, предназначенный для обеспечения высокого уровня оценки производительности приложения на всех уровнях и системах. Основное внимание уделяется взаимодействию между компонентами. Visual Studio Analyzer можно запустить и остановить из вкладки отслеживания администратора ODBC.  
  
 Чтобы просмотреть любое из событий, генерируемое ODBC, выполните следующие действия:  
  
1.  Доступ к администратору источника данных ODBC.  
  
2.  Нажмите на вкладку **Отслеживание.**  
  
3.  Нажмите **Начало Визуальный студии Анализатор Отслеживание**.  
  
4.  Нажмите кнопку **ОК**.  
  
 Визуальное Studio Analyzer событие поколения продолжается до **остановки Visual Studio Анализатор Трассировка** не выбран.
