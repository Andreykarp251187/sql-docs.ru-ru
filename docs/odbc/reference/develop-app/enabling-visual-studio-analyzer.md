# <a name="enabling-visual-studio-analyzer"></a>Включение анализатора Visual Studio
> [!IMPORTANT]  
>  Поддержка анализатор Visual Studio была удалена начиная с Windows 8 (анализатор Visual Studio была включена только в более ранних версиях Visual Studio.) Для альтернативного механизма устранения неполадок используйте трассировку ТЕНДЕРа.  
  
 Анализатор Microsoft Visual Studio — это средство анализа приложений, предназначенное для обеспечения высокого уровня производительности приложения на всех уровнях и в системах. Основное внимание уделяется взаимодействию между компонентами. Анализатор Visual Studio можно запустить и остановить на вкладке "Трассировка" администратора ODBC.  
  
 Чтобы просмотреть события, создаваемые ODBC, выполните следующие действия.  
  
1.  Доступ к администратору источников данных ODBC.  
  
2.  Щелкните вкладку **Трассировка**.  
  
3.  Нажмите кнопку **пуск Анализатор Visual Studio трассировку**.  
  
4.  Нажмите кнопку **ОК**.  
  
 Анализатор Visual Studio создание событий продолжится до тех пор, пока не будет выбрана **трассировка Анализатор Visual Studio** .
