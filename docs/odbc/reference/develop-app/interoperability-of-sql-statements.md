# <a name="interoperability-of-sql-statements"></a>Совместимость инструкций SQL
Как и в остальной части приложения, операторы S'L могут быть совместимыми или специфичными для DBMS. И, как и остальная часть приложения, выбор того, как совместимые операторы S'L должны быть, зависит от типа приложения. Пользовательские приложения с меньшей вероятностью используют совместимые операторы S'L, поскольку они, как правило, предназначены для использования возможностей одного или, возможно, двух DBMS. Общие приложения используют совместимые операторы S'L, поскольку они предназначены для работы с различными DBMS. И вертикальные приложения обычно падают где-то поперек, требуя определенного уровня функциональности, но в противном случае с использованием совместимых инструкций S'L.
