# <a name="parameter-markers-in-procedure-calls"></a>Маркеры параметров в вызовах процедуры
При вызове процедуры, которые принимают параметры, с возможностью взаимодействия приложения должны использовать маркеры параметров вместо значений параметра literal. Некоторые источники данных не поддерживают использование значений литерала параметра в вызове процедуры. Дополнительные сведения о параметрах см. в разделе [параметров инструкции](../../../odbc/reference/develop-app/statement-parameters.md). Дополнительные сведения о вызове процедур см. в разделе [вызовы процедур](../../../odbc/reference/develop-app/procedure-calls.md)далее в этом разделе.
