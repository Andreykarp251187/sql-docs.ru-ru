# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  Поддержка Visual Studio Analyzer была удалена, начиная с Windows 8 (Visual Studio Analyzer была включена только в более ранних версиях Visual Studio.). Устранение неполадок механизм альтернативные способы используйте трассировка встроенных средств диагностики.  
  
 Microsoft® Visual Studio™ Analyzer предоставляет высокоуровневый обзор приложения ODBC. Разработчик может оценить, анализ и отладка структуры, производительности и взаимодействия с точки зрения приложения, а не с точки зрения компонент или код. Данные, собранные этим Visual Studio Analyzer находится в форме событий, которые представляют некоторые виды взаимодействия между двумя компонентами приложения. События могут состоять из вызовов функций и возвращает из функции, такие как подключения, запросы к базе данных и транзакции.
