# <a name="choosing-a-level-of-interoperability"></a>Выбор уровня взаимодействия
Предполагая, что приложение будет использовать ODBC, следующим шагом является определение того, какой уровень совместимости требуется. Базовый уровень совместимости обычно зависит от типа приложения: пользовательские приложения, как правило, не совместимы, вертикальные приложения, как правило, совместимы между ограниченным числом DBMS, а общие приложения, как правило, совместимы между всеми DBMS.  
  
 Этот раздел содержит следующие подразделы.  
  
-   [Пользовательские приложения](../../../odbc/reference/develop-app/custom-applications.md)  
  
-   [Вертикальные приложения](../../../odbc/reference/develop-app/vertical-applications.md)  
  
-   [Универсальные приложения](../../../odbc/reference/develop-app/generic-applications.md)
