# <a name="choosing-a-level-of-interoperability"></a>Выбор уровня взаимодействия
Предположим, что приложение будет использовать ODBC, следующим шагом является определение того, необходим уровень совместимости. Базовый уровень взаимодействия обычно является функцией типа приложения: Пользовательские приложения, как правило, не следует с возможностью взаимодействия, вертикальные приложения, как правило, чтобы она могла взаимодействовать между ограниченное число СУБД и универсальные приложения, как правило, чтобы она могла взаимодействовать между все СУБД.

-   [Универсальные приложения](../../../odbc/reference/develop-app/generic-applications.md)
