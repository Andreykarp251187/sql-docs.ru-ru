# <a name="choosing-a-level-of-interoperability"></a>Выбор уровня взаимодействия
Если приложение будет использовать ODBC, следующим шагом будет определение того, какой уровень взаимодействия требуется. Базовый уровень взаимодействия обычно является функцией типа приложения: пользовательские приложения, как правило, не являются взаимодействующими, поэтому вертикальные приложения, как правило, взаимодействуют с ограниченным числом СУБД, и универсальные приложения, как правило, совместимы с ними между всеми СУБД.

-   [Вертикальные приложения](../../../odbc/reference/develop-app/vertical-applications.md)  
  
-   [Универсальные приложения](../../../odbc/reference/develop-app/generic-applications.md)
