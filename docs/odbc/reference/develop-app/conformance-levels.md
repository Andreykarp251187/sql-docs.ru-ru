# <a name="conformance-levels"></a>Уровни соответствия
Драйверы ODBC предоставляют приложению доступ к различным источникам данных. Каждый драйвер позволяет приложению определить во время выполнения возможности ODBC и грамматику SQL, которую поддерживает драйвер и каждый источник данных. Это не является обязательным требованием для приложений, предназначенных для работы с одним драйвером или небольшим известным набором драйверов, так как эти приложения можно просто записать в возможности этого драйвера или драйверов. Чтобы помочь приложениям обнаруживать возможности доступа к драйверам и источникам данных, доступны две области соответствия: интерфейс ODBC и грамматика SQL.
