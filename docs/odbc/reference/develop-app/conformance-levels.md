# <a name="conformance-levels"></a>Уровни соответствия
Драйверы ODBC дают приложению доступ к различным источникам данных. Каждый драйвер позволяет приложению во время выполнения определить, какие возможности ODBC и что поддерживает грамматика S'L драйвера и каждого источника данных. Это не является требованием приложений, предназначенных для работы с одним драйвером или небольшим, известным набором драйверов, потому что эти приложения могут быть просто записаны на возможности этого драйвера или драйверов. Для того чтобы помочь приложениям обнаружить возможности драйвера и источника данных, доступны две области соответствия: интерфейс ODBC и грамматика S'L.
