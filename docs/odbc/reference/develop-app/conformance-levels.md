# <a name="conformance-levels"></a>Уровни соответствия
Драйверы ODBC предоставить приложению доступ к различных источников данных. Каждый драйвер позволяет приложению определить во время выполнения какие возможности ODBC и какие грамматику SQL драйвер и поддерживает каждый источник данных. Это не требование для приложения, предназначенные для работы с одного драйвера или малого размера, известные набор драйверов, так как эти приложения могут быть написаны просто с возможностями, драйвер или драйверы. Чтобы помочь обнаружить драйвер и возможности источника данных приложений, доступны две области соответствия: интерфейс ODBC и грамматику SQL.
