# <a name="asynchronous-execution"></a>Асинхронное выполнение
ODBC поддерживает асинхронные операции для операций с инструкцией и подключением. Существует два способа для определения того, при завершении асинхронной операции:  
  
1.  [Асинхронное выполнение (метод опроса)](../../../odbc/reference/develop-app/asynchronous-execution-polling-method.md)  
  
2.  [Асинхронное выполнение (метод уведомления)](../../../odbc/reference/develop-app/asynchronous-execution-notification-method.md)
