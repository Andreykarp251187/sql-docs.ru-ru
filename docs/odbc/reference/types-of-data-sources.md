# <a name="types-of-data-sources"></a>Типы источников данных
Существует два типа источников данных: машинные источники данных и файловых источников данных. Несмотря на то, что оба содержат аналогичные сведения об источнике данных, они отличаются, эти данные сохраняются как. Из-за этих различий они используются в некоторой степени разными способами.  
  
 Этот раздел содержит следующие подразделы.  
  
-   [Машинные источники данных](../../odbc/reference/machine-data-sources.md)  
  
-   [Файловые источники данных](../../odbc/reference/file-data-sources.md)
