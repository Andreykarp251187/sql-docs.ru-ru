# <a name="standard-data-stream-protocol"></a>Стандартный протокол потока данных
Стандартный протокол потока данных является одним из способов доступа к данным в неоднородных DBMS. На самом деле, стандартный протокол потока данных уже существует:

- Стандарт удаленного доступа к базам данных ANSI/ISO (RDA): [ISO/IEC 9579:2000](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615). Хотя система ANSI/ISO показывает перспективы, она не широко реализована сегодня.
