# <a name="standard-data-stream-protocol"></a>Стандартный протокол потока данных
Стандартный протокол потока — один из способов доступа к данным в разнородных СУБД. На самом деле стандартный протокол потока уже существует:

- Стандарт ANSI/ISO удаленной базы данных доступ к данным: [ISO/IEC 9579:2000](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615). Несмотря на то, что в системе ANSI/ISO показывает promise, он не реализован широко уже сегодня.
