# <a name="sqlsetparam-function"></a>Функция SQLSetParam
**Соответствия**  
 Версия Введена: Соответствие стандартам ODBC 1.0: Deprecated  
  
 **Сводка**  
 В ODBC 2.0 функция ODBC 1.0 **S'LSetParam** была заменена **на S'LBindParameter**. Для получения более подробной информации, [см.](../../../odbc/reference/syntax/sqlbindparameter-function.md)  
  
> [!NOTE]  
>  Для получения дополнительной информации о том, что менеджер драйверов карты эту функцию, когда ODBC 2. *приложение x* работает с ODBC 3. *x* драйвер, [см. Отображение депрепротектных функций](../../../odbc/reference/appendixes/mapping-deprecated-functions.md).
