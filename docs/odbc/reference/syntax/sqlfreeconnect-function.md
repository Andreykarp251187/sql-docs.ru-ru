**Соответствие стандартам**  
 Представленные версии: Соответствие стандартам 1.0 ODBC: Устарело  
  
 **Сводка**  
 В ODBC 3 *.x*, функция ODBC 2.0 **SQLFreeConnect** был заменен классом **SQLFreeHandle**. Дополнительные сведения см. в разделе [SQLFreeHandle](../../../odbc/reference/syntax/sqlfreehandle-function.md).
