# <a name="sqlparamoptions-function"></a>Функция SQLParamOptions
**Соответствия**  
 Представленная версия: Соответствие стандартам ODBC 1,0: Устарело  
  
 **Сводка**  
 Функция ODBC 2,0 **SQLParamOptions** была заменена в ODBC 3. *x* по вызовам [SQLSetStmtAttr](../../../odbc/reference/syntax/sqlsetstmtattr-function.md).  
  
> [!NOTE]  
>  Дополнительные сведения о какие диспетчера драйверов сопоставляет эту функцию, чтобы при ODBC 2. *x* при работе с ODBC 3. *x* драйвера, см. в разделе [сопоставление устаревшей функции](../../../odbc/reference/appendixes/mapping-deprecated-functions.md) в приложении G: Рекомендации по драйверу для обеспечения обратной совместимости.

 Если приложение будет работать в 64-разрядной операционной системе, см. [сведения о ODBC 64-bit](../../../odbc/reference/odbc-64-bit-information.md).
