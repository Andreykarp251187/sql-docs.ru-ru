# <a name="constraints-of-the-gregorian-calendar"></a>Ограничения григорианского календаря
Типы данных о датах и датах, а также поля задних типов интервальных данных должны соответствовать ограничениям григорианского календаря. Эти ограничения следующие:  
  
-   Значение месячного поля должно быть между 1 и 12, включительно.  
  
-   Значение дневного поля должно находиться в диапазоне от 1 до количества дней в месяце. Количество дней в месяце определяется из значений поля года и месяцев и может быть 28, 29, 30 или 31. (Количество дней в месяце также может зависеть от того, является ли это високосным годом.)  
  
-   Значение часового поля должно быть между 0 и 23, включительно.  
  
-   Значение минутного поля должно быть между 0 и 59, включительно.  
  
-   Для поля задних секунд типов данных интервала значение поля секунд должно быть между 0 и 59.9 *(n),* включительно, где *n* будет числом цифр в дробной точности секунд.  
  
-   Для поля задних секунд типов данных времени даты значение поля секунд должно быть между 0 и 61.9 *(n),* включительно, где *n* указывает число "9" цифр и значение *n* является дробной точностью секунд. (Диапазон секунд позволяет целых две секунды для поддержания синхронизации бокового времени.)
