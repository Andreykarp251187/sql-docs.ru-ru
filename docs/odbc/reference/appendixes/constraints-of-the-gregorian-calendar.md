# <a name="constraints-of-the-gregorian-calendar"></a>Ограничения григорианского календаря
Типы данных даты и даты и времени и конечные поля интервальных типов данных, должны соответствовать ограничениям, григорианского календаря. Эти ограничения являются следующим образом:  
  
-   Значение поля месяца должно быть в диапазоне от 1 до 12 включительно.  
  
-   Значение поля день должен быть в диапазоне от 1 до количества дней в месяце. Число дней в месяце определяется на основе значения поля «год» и «месяцев и может быть 28, 29, 30 или 31. (Количество дней в месяце можно также зависят от того, является ли год високосным.)  
  
-   Значение из поля часа должен быть в диапазоне от 0 до 23 включительно.  
  
-   Значение из поля минуты должно быть в диапазоне от 0 до 59 включительно.  
  
-   Конечные поле секунд интервальных типов данных, значение в поле секунд должно быть между 0 и 59.9 (*n*) включительно, где *n* — количество цифр в точность в долях секунды.  
  
-   Конечные поля секунд типов данных даты и времени, значение в поле секунд должно быть между 0 и 61.9 (*n*) включительно, где *n* указывает количество цифр «9» и значение *n*  имеет точность в долях секунды. (До двух корректировочных секунд для синхронизации времени sidereal позволяет диапазон в секундах).
