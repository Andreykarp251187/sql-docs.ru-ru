# <a name="numeric-literals"></a>Числовые литералы
Числовые буквы используются при хранении численных значений данных в строках символов. Для преобразования численных данных S'L в SQL_C_CHAR строку или преобразования численных данных C в SQL_CHAR или SQL_VARCHAR строку, цифровой буквальный синтаксис используется для определения того, что хранится в цели. Для преобразования числовой сохраненной в виде SQL_C_CHAR строки в числовые данные S'L или числовую, хранящуюся в качестве строки SQL_CHAR к числимым данным C, этот синтаксис используется для проверки того, что хранится в источнике.  
  
 Число в перечном делении должно соответствовать синтаксису, определенному в разделе [Числового Литературного синтаксиса](../../../odbc/reference/appendixes/numeric-literal-syntax.md) в приложении C: Грамматика СЗЛ.
