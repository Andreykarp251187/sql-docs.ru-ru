# <a name="location-of-cache"></a>Расположение кэша
> [!IMPORTANT]  
>  Эта функция будет удалена в будущей версии Windows. Избегайте использования этой функции в новых разработках и планируйте модифицировать приложения, использующие эту функцию в настоящее время. Корпорация Майкрософт рекомендует использовать функцию курсора драйвера.  
  
 Библиотека курсора кэширует данные в памяти и в Windows® временных файлов. Это ограничивает размер набора результатов, с которым библиотека курсора может обрабатываться только доступным дисковым пространством. Временный файл используется, когда кэшированные данные пересекают границу сегмента, если вставляются в конце кэша библиотеки курсора. Вместо этого кэшированные данные добавляются вместо последнего сохраненного блока данных в кэше. Последний сохраненный блок данных сохраняется во временном файле. Если библиотека курсора завершается ненормально, например, когда сбой питания не работает, она может оставить временные файлы Windows на диске. Они называются zCTT*nnnn*.tmp и создаются в текущем каталоге.  
  
> [!NOTE]  
>  Если библиотека курсоров в корпорации Майкрософт® WindowsNT®/Windows2000 пытается кэшировать данные во временном файле в текущем каталоге, пока приложение работает с доступной только для чтения акцией или компактным диском (например, образец библиотеки класса Microsoft Foundation), будет возвращена библиотека S'LSTATE.RU (общая ошибка не может создать буфер).
