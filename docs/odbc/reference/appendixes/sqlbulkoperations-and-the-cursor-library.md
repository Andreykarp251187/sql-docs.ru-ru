# <a name="sqlbulkoperations-and-the-cursor-library"></a>SQLBulkOperations и библиотека курсоров
> [!IMPORTANT]  
>  Эта функция будет удалена в будущей версии Windows. Избегайте использования этой функции в новых разработках и планируйте модифицировать приложения, использующие эту функцию в настоящее время. Корпорация Майкрософт рекомендует использовать функцию курсора драйвера.  
  
 Библиотека курсоров не поддерживает вызов **S'LBulkOperations.** Для получения общей информации о **S'LbulkOperations** [см.](../../../odbc/reference/syntax/sqlbulkoperations-function.md)
