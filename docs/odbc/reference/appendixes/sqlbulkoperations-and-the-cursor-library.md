# <a name="sqlbulkoperations-and-the-cursor-library"></a>SQLBulkOperations и библиотека курсоров
> [!IMPORTANT]  
>  Эта функция будет удалена в следующей версии Windows. Избегайте использования этой функции в новых разработках и запланируйте изменение приложений, которые в настоящее время используют эту функцию. Корпорация Майкрософт рекомендует использовать функцию курсора драйвера.  
  
 Библиотека курсоров не поддерживает вызов **SQLBulkOperations**. Общие сведения о **SQLBulkOperations**см. в разделе [функция SQLBulkOperations](../../../odbc/reference/syntax/sqlbulkoperations-function.md).
