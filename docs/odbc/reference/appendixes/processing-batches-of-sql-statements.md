# <a name="processing-batches-of-sql-statements"></a>Обработка пакетов инструкций SQL
> [!IMPORTANT]  
>  Эта функция будет удалена в будущей версии Windows. Избегайте использования этой функции в новых разработках и планируйте модифицировать приложения, использующие эту функцию в настоящее время. Корпорация Майкрософт рекомендует использовать функцию курсора драйвера.  
  
 Библиотека курсоров не поддерживает пакеты инструкций S'L, в том числе инструкции S'L, для которых атрибут SQL_ATTR_PARAMSET_SIZE оператора превышает 1. Если приложение отправляет пакет инструкций s'L в библиотеку курсоров, результаты не определены.
