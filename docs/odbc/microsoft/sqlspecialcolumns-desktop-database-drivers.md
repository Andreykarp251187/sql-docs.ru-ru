# <a name="sqlspecialcolumns-desktop-database-drivers"></a>SQLSpecialColumns (драйверы для баз данных на настольном компьютере)
Уникальный индекс будет возвращен (если он существует) для флага SQL_BEST_ROWID в *fColType.* Для флага SQL_ROWVER не будет возвращен набор результатов.  
  
 Все иные иудеи строки имеют область SQL_SCOPE_CURROW.  
  
 Сопоставление шаблонов не поддерживается ни для аргумента *szTableQualifier,* ни *для szTableName.*
