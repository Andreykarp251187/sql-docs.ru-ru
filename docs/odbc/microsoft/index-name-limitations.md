# <a name="index-name-limitations"></a>Ограничения имен индекса
Если используется драйвер для Paradox, первичный индекс должен иметь имя, совпадающее с именем таблицы, на котором он определен. Другие индексы уникальный или неуникальный должен иметь имя, совпадающее с именем таблицы, на котором они определены.
