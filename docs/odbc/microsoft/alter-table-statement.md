# <a name="alter-table-statement"></a>Инструкция ALTER TABLE
Если используется Microsoft Access, dBASE или Paradoxdriver, инструкции ALTER TABLE поддерживается add и drop предложения определения ограничений таблицы. (Для драйверов Microsoft Excel или текст не поддерживаются инструкции ALTER TABLE.)  
  
> [!NOTE]  
>  При использовании драйвер для Paradox без реализации СУБД Borland, инструкции ALTER TABLE не поддерживаются; только чтения и добавления инструкции являются допустимыми.
