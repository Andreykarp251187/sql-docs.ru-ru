# <a name="alter-table-statement"></a>Инструкция ALTER TABLE
При использовании Microsoft Access, dBASE или Paradoxdriver заявление ALTER TABLE поддерживается положениями определения ограничений таблицы добавления и отпадения. (Заявления ALTER TABLE не поддерживаются для драйверов Microsoft Excel или Text.)  
  
> [!NOTE]  
>  При использовании драйвера Paradox без реализации двигателя базы данных Borland операторы ALTER TABLE не поддерживаются; разрешены только показания по чтению и приложению.
