# <a name="alter-table-statement"></a>Инструкция ALTER TABLE
При использовании Microsoft Access, dBASE или Парадоксдривер инструкция ALTER TABLE поддерживается с предложениями добавления и удаления определений табличных ограничений. (Инструкции ALTER TABLE не поддерживаются для Microsoft Excel или текстовых драйверов.)  
  
> [!NOTE]  
>  При использовании драйвера Paradox без реализации ядро СУБД Borland инструкции ALTER TABLE не поддерживаются. допускаются только инструкции Read и Append.
