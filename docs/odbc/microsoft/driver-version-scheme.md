# <a name="driver-version-scheme"></a>Схема версии драйвера
> [!IMPORTANT]  
>  Эта функция будет удалена в будущей версии Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставленный Oracle.  
  
 В следующей таблице перечислены все выпущенные версии драйвера Microsoft ODBC для Oracle.  
  
|Версия драйвера|Номер сборки|Журнал доступности|  
|--------------------|------------------|--------------------------|  
|1.0|2.00.6235|Визуальный C q 4.2 и Visual Basic 5.0, Enterprise Edition|  
|2.0|2.73.7269|Визуальная студия 97 и MDAC 1.5a|  
|2.0 Обновлено|2.73.7283.01|IIS 4.0|  
|2.0 Обновлено|2.73.7283.03|MDAC 1.5b и 1.5c|  
|2.0 Обновлено|2.73.7356|ODBC 3.5 SDK|  
|2.5|2.573.2927|Визуальная студия 6.0 и MDAC 2.0|  
|2.5 Обновлено|2.573.3513|Сервер 7.L<br /><br /> Сервер 6.5 SP5|  
  
 Build 2.00.6235 (версия 1) стал первым выпуском Драйвера Microsoft ODBC для Oracle. После выхода первой версии была принята новая конвенция о наименовании.  
  
 Например, 2.73.7283.03 можно разделить на следующие отдельные компоненты:  
  
-   2 - Номер версии.  
  
-   73 - Версия Oracle Server, для которой был разработан драйвер.  
  
-   7283.03 - номер сборки драйвера.  
  
> [!NOTE]  
>  С выпуском 2.573.2973, именование конвенции привело к некоторой путанице, что 2.573 является более ранним релизом, чем 2.73, но каждый раздел номер сборки должен рассматриваться индивидуально. Число 573 больше, чем 73, так что это новая версия. Кроме того, "2.5" указывает номер версии водителя.
