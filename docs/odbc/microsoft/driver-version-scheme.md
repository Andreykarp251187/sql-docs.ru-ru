# <a name="driver-version-scheme"></a>Схема версии драйвера
> [!IMPORTANT]  
>  Этот компонент будет удален в будущих версиях Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставляемого корпорацией Oracle.  
  
 Ниже перечислены все выпущенные версии драйвера Microsoft ODBC для Oracle.  
  
|Версия драйвера|Номер сборки|Журнал доступности|  
|--------------------|------------------|--------------------------|  
|1.0|2.00.6235|Visual C++ 4.2 и Visual Basic 5.0, Enterprise Edition|  
|2.0|2.73.7269|Visual Studio 97 и MDAC 1 5a|  
|обновление 2.0|2.73.7283.01|IIS 4.0|  
|обновление 2.0|2.73.7283.03|Компоненты MDAC 1.5b и 1.5 c|  
|обновление 2.0|2.73.7356|ODBC 3.5 SDK|  
|2.5|2.573.2927|Visual Studio 6.0 и компоненты MDAC 2.0|  
|обновить 2.5|2.573.3513|SQL Server 7.0<br /><br /> SQL Server 6.5 SP5|  
  
 Сборка 2.00.6235 (версия 1) была в первом выпуске драйвер Microsoft ODBC для Oracle. После выпуска первой версии был принят новый соглашение об именовании.  
  
 Например 2.73.7283.03 можно разделить на следующие отдельные компоненты:  
  
-   2 = номер версии.  
  
-   73 = версию сервера Oracle, для которого предназначен этот драйвер.  
  
-   7283.03 = номер сборки драйвера.  
  
> [!NOTE]  
>  С помощью выпуска 2.573.2973, соглашение об именовании привело к некоторой путанице 2.573 является более ранней версии, чем 2,73, что каждый раздел номер сборки можно рассматривать по отдельности. Номер 573 больше, чем 73, так что это более новой версии. Кроме того «2.5» указывает номер версии драйвера.
