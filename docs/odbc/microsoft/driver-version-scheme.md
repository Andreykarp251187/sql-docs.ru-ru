# <a name="driver-version-scheme"></a>Схема версии драйвера
> [!IMPORTANT]  
>  Эта функция будет удалена в следующей версии Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставляемый Oracle.  
  
 В следующей таблице перечислены все выпущенные версии драйвера Microsoft ODBC для Oracle.  
  
|Версия драйвера|Номер сборки|Журнал доступности|  
|--------------------|------------------|--------------------------|  
|1.0|2.00.6235|Visual C++ 4,2 и Visual Basic 5,0, Enterprise Edition|  
|2.0|2.73.7269|Visual Studio 97 и MDAC 1.5 a|  
|2,0 Обновлено|2.73.7283.01|IIS 4,0|  
|2,0 Обновлено|2.73.7283.03|MDAC 1.5 b и 1.5 c|  
|2,0 Обновлено|2.73.7356|ПАКЕТ SDK ДЛЯ ODBC 3,5|  
|2.5|2.573.2927|Visual Studio 6,0 и MDAC 2,0|  
|2,5 Обновлено|2.573.3513|SQL Server 7,0<br /><br /> SQL Server 6,5 SP5|  
  
 Сборка 2.00.6235 (версия 1) была первой версией драйвера Microsoft ODBC для Oracle. После выпуска первой версии было принято новое соглашение об именовании.  
  
 Например, 2.73.7283.03 можно разделить на следующие отдельные компоненты:  
  
-   2 = номер версии.  
  
-   73 = версия сервера Oracle, для которого был разработан драйвер.  
  
-   7283,03 = номер сборки драйвера.  
  
> [!NOTE]  
>  При использовании выпуска 2.573.2973 соглашение об именовании привело к путанице, что 2,573 является более ранней версией, чем 2,73, но каждый раздел номера сборки следует рассматривать по отдельности. Число 573 больше 73, поэтому это более новая версия. Кроме того, "2,5" указывает номер версии драйвера.
