# <a name="sqlgetstmtoption-desktop-database-drivers"></a>SQLGetStmtOption (драйверы для баз данных на настольном компьютере)
Закладки, возвращенный *fOption* из SQL_GETBOOKMARK допустимы только при открытом запроса, а также становятся недействительными, при повторной выдаче запроса. Постоянные закладки не поддерживаются.
