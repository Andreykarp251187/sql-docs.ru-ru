# <a name="sqlfreeenv-visual-foxpro-odbc-driver"></a>SQLFreeEnv (драйвер ODBC для Visual FoxPro)
> [!NOTE]  
>  В этом разделе содержатся сведения, относящиеся к драйверу ODBC для Visual FoxPro. Общие сведения об этой функции см. в соответствующем разделе [справочника по API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Поддержка: полная  
  
 Соответствие API ODBC: уровень ядра  
  
 Закрывает драйвер ODBC для Visual FoxPro и освобождает всю память, связанную с драйвером.  
  
 Дополнительные сведения см. в разделе [SQLFreeEnv](../../odbc/reference/syntax/sqlfreeenv-function.md) в *справочнике программиста по ODBC*.
