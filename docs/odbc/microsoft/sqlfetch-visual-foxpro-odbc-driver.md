# <a name="sqlfetch-visual-foxpro-odbc-driver"></a>SQLFetch (драйвер ODBC для Visual FoxPro)
> [!NOTE]  
>  В этом разделе содержатся сведения, относящиеся к драйверу ODBC для Visual FoxPro. Общие сведения об этой функции см. в соответствующем разделе [справочника по API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Поддержка: полная  
  
 Соответствие API ODBC: уровень ядра  
  
 Извлекает одну строку из результирующего набора в расположения, указанные предыдущими вызовами метода [SQLBindCol](../../odbc/microsoft/sqlbindcol-visual-foxpro-odbc-driver.md). Подготавливает драйвер для вызова [SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md) для непривязанных столбцов.  
  
 Дополнительные сведения см. в разделе [SQLFetch](../../odbc/reference/syntax/sqlfetch-function.md) в *справочнике программиста по ODBC*.
