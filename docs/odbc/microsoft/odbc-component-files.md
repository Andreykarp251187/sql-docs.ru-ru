# <a name="odbc-component-files"></a>Файлы компонентов ODBC
Файлы драйверов ODBC требуются каждому компоненту драйверов базы данных ODBC Desktop 4.0.  
  
 Некоторые файлы требуются более чем одному компоненту. Эти файлы должны быть перераспределены, если вы собираетесь перераспределить любой из водителей настольных баз данных ODBC с вашими коммерческими приложениями.  
  
 Эти файлы устанавливаются в каталоге систем «Windows» для Windows 95 или позже, или в каталоге «Windows»System32 для Windows NT 3.51 или более позднего и Windows 2000. Если файлы ODBC ранее были установлены в другом каталоге, убедитесь, что вы используете новые файлы в каталоге «Windows»System (или System32).
