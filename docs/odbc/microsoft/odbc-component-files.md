# <a name="odbc-component-files"></a>Файлы компонентов ODBC
Файлы драйвера ODBC необходимы для каждого компонента драйверов ODBC для классической базы данных 4,0.  
  
 Некоторые файлы необходимы более чем одному компоненту. Эти файлы необходимо распространять, если планируется повторно распространить любой из драйверов базы данных ODBC для настольных систем в коммерческих приложениях.  
  
 Эти файлы устанавливаются в каталоге \Виндовс\систем для Windows 95 или более поздней версии или в каталоге \Windows\System32. для Windows NT 3,51 или более поздней версии и Windows 2000. Если файлы ODBC были ранее установлены в другой каталог, убедитесь, что вы используете новые файлы в каталоге \Виндовс\систем (или System32).
