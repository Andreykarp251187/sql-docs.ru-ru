# <a name="using-microsoft-component-services"></a>Использование служб компонентов Майкрософт
> [!IMPORTANT]  
>  Эта функция будет удалена в будущей версии Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставленный Oracle.  
  
 Вы можете включить базу данных Oracle для работы с транзакционными компонентными службами (или МТС, если вы используете Windows NT) на Microsoft Windows NT/Windows 2000 и Microsoft Windows 95/98. Для того чтобы база данных Oracle могла работать с компонентными службами, поддерживающими транзакции, системные администраторы должны создать представление под названием V$XATRANS$. Чтобы создать этот скрипт, необходимо запустить скрипт, поставляемый Oracle. Для получения дополнительной информации ознакомьтесь с помощью в службах компонентов или документацией Oracle.
