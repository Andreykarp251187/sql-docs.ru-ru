# <a name="supported-scalar-functions-visual-foxpro-odbc-driver"></a>Поддерживаемые скалярные функции (драйвер ODBC для Visual FoxPro)
Драйвер ODBC для Visual FoxPro поддерживает три типа скалярных функций, как определено в грамматике ODBC SQL: строковые, числовые и функции даты и времени.  
  
 Этот раздел содержит следующие подразделы.  
  
-   [Строковые функции (драйвер ODBC для Visual FoxPro)](../../odbc/microsoft/string-functions-visual-foxpro-odbc-driver.md)  
  
-   [Числовые функции (драйвер ODBC для Visual FoxPro)](../../odbc/microsoft/numeric-functions-visual-foxpro-odbc-driver.md)  
  
-   [Функции даты и времени (драйвер ODBC для Visual FoxPro)](../../odbc/microsoft/time-and-date-functions-visual-foxpro-odbc-driver.md)
