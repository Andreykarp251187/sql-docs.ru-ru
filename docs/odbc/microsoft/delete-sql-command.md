# <a name="delete---sql-command"></a>DELETE (команда SQL)
Отметки записей для удаления.  
  
 Visual FoxPro ODBC Driver поддерживает родной визуальный синтаксис языка FoxPro для этой команды. Для получения информации о конкретной для водителей см.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
DELETE FROM [DatabaseName!]TableName  
   [WHERE FilterCondition1 [AND | OR FilterCondition2 ...]]  
```  
  
## <a name="arguments"></a>Аргументы  
 От *«Имя базы данных»!* *Название таблицы*  
 Означаюте таблицу, в которой записи помечены для удаления.  
  
 *Databasename!* определяется имя базы данных, содержащей таблицу, если содержащаяся база данных не является базой данных, указанной с источником данных. Необходимо включить имя базы данных, содержащей таблицу, если база данных не является базой данных, указанной с источником данных. Включите восклицательный знак (!) делимитетер после имени базы данных и перед именем таблицы.  
  
 ГДЕ *ФильтрУсловие1*»И &#124; ИЛИ *FilterCondition2*...»  
 Указывает, что Visual FoxPro маркировать только определенные записи для удаления.  
  
 *FilterCondition* определяет критерии, которым должны соответствовать записи для удаления. Вы можете включить столько условий фильтра, сколько хотите, соединяя их с оператором AND или OR. Вы также можете использовать NOT оператора, чтобы изменить значение логического выражения, или вы можете использовать **EMPTY**( ) для проверки пустого поля.  
  
## <a name="remarks"></a>Remarks  
 Если SET DELETED настроен на ON, записи, отмеченные для удаления, игнорируются всеми командами, включающие область.  
  
 DELETE - При маркировке нескольких записей для удаления в таблицах, открытых для общего доступа, используется блокировка записей. Это уменьшает взаимодействие записей в многопользовательских ситуациях, но может снизить производительность. Для максимальной производительности откройте стол для эксклюзивного использования.  
  
## <a name="driver-remarks"></a>Замечания водителя  
 Когда приложение отправляет заявление ODBC S'L DELETE в источник данных, водитель Visual FoxPro ODBC преобразует команду в команду Visual FoxPro DELETE без перевода.
