# <a name="microsoft-supplied-odbc-drivers"></a>Драйверы ODBC, предоставляемые корпорацией Майкрософт
*Драйверы* являются библиотеками, которые реализуют функции в API ODBC. Каждый драйвер специфичен для конкретной системы управления базами данных (DBMS). В большинстве случаев драйверы разоблачают возможности базового DBMS; они не обязаны реализовывать возможности, не поддерживаемые DBMS. Драйверы выполняют определенные задачи и могут быть разделены на две категории: файло-основанные и DBMS основе. Для получения дополнительной информации о драйверах и архитектуре драйверов [см.](../../odbc/reference/drivers.md)
