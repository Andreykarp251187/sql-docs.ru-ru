# <a name="microsoft-supplied-odbc-drivers"></a>Драйверы ODBC, предоставляемые корпорацией Майкрософт
*Драйверы* — это библиотеки, реализующие функции API ODBC. Каждый драйвер зависит от конкретной системы управления базами данных (СУБД). В большинстве случаев драйверы предоставляют возможности базовой СУБД. они не являются обязательными для реализации возможностей, не поддерживаемых СУБД. Драйверы выполняют определенные задачи и могут классифицироваться на две категории: на основе файлов и на основе СУБД. Дополнительные сведения о драйверах и архитектуре драйверов см. в разделе [Drivers](../../odbc/reference/drivers.md).
