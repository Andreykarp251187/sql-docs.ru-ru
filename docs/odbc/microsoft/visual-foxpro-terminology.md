# <a name="visual-foxpro-terminology"></a>Терминология Visual FoxPro
**database**  
 В Visual FoxPro файл базы данных имеет расширение. dBc и может содержать одну или несколько **таблиц**.  
  
 **Таблица базы данных**  
 В Visual FoxPro — таблица, связанная с базой данных. Сравните **свободную таблицу**.  
  
 **Свободная таблица**  
 В Visual FoxPro — таблица, не связанная с базой данных.  
  
 DBF-файл, созданный в FoxPro версии 2. x, является бесплатной таблицей, если она не преобразована в таблицу Visual FoxPro и не добавлена в базу данных Visual FoxPro. Сравните **таблицу базы данных**.  
  
 **Инструкция SQL доступный**  
 Инструкция SQL, которая еще не была обработана функцией **SQLPrepare** . Дополнительные сведения об этой функции с помощью драйвера ODBC для Visual FoxPro см. в разделе [SQLPrepare (драйвер ODBC для Visual FoxPro)](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md).  
  
 **table**  
 В Visual FoxPro записи хранятся в таблице. Каждая строка таблицы представляет запись, а столбцы таблицы представляют поля записи. Каждая таблица Visual FoxPro хранится в собственном файле с расширением DBF. Таблицы Visual FoxPro могут быть связаны с базой данных.  
  
 FoxPro версии 2. таблицы *x* не связаны с базой данных.
