# <a name="visual-foxpro-terminology"></a>Терминология Visual FoxPro
**Базы данных**  
 В Visual FoxPro файл базы данных имеет расширение .dbc и может содержать одну или несколько **таблиц.**  
  
 **таблица баз данных**  
 В Visual FoxPro таблица, связанная с базой данных. Контрастный **бесплатный стол**.  
  
 **бесплатный стол**  
 В Visual FoxPro таблица, не связанная с базой данных.  
  
 Файл .dbf, созданный в версии FoxPro 2.x, является бесплатной таблицей, если только он не преобразуется в таблицу Visual FoxPro и не добавляется в базу данных Visual FoxPro. **Таблица базы данных**контраста .  
  
 **предотвражаемая выписка по S'L**  
 Заявление, которое еще не было обработано функцией **S'LPrepare.** Для получения более подробной информации об этой функции с Visual FoxPro ODBC Драйвер, [см.](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)  
  
 **Таблице**  
 В Visual FoxPro записи хранятся в таблице. Каждая строка таблицы представляет собой запись, а столбцы таблицы представляют поля записи. Каждая таблица Visual FoxPro хранится в своем собственном файле с расширением .dbf. Визуальные таблицы FoxPro могут быть связаны с базой данных.  
  
 FoxPro версия 2. *таблицы x* не связаны с базой данных.
