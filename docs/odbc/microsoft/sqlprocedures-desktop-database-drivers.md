# <a name="sqlprocedures-desktop-database-drivers"></a>SQLProcedures (драйверы для баз данных на настольном компьютере)
**SLProcedures** будет возвращать только строки для тех процедур, которые имеют по крайней мере один аргумент. Процедуры, которые не имеют аргументов, рассматриваются как мнения.  
  
|Столбец|Комментарии|  
|------------|--------------|  
|PROCEDURE_QUALIFIER|Путь к файлу базы данных.|  
|PROCEDURE_OWNER|NULL|  
|PROCEDURE_NAME|Неразграниченное название процедуры|  
|PROCEDURE_TYPE|SQL_PT_PROCEDURE|
