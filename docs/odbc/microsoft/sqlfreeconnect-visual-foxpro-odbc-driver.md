# <a name="sqlfreeconnect-visual-foxpro-odbc-driver"></a>SQLFreeConnect (драйвер ODBC для Visual FoxPro)
> [!NOTE]  
>  В этом разделе содержатся сведения, относящиеся к драйверу ODBC для Visual FoxPro. Общие сведения об этой функции см. в соответствующем разделе [справочника по API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Поддержка: полная  
  
 Соответствие API ODBC: уровень ядра  
  
 Освобождает дескриптор подключения и освобождает всю память, выделенную для дескриптора.  
  
 Дополнительные сведения см. в разделе [SQLFreeConnect](../../odbc/reference/syntax/sqlfreeconnect-function.md) в *справочнике программиста по ODBC*.
