# <a name="files-to-use-with-the-desktop-database-drivers"></a>Файлы, используемые с драйверами для баз данных на настольном компьютере
Драйверы базы данных ODBC для настольных систем используются с Microsoft Jet ядро СУБД, устанавливаемыми файлами ISAM и Microsoft ODBC. Не все они необходимы для повторного распространения продукта, но не все являются основными компонентами.  
  
|Тема|См. следующие документы.|  
|---------------------------|---------|  
|Файлы, используемые драйвером базы данных для каждого рабочего стола|Файлы драйвера ODBC|  
|Файлы Microsoft Jet, используемые драйвером базы данных для всех рабочих столов|Файлы Microsoft Jet|  
|Файлы драйверов IISAM, обеспечивающие доступ к данным между Microsoft Jet и другой СУБД|Устанавливаемые файлы драйвера ISAM|
