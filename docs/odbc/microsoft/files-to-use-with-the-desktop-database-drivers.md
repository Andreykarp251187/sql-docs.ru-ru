# <a name="files-to-use-with-the-desktop-database-drivers"></a>Файлы, используемые с драйверами для баз данных на настольном компьютере
Драйверы настольных баз данных ODBC используются с движки базы данных Microsoft Jet, установленными файлами ISAM и файлами Microsoft ODBC. Не все необходимые для перераспределения продукта, и не все компоненты являются основными.  
  
|Сведения о|См.|  
|---------------------------|---------|  
|Файлы, используемые каждым драйвером базы данных рабочего стола|Файлы драйверов ODBC|  
|Файлы Microsoft Jet, используемые каждым драйвером базы данных рабочего стола|Файлы Microsoft Jet|  
|Файлы драйверов IISAM, обеспечивающие доступ к данным между Microsoft Jet и другими DBMS|Устанавливаемые файлы драйверов ISAM|
