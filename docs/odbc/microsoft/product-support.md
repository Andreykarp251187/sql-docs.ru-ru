# <a name="product-support"></a>Поддержка продуктов
Поддержка продуктов для ODBC обеспечивается службами поддержки продуктов Майкрософт. Поскольку многие продукты Майкрософт используют ODBC в качестве основного компонента, а также перераспределяют драйверы ODBC, службы поддержки продуктов Майкрософт считают ODBC частью каждого продукта, который включает ODBC в качестве компонента. Поддержка ODBC предоставляется в соответствии с соглашением о поддержке этого продукта. Дополнительную информацию можно узнать из индивидуальной документации по продукту.
