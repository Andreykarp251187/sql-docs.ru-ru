# <a name="product-support"></a>Поддержка продуктов
Поддержка продуктов ODBC предоставляется службой технической поддержки Майкрософт. Поскольку многие продукты Майкрософт используют ODBC в качестве основного компонента, а также повторно распространяют драйверы ODBC, служба технической поддержки Майкрософт считает, что ODBC является частью каждого продукта, включающего в себя ODBC в качестве компонента. Поддержка ODBC предоставляется в соответствии с соглашением о поддержке этого продукта. Дополнительные сведения см. в документации по отдельным продуктам.
