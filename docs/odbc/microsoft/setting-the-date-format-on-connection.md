# <a name="setting-the-date-format-on-connection"></a>Установка формата даты для подключения
> [!IMPORTANT]  
>  Этот компонент будет удален в будущих версиях Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставляемого корпорацией Oracle.  
  
 Новая версия драйвера ODBC для Oracle автоматически не задан формат даты для поля даты в Oracle. Ранее при подключении драйвер, его использовать `ALTER SESSION SET NLS_DATE_FORMAT ='YYYY-MM-DD HH:MI:SS'`.  
  
 Чтобы задать формат даты, вызвать SET СЕАНСА ALTER и затем выполнять операции вставки. Пример:  
  
```  
conn.Execute "ALTER SESSION SET NLS_DATE_FORMAT = 'YYYY-MM-DD HH:MI:SS' "  
sSql = "INSERT INTO DATETEST VALUES (24,'1988-12-01 10:23:03')"  
conn.Execute sSql  
```
