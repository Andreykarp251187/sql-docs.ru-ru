# <a name="api-functions-odbc-driver-for-oracle"></a>Функции API (драйвер ODBC для Oracle)
> [!IMPORTANT]  
>  Эта функция будет удалена в следующей версии Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставляемый Oracle.  
  
 Драйвер Microsoft ODBC для Oracle поддерживает функции API уровня ядра, уровня 1 и уровня 2. Эти функции перечислены в списке [уровни соответствия ODBC](../../odbc/microsoft/odbc-driver-for-oracle-conformance-levels.md).  
  
-   Соответствие [интерфейсу уровня ядра](../../odbc/microsoft/core-level-api-functions-odbc-driver-for-oracle.md) (CLI) предоставляет функции, определенные в спецификации интерфейса командной строки ISO, и обязательные функции, определенные в спецификации Open Group CLI.  
  
-   Соответствие [уровня 1](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md) обеспечивает функциональность интерфейса базового уровня, а также дополнительные функции, такие как транзакции.  
  
-   Соответствие [уровня 2](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md) обеспечивает функциональность уровня 1, а также дополнительные функции, такие как закладки, динамические параметры и асинхронное выполнение функций ODBC.
