# <a name="api-functions-odbc-driver-for-oracle"></a>Функции API (драйвер ODBC для Oracle)
> [!IMPORTANT]  
>  Эта функция будет удалена в будущей версии Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставленный Oracle.  
  
 Драйвер Microsoft ODBC для Oracle поддерживает функции API core Level, Level 1 и Уровня 2. Эти функции перечислены в [уровнях соответствия ODBC.](../../odbc/microsoft/odbc-driver-for-oracle-conformance-levels.md)  
  
-   [Соответствие core Level Interface](../../odbc/microsoft/core-level-api-functions-odbc-driver-for-oracle.md) (CLI) содержит функции, определенные в спецификации ISO CLI, и обязательные функции, определенные в спецификации Open Group CLI.  
  
-   [Соответствие уровня 1](../../odbc/microsoft/level-1-api-functions-odbc-driver-for-oracle.md) обеспечивает функциональность интерфейса Core Level, а также дополнительные функции, такие как транзакции.  
  
-   [Соответствие уровня 2](../../odbc/microsoft/level-2-api-functions-odbc-driver-for-oracle.md) обеспечивает функциональность уровня 1, а также дополнительные функции, такие как закладки, динамические параметры и асинхронное выполнение функций ODBC.
