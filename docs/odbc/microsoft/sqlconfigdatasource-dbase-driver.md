# <a name="sqlconfigdatasource-dbase-driver"></a>SQLConfigDataSource (драйвер для dBASE)
> [!NOTE]  
>  В этом разделе приводятся сведения, относящиеся к драйверу dBASE. Общие сведения об этой функции см. в соответствующем разделе [справочника по API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Функция **SQLConfigDataSource** , которая используется для динамического добавления, изменения или удаления источника данных, использует следующие ключевые слова.  
  
|Ключевое слово|Описание|  
|-------------|-----------------|  
|коллатингсекуенце|Последовательность, в которой сортируются поля.<br /><br /> Последовательность может быть: ASCII (по умолчанию) или международная.<br /><br /> Этот параметр устанавливается так же, как и **последовательность параметров сортировки** в диалоговом окне настройки.|  
|дефаултдир|Спецификация пути к каталогу.|  
|DELETED|Для драйвера dBASE указывает, можно ли получить или разместить строки, помеченные как удаленные. Если задано значение 1, удаленные строки не отображаются. Если задано значение 0, удаленные строки обрабатываются так же, как и Неудаленные строки.<br /><br /> Этот параметр задается так же, как и **Отображение удаленных строк** в диалоговом окне установки.|  
|DESCRIPTION|Описание данных в источнике данных.<br /><br /> Этот параметр устанавливается так же, как и **Описание** в диалоговом окне установки.|  
|DRIVER|Спецификация пути к библиотеке DLL драйвера.|  
|дриверид|Целочисленный идентификатор драйвера.<br /><br /> 21 (dBASE III)<br /><br /> 277 (dBASE IV)<br /><br /> 533 (dBASE 5,0)|  
|ФАЙ|Тип файла dBase III, dBase IV или dBase 5|  
|пажетимеаут|Указывает период времени (в десятых долях секунды), в течение которого страница (если она не используется) остается в буфере перед удалением. Значение по умолчанию — 600 десятых долей секунды (60 секунд). Обратите внимание, что этот параметр применяется ко всем источникам данных, использующим драйвер ODBC.<br /><br /> Этот параметр устанавливается в диалоговом окне установки того же параметра, что и **время ожидания страницы** .|  
|READONLY|Значение TRUE, чтобы сделать файл доступным только для чтения; Значение FALSE, чтобы сделать файл недоступным только для чтения.<br /><br /> Этот параметр задается в диалоговом окне установки того же параметра, что и « **только чтение** ».|  
|STATISTICS|Для драйвера dBASE определяет, приблизительна ли статистика размера таблицы. Обратите внимание, что этот параметр применяется ко всем источникам данных, использующим драйвер ODBC.<br /><br /> Это задает тот же параметр, что и **приблизительное число строк** в диалоговом окне программы установки.|  
|МНОГОПОТОЧНО|Количество фоновых потоков, используемых ядром. Это значение равно 3 и не может быть изменено.<br /><br /> Это позволяет задать тот же параметр, что и **потоки** в диалоговом окне программы установки.|
