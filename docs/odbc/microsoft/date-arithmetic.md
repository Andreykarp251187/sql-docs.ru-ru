# <a name="date-arithmetic"></a>Арифметические операции с датами
Драйвер поддерживает добавление и вычитание рядового столбца DATE, TIME или TIMESTAMP. Для столбца DATE целый ряд определяет количество дней для добавления или вычитания. Для столбца TIME или TIMESTAMP целый ряд определяет количество секунд для добавления или вычитания.
