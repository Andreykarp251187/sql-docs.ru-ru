# <a name="date-arithmetic"></a>Арифметические операции с датами
Драйвер поддерживает добавление и вычитание целого числа из столбца даты, времени или отметки времени. Для столбца DATE целое число указывает количество дней для добавления или вычитания. Для столбца TIME или TIMESTAMP целое число указывает количество секунд для добавления или вычитания.
