# <a name="date-arithmetic"></a>Арифметические операции с датами
Драйвер поддерживает добавление и вычитание целое число из столбца даты, времени или метки времени. Для столбца даты целочисленное значение указывает число дней для добавления или вычитания. Для столбца времени или метки времени целое число указывает количество секунд для добавления или вычитания.
