# <a name="visual-foxpro-odbc-driver"></a>Драйвер ODBC для Visual FoxPro
Microsoft Visual FoxPro — это мощная среда объектно ориентированного конструирования и приложения разработки баз данных. Драйвер ODBC Microsoft Visual FoxPro позволяет приложениям откройте запрос и обновления данных в Visual FoxPro и более ранних версиях FoxPro через интерфейс откройте Database Connectivity (ODBC).  
  
 Например с помощью драйвера ODBC Microsoft Visual FoxPro могут сделайте следующее:  
  
-   Используйте Microsoft Query для запроса и обновления данных Visual FoxPro из книг Microsoft Excel.  
  
-   Создание слияния буквы с использованием данных Visual FoxPro с помощью Microsoft Word.  
  
-   Запроса и обновления Visual FoxPro представлений и таблиц из Microsoft Access.  
  
-   Используйте Visual FoxPro как хранилище данных для приложений Microsoft Visual Basic, Microsoft Visual C++ и C.
