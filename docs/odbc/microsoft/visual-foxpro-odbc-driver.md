# <a name="visual-foxpro-odbc-driver"></a>Драйвер ODBC для Visual FoxPro
Microsoft Visual FoxPro — это мощная объектно-ориентированная среда для построения баз данных и разработки приложений. Microsoft Visual FoxPro ODBC Driver позволяет приложениям открывать, закажкить и обновить данные в Visual FoxPro и более ранних версиях FoxPro через интерфейс Open Database Connectivity (ODBC).  
  
 Например, с Microsoft Visual FoxPro ODBC Driver вы можете сделать следующее:  
  
-   Используйте запрос Microsoft Query для запроса и обновления данных Visual FoxPro из листов Microsoft Excel.  
  
-   Создавайте письма с почтовым слиянием с помощью визуальных данных FoxPro с помощью Microsoft Word.  
  
-   Запрос и обновление визуальных представлений и таблиц FoxPro от Microsoft Access.  
  
-   Используйте Visual FoxPro в качестве хранилища данных для приложений Microsoft Visual Basic, Microsoft Visual C и C.
