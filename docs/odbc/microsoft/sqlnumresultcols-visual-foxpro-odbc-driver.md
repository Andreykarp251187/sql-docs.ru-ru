> [!NOTE]

 Возвращает число столбцов в курсоре результирующего набора.  
  
 Дополнительные сведения см. в разделе [SQLNumResultCols](../../odbc/reference/syntax/sqlnumresultcols-function.md) в *справочнике программиста по ODBC*.
