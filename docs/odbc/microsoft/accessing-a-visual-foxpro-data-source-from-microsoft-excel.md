# <a name="accessing-a-visual-foxpro-data-source-from-microsoft-excel"></a>Доступ к источнику данных Visual FoxPro из Microsoft Excel
Если у вас установлен запрос Майкрософт, можно создать источник данных в Microsoft Excel, который подключается к данным Visual FoxPro.  
  
### <a name="to-access-visual-foxpro-data-from-microsoft-excel"></a>Для доступа к визуальным данным FoxPro от Microsoft Excel  
  
1.  Откройте электронную таблицу Microsoft Excel.  
  
2.  Из меню данных выберите Получить внешние данные. Открывается запрос Microsoft.  
  
3.  В диалоговом окне «Выберите источник данных» щелкните «Другие».  
  
4.  В диалоговом окне ODBC Data Sources нажмите New.  
  
5.  В диалоговом окне Добавить источник данных выберите Microsoft Visual FoxPro Driver из установленного окна списка драйверов ODBC и нажмите OK.  
  
6.  В [диалоговом окне ODBC Visual FoxPro](../../odbc/microsoft/odbc-visual-foxpro-setup-dialog-box.md)введите имя источника данных, выберите тип базы данных, введите путь в базу данных или каталог и нажмите OK.  
  
     Новое имя источника данных отображается в текстовом ящике источника данных Enter Data Source в диалоговом ящике ODBC Data Sources.  
  
7.  Щелкните ОК.  
  
     Новое имя источника данных выбрано в текстовом окне исходного кода диалогового окна Select Data Source.  
  
8.  Нажмите кнопку Использовать.  
  
 Теперь можно добавить таблицы в открытый запрос. Для получения дополнительной информации о создании запроса см. [Импорт данных в Microsoft Excel из визуальной базы данных FoxPro](../../odbc/microsoft/importing-data-into-microsoft-excel-from-a-visual-foxpro-database.md).
