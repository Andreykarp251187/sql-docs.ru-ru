# <a name="creating-mailing-labels-in-microsoft-word-using-visual-foxpro-data"></a>Создание почтовых наклеек в Microsoft Word с использованием данных Visual FoxPro
Данные Visual FoxPro можно использовать в документе Microsoft Word для Windows 95 или Windows 98. Например, может потребоваться создать почтовые наклейки из сведений о клиенте, хранящихся в таблице Visual FoxPro.  
  
### <a name="to-create-mailing-labels"></a>Создание почтовых наклеек  
  
1.  В Microsoft Word создайте новый пустой документ.  
  
2.  В меню Сервис выберите пункт Слияние почты.  
  
3.  В поле помощник слияния почты выберите Создать, а затем — почтовые наклейки.  
  
4.  В разделе основной документ выберите активное окно.  
  
5.  В разделе Источник данных выберите получить данные, а затем выберите Открыть источник данных.  
  
6.  В диалоговом окне Открытие источника данных выберите MS Query.  
  
7.  В диалоговом окне Выбор источника данных выберите источник данных Visual FoxPro и нажмите кнопку использовать.  
  
8.  Если база данных, к которой обращается источник данных, содержит таблицы, выберите таблицу в диалоговом окне Добавление таблиц. Microsoft Query отобразит добавленную таблицу в верхней половине конструктора запросов.  
  
9. Выберите поля для запроса, перетащив их из таблицы в нижнюю половину конструктора.  
  
10. В меню Файл выберите пункт вернуть данные в Microsoft Word. Microsoft Query закрывается, и выбранные данные доступны для использования в документе слияния.  
  
11. В разделе основной документ выберите Настройка.  
  
12. В диалоговом окне параметры метки выберите нужный принтер и сведения о метке, а затем нажмите кнопку ОК.  
  
13. В диалоговом окне Создание меток выберите поля, которые нужно напечатать на наклейках корреспонденции, а затем нажмите кнопку ОК.  
  
14. В модуле поддержки слияния почты в разделе Слияние данных с документом нажмите кнопку объединить.  
  
15. В диалоговом окне Слияние выберите нужные параметры и нажмите кнопку объединить.
