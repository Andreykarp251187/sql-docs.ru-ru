# <a name="bookmark-support-visual-foxpro-odbc-driver"></a>Поддержка закладок (драйвер ODBC для Visual FoxPro)
Visual FoxPro ODBC Driver поддерживает простые закладки. При звонке [s'LGetInfo](../../odbc/microsoft/sqlgetinfo-visual-foxpro-odbc-driver.md) с SQL_BOOKMARK_PERSISTENCE *InfoType*значение возврата SQL_BP_SCROLL.  
  
 Для получения дополнительной информации о закладках, см [Закладки (ODBC)](../../odbc/reference/develop-app/bookmarks-odbc.md).
