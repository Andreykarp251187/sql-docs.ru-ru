# <a name="bookmark-support-visual-foxpro-odbc-driver"></a>Поддержка закладок (драйвер ODBC для Visual FoxPro)
Драйвер ODBC для Visual FoxPro поддерживает простые закладки. При вызове [SQLGetInfo](../../odbc/microsoft/sqlgetinfo-visual-foxpro-odbc-driver.md) с SQL_BOOKMARK_PERSISTENCE *инфотипе*возвращается значение SQL_BP_SCROLL.  
  
 Дополнительные сведения о закладках см. в разделе [закладки (ODBC)](../../odbc/reference/develop-app/bookmarks-odbc.md).
