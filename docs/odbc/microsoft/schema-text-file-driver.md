# <a name="schema-text-file-driver"></a>Схема (драйвер для текстовых файлов)
Схема включает сведения о каждой таблице (текстовом файле) в источнике данных, включая формат таблицы, число строк для поиска, определяющих типы столбцов, будь то первая строка таблицы содержит имена столбцов, независимо от того, записан ли исходный файл с помощью OEM или Кодовая страница ANSI и имя каждого столбца, тип данных и ширина.
