# <a name="schema-text-file-driver"></a>Схема (драйвер для текстовых файлов)
Схема содержит информацию о каждой таблице (текстовом файле) в источнике данных, включая формат таблицы, количество строк для сканирования для определения типов столбцов, содержит ли первая строка таблицы имена столбцов, пишется ли исходный файл с помощью кодовой страницы OEM или ANSI, и имя каждого столбца, тип данных и ширина.
