# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>Создание и удаление представлений и процедур с использованием ExtendedAnsiSQL
Если включен флаг ExtendedAnsiSQL, Jet 4,0 поддерживает возможность создания и удаления представлений и процедур с помощью SQL. Эти представления и процедуры не будут видны в Access.
