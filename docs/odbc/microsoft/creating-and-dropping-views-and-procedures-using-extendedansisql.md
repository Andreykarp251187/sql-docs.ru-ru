# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>Создание и удаление представлений и процедур с использованием ExtendedAnsiSQL
При включении флага ExtendedAnsiS'L Jet 4.0 поддерживает возможность создания и отбрасывания представлений и процедур с помощью S'L. Эти представления и процедуры не будут видны через Access.
