# <a name="using-synonyms-with-stored-procedures"></a>Использование синонимов с хранимыми процедурами
> [!IMPORTANT]  
>  Эта функция будет удалена в будущей версии Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставленный Oracle.  
  
 Драйвер Microsoft ODBC для версий Oracle 2.0 и 2.5 не поддерживает синонимы при вызове процедур хранения Oracle. Синонимы работают, как и ожидалось, при использовании с другими объектами базы данных Oracle, такими как таблицы.
