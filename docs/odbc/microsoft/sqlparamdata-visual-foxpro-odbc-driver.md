# <a name="sqlparamdata-visual-foxpro-odbc-driver"></a>SQLParamData (драйвер ODBC для Visual FoxPro)
> [!NOTE]  
>  В этом разделе содержатся сведения, относящиеся к драйверу ODBC для Visual FoxPro. Общие сведения об этой функции см. в соответствующем разделе [справочника по API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Поддержка: полная  
  
 Соответствие API ODBC: уровень 1  
  
 Используется в сочетании с [SQLPutData](../../odbc/microsoft/sqlputdata-visual-foxpro-odbc-driver.md) для указания данных параметра во время выполнения инструкции.  
  
 Дополнительные сведения см. в разделе [метод SQLParamData](../../odbc/reference/syntax/sqlparamdata-function.md) в *справочнике программиста по ODBC*.
