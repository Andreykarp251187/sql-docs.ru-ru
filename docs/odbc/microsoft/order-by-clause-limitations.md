# <a name="order-by-clause-limitations"></a>Ограничения предложения ORDER BY
Если заявление SELECT содержит оговорку GROUP BY и оговорку ORDER BY, то оговорка ORDER BY может содержать только столбец в наборе результатов или выражение в оговорке GROUP BY.
