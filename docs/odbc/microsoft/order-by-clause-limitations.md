# <a name="order-by-clause-limitations"></a>Ограничения предложения ORDER BY
Если инструкция SELECT содержит предложение GROUP BY и предложение ORDER BY, предложение ORDER BY может содержать только столбец в результирующем наборе или выражение в предложении GROUP BY.
