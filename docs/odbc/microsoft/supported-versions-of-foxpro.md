# <a name="supported-versions-of-foxpro"></a>Поддерживаемые версии FoxPro
Для доступа к данным, хранящимся в таблицах FoxPro, можно использовать ® Визуал FoxPro® драйвер ODBC. Поддерживаются следующие версии данных FoxPro:  
  
-   2.0  
  
-   2.5  
  
-   2.6  
  
-   Визуальный FoxPro (все версии)  
  
 При доступе к данным, хранящимся в Visual FoxPro, можно подключиться к [базе данных,](../../odbc/microsoft/visual-foxpro-terminology.md) содержащей ноль или более [таблиц,](../../odbc/microsoft/visual-foxpro-terminology.md) или к каталогу [свободных таблиц.](../../odbc/microsoft/visual-foxpro-terminology.md)  
  
 Для получения дополнительной информации о том, как подключиться к источнику данных, [см.](../../odbc/microsoft/adding-a-visual-foxpro-data-source.md)  
  
 Чтобы загрузить драйвер, смотрите [визуальный FoxPro ODBC Драйвер скачать сайт](https://go.microsoft.com/fwlink/?LinkId=121318).
