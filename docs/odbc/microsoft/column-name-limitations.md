# <a name="column-name-limitations"></a>Ограничения имен столбцов
Имена столбцов могут содержать любые допустимые символы (например, пробелы). Если названия столбцов содержат какие-либо символы, кроме букв, цифр и подчеркнутостей, имя должно быть разграничено, привязав его в задние кавычки (').  
  
 При использовании драйвера Microsoft Access или Microsoft Excel имена столбцов ограничены 64 символами, а более длинные имена создают ошибку. При использовании драйвера Парадокс атакжем максимальным значением столбца является 25 символов. При использовании драйвера текста максимальное имя столбца составляет 64 символа, а более длинные имена усечены.  
  
 При использовании драйвера dBASE символы со значением ASCII больше 127 преобразуются в подчеркивание.  
  
 При использовании драйвера Microsoft Excel, если имена столбцов присутствуют, они должны быть в первом ряду. Имя, которое в Microsoft Excel будет использовать "!" Символ должен быть заключен в задние котировки ('). Персонаж "!" преобразуется в символ "$", потому что символ "!" не является законным в названии ODBC, даже если имя заключено в задние кавычки. Все другие действительные символы Microsoft Excel (кроме символа трубы (&#124;)) могут быть использованы в названии столбца, включая пробелы. Для имени столбца Microsoft Excel необходимо использовать раздеточное идентификатор, чтобы включить пробел. Неопределенные названия столбцов будут заменены именами драйверов, например, "Col1" для первого столбца.  
  
 Символ трубы (&#124;) не может быть использован в названии столбца, независимо от того, заключено ли имя в задние кавычки или нет.  
  
 При использовании драйвера Text драйвер указывает имя по умолчанию, если имя столбца не указано. Например, водитель вызывает первую колонку F1, вторую колонку F2 и так далее.
