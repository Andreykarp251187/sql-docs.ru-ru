# <a name="configuring-the-odbc-driver-for-oracle"></a>Настройка драйвера ODBC для Oracle
> [!IMPORTANT]  
>  Этот компонент будет удален в будущих версиях Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставляемого корпорацией Oracle.  
  
 Вы можете управлять производительности драйвера ODBC для Oracle, зная среды данных и правильность настройки параметров соединения с источником данных через [администратор источников данных ODBC](../../odbc/admin/odbc-data-source-administrator.md) диалогового окна поле или через подключение Параметры строки. Диалоговое окно предоставляет следующие элементы управления для подключения к источнику данных с помощью диалогового окна или с помощью объединение строк:  
  
-   **Вкладке "DSN пользователя"** перечислены имена источников данных, которые являются локальными для компьютера.  
  
-   **Вкладке "DSN системы"** позволяет добавить или удалить системный источник данных. Системные источники данных может осуществляться всеми пользователями на локальном компьютере.  
  
-   **Вкладка "файл DSN"** позволяет добавить или удалить файл источника данных с локального компьютера. Файловые источники данных могут выполнять все пользователи, имеющие один и тот же установлен драйвер.  
  
-   **Вкладку "драйверы"** список установленных драйверов ODBC.  
  
-   **Вкладка "Трассировка"** позволяет указать, каким образом диспетчер драйверов ODBC выполняет трассировку вызовов функций ODBC. Вы можете настроить трассировку отдельно для каждого установленного приложения ODBC.  
  
-   **Создание пула подключений, вкладка** позволяет выбрать параметры подключения для каждого установленного драйвера.  
  
-   **О вкладке** перечислены установленные файлы компонентов ODBC.  
  
 После добавления источника данных, можно использовать **администратор источников данных ODBC** диалоговое окно, чтобы настроить доступ к источнику данных. Выберите источник данных и затем щелкните одну из вкладок, чтобы изменить или просмотрите сведения.
