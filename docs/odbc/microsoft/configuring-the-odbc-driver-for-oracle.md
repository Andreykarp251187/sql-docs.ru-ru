# <a name="configuring-the-odbc-driver-for-oracle"></a>Настройка драйвера ODBC для Oracle
> [!IMPORTANT]  
>  Эта функция будет удалена в будущей версии Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставленный Oracle.  
  
 Вы можете контролировать производительность драйвера ODBC для Oracle, зная среду данных и правильно устанавливая параметры соединения источника данных через диалоговую будку [администратора источников данных ODBC](../../odbc/admin/odbc-data-source-administrator.md) или через параметры строки подключения. Диалоговый ящик обеспечивает следующие элементы управления для подключения к источнику данных с помощью диалогового окна или с помощью строк подключения:  
  
-   **Вкладка пользователя DSN** Перечисляет имена источников данных, которые являются локальными для компьютера.  
  
-   **Вкладка системы DSN** Позволяет добавлять или удалять источник системных данных. Доступ к источникам системных данных могут быть доступны всем пользователям на локальном компьютере.  
  
-   **Вкладка файла DSN** Позволяет добавлять или удалять источник данных файла с локального компьютера. Источниками файлов могут быть предоставлены общие сведения всем пользователям, у которых установлен один и тот же драйвер.  
  
-   **Вкладка драйверов** Перечисляет установленные драйверы ODBC.  
  
-   **Вкладка трассировки** Позволяет указать, как диспетчер драйверов ODBC отслеживает вызовы к функциям ODBC. Вы можете настроить трассировку отдельно для каждого установленного приложения ODBC.  
  
-   **Вкладка подключения** Позволяет выбрать параметры подключения для каждого установленного драйвера.  
  
-   **О вкладке** Перечисляет установленные файлы компонентов ODBC.  
  
 После добавления источника данных можно использовать диалоговый ящик **администратора данных ODBC Data Administrator** для настройки доступа к источнику данных. Выберите источник данных, а затем нажмите одну из вкладок, чтобы отсеить или просмотреть информацию.
