# <a name="sorting-limitations"></a>Ограничения сортировки
Максимальная длина ключа сортировки в предложении GROUP BY, предложение ORDER BY инструкции SELECT DISTINCT или внешнего соединения составляет 255 байт; Максимальная длина всех ключей сортировки, в строке сортировки составляет 65 500 байт.
