# <a name="sorting-limitations"></a>Ограничения сортировки
Максимальная длина ключа сортировки в оговорке GROUP BY, оговорке ORDER BY, выписке SELECT DISTINCT или внешнем соединении составляет 255 байтов; максимальная длина всех сортированных ключей в сортировочном ряду составляет 65 500 байтов.
