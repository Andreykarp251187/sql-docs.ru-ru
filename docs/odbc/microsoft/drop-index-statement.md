# <a name="drop-index-statement"></a>Инструкция DROP INDEX
При использовании драйвера Microsoft Access, dBASE или Paradox синтаксис оператора DROP INDEX — «DROP INDEX a on b», где «a» — это название индекса, а «b» — это название таблицы (не *drop INDEX-имя).*  
  
 При использовании драйвера Парадокса заявление DROP INDEX удаляет вторичные индексные файлы Paradox.  
  
 Заявление DROP INDEX не поддерживается для драйверов Microsoft Excel или Text.
