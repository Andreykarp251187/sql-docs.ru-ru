# <a name="sqlconfigdatasource-text-file-driver"></a>SQLConfigDataSource (драйвер для текстовых файлов)
> [!NOTE]  
>  Эта тема предоставляет информацию о драйверах, специфийных для драйверов текста. Для получения общей информации об этой [ODBC API Reference](../../odbc/reference/syntax/odbc-api-reference.md)функции, см.  
  
 Функция **S'LConfigDataSource,** используемая для добавления, изменения или удаления источника данных, динамически использует следующие ключевые слова.  
  
|Ключевое слово|Описание|  
|-------------|-----------------|  
|НАБОР СИМВОЛОВ|Для драйвера текста, OEM или ANSI.|  
|КОЛНАМХЕДЕР|Для драйвера текста указывается, будет ли первая запись данных указывать имена столбцов. Либо ПРАВДА, либо FALSE.|  
|ПО УМОЛЧАНИДИР|Спецификация пути к каталогу.|  
|DESCRIPTION|Описание данных в источнике данных.<br /><br /> При этом устанавливается та же опция, что и **описание** в поле диалога установки.|  
|DRIVER|Спецификация пути для драйвера DLL.|  
|ВОДИТЕЛЬ|Идентификатор разных чиханий для водителя. 27 (Текст)|  
|Расширения|Перечисляет расширения имен файлов текста в источнике данных.<br /><br /> При этом устанавливается тот же вариант, что и **список расширений** в поле диалога установки.|  
|Fil|Текст типа файла|  
|Filetype|Тип файла для драйвера текста (Текст).|  
|FORMAT|Для драйвера текста могут быть FIXEDLENGTH, TABDELIMITED, CSVDELIMITED (по запятой) или DELIMITED () (по специальному символу, указанному в скобках). Особый символ является одним символом в длину и может быть в характере, десятичной или гексадецимальном формате.|  
|МАКССКАНРОУВ|Количество строк, которые необходимо сканировать при настройке типа данных столбца на основе существующих данных.<br /><br /> Для драйвера Текста можно ввести номер от 1 до 32767 для количества строк для сканирования; однако значение всегда будет по умолчанию до 25. (Номер за пределами лимита вернет ошибку.)<br /><br /> При этом устанавливается та же опция, что и **строки для сканирования** в поле диалога установки.|  
|READONLY|TRUE, чтобы сделать файл только читать; FALSE, чтобы файл не только читать.<br /><br /> При этом устанавливается та же опция, что и **Read Only** в поле диалога установки.|
