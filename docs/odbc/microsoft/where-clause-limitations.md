# <a name="where-clause-limitations"></a>Ограничения предложения WHERE
Максимальное количество положений в пункте WHERE составляет 40.  
  
 Столбцы LONGVARBINARY и LONGVARCHAR можно сравнить с буквальными буквами до 255 символов в длину, но не могут быть сопоставлены с использованием параметров.
