# <a name="insert-statement-limitations"></a>Ограничения инструкции INSERT
Вставленные данные усекается справа без предупреждения, если это слишком много времени, чтобы поместиться в столбец.  
  
 Попытка вставить значение, которое находится за пределами диапазона типа данных столбца вызовет NULL вставляется в столбец.  
  
 При использовании dBASE, Microsoft Excel, Paradox или Textdriver вставки строки нулевой длины в столбец фактически вставляет значение NULL вместо него.  
  
 При использовании драйвера Microsoft Excel, если пустая строка вставляется в столбец, пустая строка преобразуется в значение NULL; Искомая инструкции SELECT, который выполняется с пустой строкой в предложении WHERE не будет выполнено по этому столбцу.  
  
 Таблицу, драйвер для Paradox в двух случаях:  
  
-   Если в таблице не определен уникальный индекс. Это не так для пустых таблиц, которое может быть обновлено с одной строкой, даже если не определен уникальный индекс для таблицы. Если отдельная строка вставляется в пустую таблицу, не имеет уникального индекса, приложение не может создать уникальный индекс или вставлять дополнительные данные, после вставки одной строки.  
  
-   Если ядро базы данных Borland не используется, только чтения и добавления инструкции являются допустимыми для Paradox таблицы.  
  
 При использовании драйвера текстовых значений NULL представлены с помощью дополняются пустой строки в файлы с фиксированной длиной, но представленных без пробелов в файлы с разделителями. Например в следующей строке, содержащей три поля, второе поле имеет значение NULL:  
  
```  
"Smith:,, 123  
```  
  
 Если используется драйвер для текстовых, все значения столбцов могут дополняются начальных пробелов. Длина любой из строк должна быть меньше или равно 65,543 байт.
