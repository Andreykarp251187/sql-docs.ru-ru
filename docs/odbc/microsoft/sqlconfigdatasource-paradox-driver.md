# <a name="sqlconfigdatasource-paradox-driver"></a>SQLConfigDataSource (драйвер для Paradox)
> [!NOTE]  
>  В этом разделе приводятся сведения, относящиеся к драйверу Paradox. Общие сведения об этой функции см. в соответствующем разделе [справочника по API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Функция **SQLConfigDataSource** , которая используется для динамического добавления, изменения или удаления источника данных, использует следующие ключевые слова.  
  
|Ключевое слово|Описание|  
|-------------|-----------------|  
|коллатингсекуенце|Последовательность, в которой сортируются поля.<br /><br /> При использовании драйвера Paradox в качестве последовательности может использоваться ASCII (по умолчанию), Международный, Шведский-финский или норвежский (Датский).<br /><br /> Этот параметр устанавливается так же, как и **последовательность параметров сортировки** в диалоговом окне настройки.|  
|дбк|Имя файла базы данных.<br /><br /> Этот параметр устанавливается так же, как и **база данных** в диалоговом окне «Установка».|  
|дефаултдир|Спецификация пути к каталогу.|  
|DESCRIPTION|Описание данных в источнике данных.<br /><br /> Этот параметр устанавливается так же, как и **Описание** в диалоговом окне установки.|  
|DRIVER|Спецификация пути к библиотеке DLL драйвера.|  
|дриверид|Целочисленный идентификатор драйвера.<br /><br /> 26 (Paradox 3. x)<br /><br /> 282 (Paradox 4. x)<br /><br /> 538 (Paradox 5. x)|  
|ВЗАИМОИСКЛЮЧАЮЩИМ|Определяет, будет ли база данных открыта в монопольном режиме (доступном только одному пользователю) или в общем режиме (доступ к нему осуществляется несколькими пользователями за раз). Может иметь значение true (монопольный режим) или false (общий режим).<br /><br /> В этом случае в диалоговом окне настройки будет выбран тот же параметр, что и **Эксклюзивная** .|  
|ФАЙ|Тип файла Paradox 3. x, Paradox 4. x или Paradox 5. x|  
|FILETYPE|Тип файла для драйвера текста (текст).|  
|пажетимеаут|Указывает период времени (в десятых долях секунды), в течение которого страница (если она не используется) остается в буфере перед удалением. Значение по умолчанию — 600 десятых долей секунды (60 секунд). Обратите внимание, что этот параметр применяется ко всем источникам данных, использующим драйвер ODBC.<br /><br /> Этот параметр устанавливается в диалоговом окне установки того же параметра, что и **время ожидания страницы** .|  
|парадокснетпас|Полный путь к каталогу, содержащему базу данных блокировки Paradox, так как он содержит либо файл PDOXUSRS.net (в Paradox 4).* x*) или Paradox.NET-файл (в Paradox 5.* x*). Если каталог не содержит ни одного из этих файлов, Драйвер Paradox создаст его. Дополнительные сведения об этих файлах см. в документации по Paradox.<br /><br /> Прежде чем можно будет выбрать сетевой каталог, необходимо указать имя пользователя Paradox.<br /><br /> Этот параметр задается так же, как и в диалоговом окне « **Выбор сетевого каталога** ».|  
|парадокснетстиле|Для драйвера Paradox используется стиль доступа к сети, используемый при доступе к данным Paradox: "3. x" для Paradox 3. *x* или 4. x для Paradox 4. *x* или 5. *x*. Можно задать значение "3. x" или "4. x", если используется версия Paradox 4. *x* или 5. *x*; Если используется версия Paradox 3. *x*, стиль должен иметь значение "3. x".<br /><br /> Это позволяет задать тот же параметр, что и **стиль NET** в диалоговом окне установки.|  
|парадоксусернаме|Для драйвера Paradox — имя пользователя Paradox.<br /><br /> Он задает тот же параметр, что и **имя пользователя** в диалоговом окне программы установки.|  
|PWD|Пароль.<br /><br /> Это необязательное ключевое слово, которое никогда не будет записываться в файл драйвером. Он используется в вызове **SQLDriverConnect** для файлов Paradox, защищенных паролем. Используемый пароль будет действителен при каждом открытии таблицы. Если в строке подключения не передается пароль, для этой таблицы пароль не устанавливается. Если в таблицах используются разные пароли, то в одном сеансе нельзя открыть несколько, а таблицы не могут быть присоединены.|  
|READONLY|Значение TRUE, чтобы сделать файл доступным только для чтения; Значение FALSE, чтобы сделать файл недоступным только для чтения.<br /><br /> Этот параметр задается в диалоговом окне установки того же параметра, что и « **только чтение** ».|  
|МНОГОПОТОЧНО|Количество фоновых потоков, используемых ядром. Это значение равно 3 и не может быть изменено.<br /><br /> Это позволяет задать тот же параметр, что и **потоки** в диалоговом окне программы установки.|
