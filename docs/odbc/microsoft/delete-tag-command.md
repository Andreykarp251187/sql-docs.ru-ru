# <a name="delete-tag-command"></a>Команда DELETE TAG
Удаляет тег или теги из составного индекса (.cdx) файла.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
DELETE TAG TagName1 [OF CDXFileName1]  
   [, TagName2 [OF CDXFileName2]] ...  
  Or   
DELETE TAG ALL [OF CDXFileName]  
```  
  
## <a name="arguments"></a>Аргументы  
 *TagName1*OF *CDXFileName1*[, *TagName2*[OF *CDXFileName2*]]...  
 Указывает тег, чтобы удалить из файла составного индекса. Можно удалить несколько тегов с один ТЕГ, удалите, включая список разделенных запятыми имен тегов. Если два или несколько тегов, с тем же именем существует в файлах открыть индекс, можно удалить тег из файла конкретного индекса, включая OF *CDXFileName*.  
  
 ВСЕ [OF *CDXFileName*]  
 Удаляет каждый тег из файла составного индекса. Если текущая таблица содержит файл структурные составного индекса, все теги удаляются из файла индекса, индекс файл будет удален с диска и этот флаг в заголовке таблицы, что указывает на наличие файла связанный структурные составной индекс удален. Использование с OF *CDXFileName* Чтобы удалить все теги из файла открыть составной индекс, отличного от файла структурные составного индекса.  
  
## <a name="remarks"></a>Примечания  
 Составной индекс файлы, созданные с помощью ИНДЕКСА, содержат теги, соответствующий записи индекса. УДАЛИТЬ ТЕГ используется для удаления тег или теги из файлов открыть составной индекс. Вы можете удалить только теги из составного индекса файлы, открытые в текущей рабочей области. Если удалить все теги из файла составной индекс, файл удаляется с диска.  
  
 Visual FoxPro сначала выполнит поиск тегов в файле структурные составного индекса (если он открыт). Если тег отсутствует в файле структурные составного индекса, Visual FoxPro выполняет поиск тега в других файлах открыть составной индекс.
