# <a name="sqlsetscrolloptions-desktop-database-drivers"></a>SQLSetScrollOptions (драйверы для баз данных на настольном компьютере)
Вперед и статические курсоры поддерживаются для SQL_CONCUR_READ_ONLY.  
  
 Только курсоры, управляемые клавиатурой, поддерживаются для аргумента *fConcurrency* SQL_CONCUR_LOCK.  
  
 Аргумент *fConcurrency* SQL_CONCUR_ROWVER не поддерживается.  
  
 Динамические курсоры и смешанные курсоры не поддерживаются.
