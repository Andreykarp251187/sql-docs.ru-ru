# <a name="sqlsetpos-visual-foxpro-odbc-driver"></a>SQLSetPos (драйвер ODBC для Visual FoxPro)
> [!NOTE]  
>  Эта тема содержит Visual FoxPro ODBC Драйвер-специфической информации. Для получения общей информации об этой [ODBC API Reference](../../odbc/reference/syntax/odbc-api-reference.md)функции, см.  
  
 Поддержка: Полная  
  
 Соответствие ODBC API: Уровень 2  
  
 Устанавливает положение курсора в ряд. Для извлечения строк из несвязанных столбцов можно использовать **S'LSetPos** с [помощью S'LGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md) после позиционирования курсора в определенной строке в строке.  
  
 Для получения более подробной информации, *ODBC Programmer's Reference*см. [SQLSetPos](../../odbc/reference/syntax/sqlsetpos-function.md)
