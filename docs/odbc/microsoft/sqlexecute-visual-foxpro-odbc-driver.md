# <a name="sqlexecute-visual-foxpro-odbc-driver"></a>SQLExecute (драйвер ODBC для Visual FoxPro)
> [!NOTE]  
>  В этом разделе содержатся сведения, относящиеся к драйверу ODBC для Visual FoxPro. Общие сведения об этой функции см. в соответствующем разделе [справочника по API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Поддержка: полная  
  
 Соответствие API ODBC: уровень ядра  
  
 Выполняет подготовленную инструкцию SQL (инструкцию, уже подготовленную [SQLPrepare](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md)). Драйвер использует текущие значения переменных маркера параметра, если в инструкции существуют какие либо параметры.  
  
 Дополнительные сведения см. в разделе [SQLExecute](../../odbc/reference/syntax/sqlexecute-function.md) в *справочнике программиста по ODBC*.
