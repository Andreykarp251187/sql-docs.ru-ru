# <a name="odbc-driver-for-oracle-conformance-levels"></a>Уровни соответствия драйвера ODBC для Oracle
> [!IMPORTANT]  
>  Эта функция будет удалена в будущей версии Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставленный Oracle.  
  
 ODBC определяет два типа стандартов соответствия драйверам: стандарт соответствия API и стандарт соответствия грамматике S'L. Соответствие API относится к функциям, которые поддерживает драйвер. Соответствие СЗЛ относится к грамматике S'L, которую поддерживает драйвер. Каждый стандарт соответствия состоит из уровней.  
  
 В этом разделе содержится информация о соответствии драйвера ODBC для Oracle.
