# <a name="sqlstate"></a>SQLSTATE
Для ошибок, возникающих в источнике данных, драйвер ODBC отображает возвращенную родную ошибку в соответствующем S'LSTATE. Для ошибок, обнаруженных водителем или менеджером драйвера, драйвер ODBC или менеджер драйвера генерирует соответствующий S'LSTATE.
