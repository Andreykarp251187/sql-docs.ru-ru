# <a name="installing-the-software-odbc"></a>Установка программного обеспечения (ODBC)
> [!IMPORTANT]  
>  Эта функция будет удалена в будущей версии Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставленный Oracle.  
  
 Драйвер ODBC для Oracle является одним из компонентов доступа к данным. Он сопровождает другие компоненты ODBC, такие как администратор источника данных ODBC, и уже должен быть установлен. Драйвер также можно найти в соответствии с "Драйверы и другие загрузки" на веб-сайте службы поддержки продуктов Майкрософт онлайн в [www.microsoft.com](https://www.microsoft.com).  
  
 Сетевое программное обеспечение должно быть установлено в соответствии с его собственной документацией. Драйвер ODBC для Oracle не требует особых соображений установки до тех пор, пока поддерживается сетевое программное обеспечение.  
  
 Программное обеспечение Oracle должно быть установлено в соответствии со своей собственной документацией. Драйвер ODBC для Oracle обычно не требует особых соображений установки до тех пор, пока драйвер поддерживает версию. Однако, чтобы сохранить продукты совместимы, установите драйвер ODBC для Oracle последним, чтобы убедиться, что у вас есть последняя версия драйвера. Oracle поддерживает общедоступный сайт FTP, где он размещает, среди прочего, патчи для серверных продуктов Oracle и клиентского компонента, который поставляется с серверными продуктами. Эти патчи необходимы для правильного функционирования нескольких продуктов и технологий Майкрософт. Для получения дополнительной информации об этом сайте, см [Oracle Software патчи](../../odbc/microsoft/oracle-software-patches.md).  
  
> [!CAUTION]  
>  Установка программного обеспечения Oracle на MDAC/Windows DAC может перезаписать текущие версии MDAC. Если возникают проблемы с использованием компонентов ODBC, переустановите MDAC.
