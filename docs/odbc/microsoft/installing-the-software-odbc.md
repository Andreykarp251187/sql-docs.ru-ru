# <a name="installing-the-software-odbc"></a>Установка программного обеспечения (ODBC)
> [!IMPORTANT]  
>  Эта функция будет удалена в следующей версии Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставляемый Oracle.  
  
 Драйвер ODBC для Oracle является одним из компонентов доступа к данным. Он поставляется с другими компонентами ODBC, такими как администратор источников данных ODBC, и уже должен быть установлен. Драйвер также можно найти в разделе "драйверы и другие файлы для загрузки" на веб-узле службы технической поддержки Майкрософт по адресу [www.Microsoft.com](https://www.microsoft.com).  
  
 Сетевое программное обеспечение должно быть установлено в соответствии с его собственной документацией. Драйвер ODBC для Oracle не требует особых рекомендаций по установке при условии, что сетевое программное обеспечение поддерживается.  
  
 Программное обеспечение Oracle должно быть установлено в соответствии с его собственной документацией. Драйвер ODBC для Oracle обычно не требует особых рекомендаций по установке, если драйвер поддерживает версию. Однако для обеспечения совместимости продуктов установите драйвер ODBC для Oracle, чтобы убедиться, что у вас установлена последняя версия драйвера. Oracle поддерживает общедоступный FTP-сайт, где он отправляет, помимо прочего, исправления для серверных продуктов Oracle и клиентский компонент, поставляемый с серверными продуктами. Эти исправления необходимы для правильной работы нескольких продуктов и технологий Майкрософт. Дополнительные сведения об этом сайте см. в разделе [обновления программного обеспечения Oracle](../../odbc/microsoft/oracle-software-patches.md).  
  
> [!CAUTION]  
>  Установка программного обеспечения Oracle через MDAC или Windows DAC может перезаписывать текущие версии MDAC. Если проблемы возникают при использовании компонентов ODBC, переустановите MDAC.
