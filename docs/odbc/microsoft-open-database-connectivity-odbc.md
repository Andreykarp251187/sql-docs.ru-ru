# <a name="microsoft-open-database-connectivity-odbc"></a>Microsoft Open Database Connectivity (ODBC)
Интерфейс Microsoft Open Database Connectivity (ODBC) представляет собой языковой интерфейс C, который позволяет приложениям получать доступ к данным из различных систем управления базами данных (DBMS). ODBC — это низкоуровневый высокопроизводительный интерфейс, разработанный специально для реляционных хранилив данных.  
  
 Интерфейс ODBC обеспечивает максимальную совместимость - приложение может получить доступ к данным в различных DBMS через единый интерфейс. Кроме того, это приложение будет независимым от любых DBMS, из которых он получает доступ к данным. Пользователи приложения могут добавлять компоненты программного обеспечения, называемые драйверами, которые взаимодействуют между приложением и конкретным DBMS.
