# <a name="viewing-drivers"></a>Просмотр драйверов
Чтобы настроить источники данных, необходимо установить по крайней мере один драйвер в системе. На вкладке **драйверы** в диалоговом окне **Администратор источников данных ODBC** перечислены все драйверы, установленные на компьютере, включая имя, версию, название компании, имя файла и дату создания файла для каждого драйвера. Для добавления или удаления драйвера из системы необходимо использовать программу установки драйвера. Дополнительные сведения об изменении драйверов см. в разделе [Управление источниками данных](../../odbc/admin/managing-data-sources.md).
