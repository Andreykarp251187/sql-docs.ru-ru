# <a name="viewing-drivers"></a>Просмотр драйверов
Чтобы настроить источники данных, необходимо установить в системе по крайней мере один драйвер. Вкладка **Драйверов** в диалоговом окне администратора **данных ODBC Data Administrator** перечисляет все драйверы, установленные на вашем компьютере, включая имя, версию, компанию, имя файла и дату создания файла каждого драйвера. Необходимо использовать программу настройки драйвера для добавления или удаления драйвера из системы. Для получения дополнительной информации об изменении драйверов [см.](../../odbc/admin/managing-data-sources.md)
