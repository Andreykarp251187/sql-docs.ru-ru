# <a name="viewing-drivers"></a>Просмотр драйверов
Для настройки источников данных, необходимо установить по крайней мере один драйвер в системе. **Драйверы** вкладке **администратор источников данных ODBC** диалоговом окне приводится список всех драйверов, установленных на компьютере, в том числе имя, версию, компании, имя файла и Дата создания файла каждого драйвера. Чтобы добавить или удалить драйвер из системы, необходимо использовать программу установки драйвера. Дополнительные сведения об изменении драйверов см. в разделе [Управление источниками данных](../../odbc/admin/managing-data-sources.md).
