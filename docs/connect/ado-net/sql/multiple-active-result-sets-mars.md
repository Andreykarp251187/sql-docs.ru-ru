# <a name="multiple-active-result-sets-mars"></a>Режим MARS

![Download-DownArrow-Circled](../../../ssdt/media/download.png)[Скачать ADO.NET](../../sql-connection-libraries.md#anchor-20-drivers-relational-access)

Режим MARS — это функция, которая позволяет выполнять несколько пакетов в одном соединении. В предыдущих версиях для одного соединения можно было выполнить только один пакет за раз. Выполнение нескольких пакетов с режимом MARS не подразумевает одновременное выполнение операций.
