# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  Интерфейс метода IBCPSession2 — это расширение IBCPSession, которое предоставляет функцию-член, которая является альтернативой вызову IBCPSession:: BCPColFmt для каждого столбца.  Метода IBCPSession2 наследует от IBCPSession и добавляет один новый метод: [метода IBCPSession2:: BCPSetBulkMode](../../oledb/ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md).
