# <a name="step-1-configure-environment-for-php-development"></a>Шаг 1. Настройка среды для разработки на PHP
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]




* Укажите версию драйвера PHP, которая будет использоваться на основе вашей среды, как указано здесь: [требования к системе для драйверов Майкрософт для PHP для SQL Server](../../connect/php/system-requirements-for-the-php-sql-driver.md)
* Скачайте и установите соответствующий драйвер PHP: [скачайте драйвер Microsoft PHP Driver](https://www.microsoft.com/download/details.aspx?id=20098)  
* Скачайте и установите соответствующий драйвер ODBC: [скачайте драйвер ODBC для SQL Server](../../connect/odbc/download-odbc-driver-for-sql-server.md)  
* Настройте драйвер PHP и веб-сервер для конкретной операционной системы.

### <a name="windows"></a>Windows  
  

* Настройте загрузку драйвера PHP, как указано здесь: [Загрузка драйверов Майкрософт для PHP для SQL Server](../../connect/php/loading-the-php-sql-driver.md) 
* Настройте службы IIS для размещения приложений PHP, как указано здесь: [Настройка служб IIS для драйверов Майкрософт для PHP для SQL Server](../../connect/php/configuring-iis-for-php-sql-driver.md)

### <a name="linux-and-mac"></a>Linux и Mac


*   Настройте загрузку драйвера PHP и настройте веб-сервер для размещения приложений PHP, как указано здесь: [руководство по установке драйверов PHP Linux и Mac](../../connect/php/installation-tutorial-linux-mac.md)
