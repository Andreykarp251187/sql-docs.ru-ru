# <a name="step-1-configure-development-environment-for-java-development"></a>Шаг 1. Настройка среды разработки для разработки на Java
  
## <a name="windows"></a>Windows  
  
* Укажите версию драйвера JDBC, которая будет использоваться на основе вашей среды, как указано здесь: [требования к системе для драйвера JDBC](../../connect/jdbc/system-requirements-for-the-jdbc-driver.md) .  
* Скачайте и установите соответствующий драйвер JDBC: [скачайте драйвер Microsoft JDBC Driver для SQL Server](../../connect/jdbc/download-microsoft-jdbc-driver-for-sql-server.md)  
* Задайте путь к классу на основе версии драйвера, как указано здесь: [Использование драйвера JDBC](../../connect/jdbc/using-the-jdbc-driver.md)
