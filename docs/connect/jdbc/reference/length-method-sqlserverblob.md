# <a name="length-method-sqlserverblob"></a>Метод length (SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает число байтов в большом двоичном объекте.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public long length()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Значение **long**, указывающее число байтов.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Этот метод length задается с помощью метода length в интерфейсе java.sql.Blob.
