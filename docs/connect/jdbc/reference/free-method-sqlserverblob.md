# <a name="free-method-sqlserverblob"></a>Метод free (SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Этот метод освобождает большой двоичный объект и занятые им ресурсы.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public void free()  
```  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Этот метод free определен с помощью метода free в интерфейсе java.sql.Blob.
