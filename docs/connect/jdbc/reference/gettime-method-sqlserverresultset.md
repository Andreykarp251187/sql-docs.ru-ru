# <a name="gettime-method-sqlserverresultset"></a>Метод getTime (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Извлекает значение заданного столбца в текущей строке этого объекта [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) в виде объекта java.sql.Time на языке программирования Java.  
  
## <a name="overload-list"></a>Список перегрузок  
  
|Имя|Description|  
|----------|-----------------|  
|[Метод getTime (int)](../../../connect/jdbc/reference/gettime-method-int-sqlserverresultset.md)|Получает значение заданного индекса столбца в текущей строке этого объекта [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) в виде объекта java.sql.Time на языке программирования Java.|  
|[Метод getTime (int, java.util.Calendar)](../../../connect/jdbc/reference/gettime-method-int-java-util-calendar-sqlserverresultset.md)|Получает значение индекса заданного столбца в текущей строке этого объекта [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) в виде объекта java.sql.Time на языке программирования Java с помощью заданного объекта Calendar.|  
|[Метод getTime (java.lang.String)](../../../connect/jdbc/reference/gettime-method-java-lang-string-sqlserverresultset.md)|Получает значение имени заданного столбца в текущей строке этого объекта [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) в виде объекта java.sql.Time на языке программирования Java.|  
|[Метод getTime (java.lang.String, java.util.Calendar)](../../../connect/jdbc/reference/gettime-method-java-lang-string-java-util-calendar-sqlserverresultset.md)|Возвращает значение имени заданного столбца в текущей строке этого объекта [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) в виде объекта java.sql.Time на языке программирования Java с помощью указанного объекта Calendar.|
