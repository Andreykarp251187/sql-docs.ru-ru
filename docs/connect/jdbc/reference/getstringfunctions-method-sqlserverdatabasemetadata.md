# <a name="getstringfunctions-method-sqlserverdatabasemetadata"></a>Метод getStringFunctions (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает список функций **String** (разделенных запятыми), доступных в этой базе данных.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public java.lang.String getStringFunctions()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Объект **строка** , содержащий **строка** функции.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Этот метод getStringFunctions указывается с помощью метода getStringFunctions в интерфейсе java.sql.DatabaseMetaData.
