# <a name="getarray-method-sqlservercallablestatement"></a>Метод getArray (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Извлекает значение указанного параметра в виде объекта Array.

|----------|-----------------|  
|[getArray (int)](../../../connect/jdbc/reference/getarray-method-int.md)|Извлекает значение указанного параметра в виде объекта Array по заданному индексу параметра.|  
|[getArray (java.lang.String)](../../../connect/jdbc/reference/getarray-method-java-lang-string.md)|Извлекает значение указанного параметра в виде объекта Array по заданному имени параметра.|
