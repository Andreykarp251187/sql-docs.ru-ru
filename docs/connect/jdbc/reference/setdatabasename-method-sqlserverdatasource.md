# <a name="setdatabasename-method-sqlserverdatasource"></a>Метод setDatabaseName (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Задает имя базы данных для соединения.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public void setDatabaseName(java.lang.String databaseName)  
```  
  
#### <a name="parameters"></a>Параметры  
 *databaseName*  
  
 Значение **String**, содержащее имя базы данных.
