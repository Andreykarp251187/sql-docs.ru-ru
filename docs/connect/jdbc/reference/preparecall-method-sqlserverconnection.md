# <a name="preparecall-method-sqlserverconnection"></a>Метод prepareCall (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Создает объект [SQLServerCallableStatement](../../../connect/jdbc/reference/sqlservercallablestatement-class.md) для вызова хранимых процедур базы данных.  
  
## <a name="overload-list"></a>Список перегрузок  
  
|Имя|Описание|  
|----------|-----------------|  
|[prepareCall (java.lang.String)](../../../connect/jdbc/reference/preparecall-method-java-lang-string.md)|Создает объект SQLServerCallableStatement для вызова хранимых процедур базы данных.|  
|[prepareCall (java.lang.String, int, int)](../../../connect/jdbc/reference/preparecall-method-java-lang-string-int-int.md)|Создает объект SQLServerCallableStatement, который создает объекты [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) с заданным типом и видом параллелизма.|  
|[prepareCall (java.lang.String, int, int, int)](../../../connect/jdbc/reference/preparecall-method-java-lang-string-int-int-int.md)|Создает объект SQLServerCallableStatement, который создает объекты SQLServerResultSet с заданным типом, видом параллелизма и возможностью сохранения.|
