# <a name="getcharacterstream-method-sqlserverclob"></a>Метод getCharacterStream (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Извлекает значение **Clob**, определенное объектом java.sql.Clob в качестве объекта java.io.Reader.  
  
## <a name="overload-list"></a>Список перегрузок  
  
|Имя|Описание|  
|----------|-----------------|  
|[Метод getCharacterStream ()](../../../connect/jdbc/reference/getcharacterstream-method.md)|Возвращает данные Clob в виде объекта java.io.Reader или потока символов.|  
|[Метод getCharacterStream (long, long)](../../../connect/jdbc/reference/getcharacterstream-method-long-long.md)|Возвращает данные Clob в виде объекта java.io.Reader или потока символов с указанной позицией и длиной.|
