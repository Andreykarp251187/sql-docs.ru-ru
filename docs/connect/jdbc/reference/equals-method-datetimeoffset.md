# <a name="equals-method-datetimeoffset"></a>Метод equals (DateTimeOffset)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Сравнивает этот **DateTimeOffset** объект с другим объектом **объект**.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public boolean equals(Object o)  
```  
  
#### <a name="parameters"></a>Параметры  
 *o*  
  
 Объект для сравнения с текущим экземпляром.  
  
## <a name="return-value"></a>Возвращаемое значение  
 Возвращает значение true, если данный **DateTimeOffset** объект представляет ту же точку во времени, в то же смещение относительно GMT, что и другой **DateTimeOffset**.
