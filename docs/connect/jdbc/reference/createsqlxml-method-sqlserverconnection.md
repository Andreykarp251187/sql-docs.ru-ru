# <a name="createsqlxml-method-sqlserverconnection"></a>Метод createSQLXML (SQLServerConnection)

  Создает объект **SQLXML** без каких-либо данных.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public java.sql.SQLXML createSQLXML()  
```
