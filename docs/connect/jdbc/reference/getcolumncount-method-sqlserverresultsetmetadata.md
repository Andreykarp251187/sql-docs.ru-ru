# <a name="getcolumncount-method-sqlserverresultsetmetadata"></a>Метод getColumnCount (SQLServerResultSetMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает число столбцов в результирующем наборе.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public int getColumnCount()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Значение **int**, указывающее количество столбцов.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Этот метод getColumnCount указывается с помощью метода getColumnCount в интерфейсе java.sql.ResultSetMetaData.
