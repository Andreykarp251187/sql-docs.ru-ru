# <a name="supportstransactions-method-sqlserverdatabasemetadata"></a>Метод supportsTransactions (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает значение, определяющее, поддерживает ли эта база данных транзакции.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public boolean supportsTransactions()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 **значение true** , если поддерживается. В противном случае — **false**.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Этот метод supportsTransactions задается методом supportsTransactions в интерфейсе Java. SQL. DatabaseMetaData.
