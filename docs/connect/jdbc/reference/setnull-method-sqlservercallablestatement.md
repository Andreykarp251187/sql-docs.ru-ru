# <a name="setnull-method-sqlservercallablestatement"></a>Метод setNull (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Устанавливает значение NULL для параметра, определяемого по заданному типу.

|[setNull (java.lang.String, int, java.lang.String)](../../../connect/jdbc/reference/setnull-method-java-lang-string-int-java-lang-string.md)|Устанавливает указанный параметр в значение NULL по заданному имени и типу параметра.|
