# <a name="concurssoptimisticcc-field-sqlserverresultset"></a>Поле CONCUR_SS_OPTIMISTIC_CC (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Указывает тип оптимистичного параллелизма чтения-записи в [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] без блокировки строк.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public static final int CONCUR_SS_OPTIMISTIC_CC  
```  
  
## <a name="field-value"></a>Значение поля  
 Значение **int** , равное 1008.
