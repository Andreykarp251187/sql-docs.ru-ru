# <a name="datetimeoffset-members"></a>Элементы DateTimeOffset
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  В следующих таблицах перечислены доступные члены класса [DateTimeOffset](../../../connect/jdbc/reference/datetimeoffset-class.md).  
  
## <a name="constructors"></a>Конструкторы  
 Нет.  
  
## <a name="fields"></a>Поля  
 Нет.  
  
## <a name="methods"></a>Методы  
  
|Имя|Описание|  
|----------|-----------------|  
|[compareTo](../../../connect/jdbc/reference/compareto-method-datetimeoffset.md)|Сравнивает этот объект **DateTimeOffset** с другим объектом **DateTimeOffset** в зависимости от времени по Гринвичу.|  
|[equals](../../../connect/jdbc/reference/equals-method-datetimeoffset.md)|Сравнивает этот объект **DateTimeOffset** с другим **объектом**.|  
|[getMinutesOffset](../../../connect/jdbc/reference/getminutesoffset-method-datetimeoffset.md)|Возвращает смещение данного объекта **DateTimeOffset** в МИНУТАХ от GMT.|  
|[getTimestamp](../../../connect/jdbc/reference/gettimestamp-method-datetimeoffset.md)|Возвращает точку во времени, представленную этим объектом **DateTimeOffset** , в виде Java. SQL. timestamp.|  
|[hashCode](../../../connect/jdbc/reference/hashcode-method-datetimeoffset.md)|Возвращает значение хэш-кода для объекта.|  
|[toString](../../../connect/jdbc/reference/tostring-method-datetimeoffset.md)|Возвращает строковое представление объекта **DateTimeOffset** .|  
|[valueOf](../../../connect/jdbc/reference/valueof-method-datetimeoffset.md)|Создает объект **DateTimeOffset**, представляющий точку во времени с указанным смещением относительно времени по Гринвичу (GMT).|
