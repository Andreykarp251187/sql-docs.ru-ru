# <a name="typessscrollstatic-field-sqlserverresultset"></a>Поле TYPE_SS_SCROLL_STATIC (SQLServerResultSet)

## <a name="syntax"></a>Синтаксис  
  
```  
  
public static final int TYPE_SS_SCROLL_STATIC
```  
