# <a name="typessdirectforwardonly-field-sqlserverresultset"></a>Поле TYPE_SS_DIRECT_FORWARD_ONLY (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

