# <a name="supportsstoredprocedures-method-sqlserverdatabasemetadata"></a>Метод supportsStoredProcedures (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает значение, определяющее, поддерживает ли эта база данных вызовы хранимых процедур с использованием синтаксиса перехода.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public boolean supportsStoredProcedures()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 **true**, если поддерживается. В противном случае — **false**.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Этот метод supportsStoredProcedures задается с помощью метода supportsStoredProcedures в интерфейсе java.sql.DatabaseMetaData.
