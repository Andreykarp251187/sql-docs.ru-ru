# <a name="change-data-capture-for-oracle-by-attunity"></a>Change Data Capture для Oracle от Attunity

Скачиваемый файл Change Data Capture для Oracle от Attunity содержит 2 указанных ниже компонента.

-   [Конструктор системы отслеживания измененных данных для Oracle компании Attunity](../../integration-services/change-data-capture/change-data-capture-designer-for-oracle-by-attunity.md)

## <a name="get-the-change-data-capture-for-oracle-by-attunity-download"></a>Получить скачиваемый файл Change Data Capture для Oracle от Attunity
Конструктор и служба системы отслеживания измененных данных Microsoft® для Oracle от Attunity для Microsoft SQL Server® 2016 включены в пакет дополнительных компонентов SQL Server 2016. Скачать компоненты пакета дополнительных компонентов можно со страницы [Microsoft® SQL Server® 2016 Feature Pack](https://go.microsoft.com/fwlink/?LinkId=746297)(Пакет дополнительных компонентов Microsoft® SQL Server® 2016).
