# <a name="how-to-create-the-sql-server-change-database-instance"></a>Как создать экземпляр изменения базы данных SQL Server

[!INCLUDE[ssis-appliesto](../../includes/ssis-appliesto-ssvrpluslinux-asdb-asdw-xxx.md)]


  В этой процедуре описано создание экземпляров CDC с помощью консоли конструктора CDC.  
  
### <a name="to-create-cdc-instances"></a>Создание экземпляров CDC  
  
1.  В меню **Пуск** выберите **Консоль конструктора CDC**.  
  
2.  На левой панели разверните узел **Отслеживание измененных данных**.  
  
3.  Выберите службу, в которой необходимо создать новый экземпляр CDC.  
  
4.  На панели **Действия** с правой стороны консоли конструктора CDC выберите **Создать экземпляр Oracle CDC**.  
  
     Можно также щелкнуть правой кнопкой мыши службу, в которой необходимо создать новый экземпляр CDC, а затем выбрать команду **Создать экземпляр CDC Oracle**.  
  
5.  Введите необходимые сведения в мастере создания экземпляра, чтобы создать новый экземпляр. Сведения о том, какие данные требуются мастеру, см. в разделе [Use the New Instance Wizard](../../integration-services/change-data-capture/use-the-new-instance-wizard.md).
